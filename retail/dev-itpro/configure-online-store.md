# <a name="configure-an-online-store"></a>Konfigurowanie sklepu internetowego

Ten artykuł zawiera łącza do tematów, które pomogą Ci centralnie konfigurować i zarządzać i sklepu internetowego z Microsoft Dynamics 365 dla operacji.

Tematy wymienione w tabeli poniżej pomagają skonfigurować Dynamics 365 dla operacji - Moduł Retail components i sklepu internetowego sieci sprzedaży w usłudze Dynamics 365 dla operacji klienta.

## <a name="configure-an-online-store"></a>Konfigurowanie sklepu internetowego
| Zadanie                                                | Szczegóły                                                                                                                                                                                                                                                                                                                                                   | Tematy                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Konfigurowanie składników sieci sprzedaży.                        | Konfigurowanie i obsługa informacji dla operacji detalicznych. Informacje te obejmują sklepy, podatków, produkty, kart upominkowych, promocji i rabatów.                                                                                                                                                                                                          | [Konfigurowanie i obsługiwanie sieci sprzedaży](https://technet.microsoft.com/en-us/library/hh597201.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                          |
| Konfigurowanie hierarchii nawigacji kanału sprzedaży.    | Utworzenie hierarchii kategorii nawigacji kanału sprzedaży detalicznej konfigurowania struktury kategorii dla produktów, które oferujesz za pośrednictwem sklepu internetowego. Zdefiniowanie hierarchii kategorii i przypisać do kategorii produktów, grupy atrybutów produktu i wartości atrybutów. Następnie przypisz hierarchię kategorii do sklepu internetowego.                            | [Konfigurowanie hierarchii sieci sprzedaży](https://technet.microsoft.com/en-us/library/hh580593.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012) [Ustawianie atrybutów i typów atrybutu](https://technet.microsoft.com/en-us/library/hh227548.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012) [Ustawianie grup atrybutów sieci sprzedaży](https://technet.microsoft.com/en-us/library/jj728713.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012) |
| Dodawanie sklepu internetowego do hierarchii organizacyjnej. | Zanim można przypisać asortymentów produktów lub realizacji zamówień dla sklepu internetowego, który został utworzony lub wygenerować raporty zawierające informacje o sklepie internetowym, należy przypisać sklepu do jednej lub kilku hierarchii organizacji. Na poziomie minimum, należy przypisać sklep internetowy do hierarchii organizacyjnej, która zawiera asortymenty produktów. | [Konfigurowanie sklepu internetowego](https://technet.microsoft.com/en-us/library/jj682095.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Dodawanie metod dostawy do sklepu internetowego.          | Wybierz metody dostawy, które oferuje sklepu internetowego.                                                                                                                                                                                                                                                                                                 | [Konfigurowanie sklepu internetowego](https://technet.microsoft.com/en-us/library/jj682095.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Mapowanie atrybutów i dodawanie metadanych.                   | Wybierz opcje, które wskazują, jak atrybuty dla każdej kategorii lub kanału produktu powinny zachowywać się w sklepie online w witrynie programu Microsoft SharePoint.                                                                                                                                                                                              | [Konfigurowanie sklepu internetowego](https://technet.microsoft.com/en-us/library/jj682095.aspx) (zawartość TechNet systemu Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
