# <a name="create-a-journal-entry-using-a-template"></a><span data-ttu-id="040fe-103">Tworzenie wpisu w arkuszu za pomocą szablonu</span><span class="sxs-lookup"><span data-stu-id="040fe-103">Create a journal entry using a template</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="040fe-104">Zaksięgowane arkusze załączników mogą być zapisywane jako szablony załączników i stosowane w nowych załącznikach arkuszy.</span><span class="sxs-lookup"><span data-stu-id="040fe-104">Posted journal vouchers can be saved as Voucher templates and applied in a new journal voucher.</span></span> <span data-ttu-id="040fe-105">Ta procedura wykorzystuje firmę demonstracyjną USMF.</span><span class="sxs-lookup"><span data-stu-id="040fe-105">This procedure uses the USMF demo company.</span></span>

    * <span data-ttu-id="040fe-108">Ta procedura rozpoczyna się od utworzenia i zaksięgowania załącznika arkusza, ale wszelkie wcześniej zaksięgowane załączniki arkusze można zapisać jako szablon.</span><span class="sxs-lookup"><span data-stu-id="040fe-108">This procedure starts by creating and posting a journal voucher, but any previously posted journal voucher can be saved as a template.</span></span>

3. <span data-ttu-id="040fe-110">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="040fe-110">In the list, find and select the desired record.</span></span>
4. <span data-ttu-id="040fe-111">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="040fe-111">In the list, click the link in the selected row.</span></span>

6. <span data-ttu-id="040fe-113">Wprowadź konto w polu Typ konta.</span><span class="sxs-lookup"><span data-stu-id="040fe-113">Enter an account for the Account type.</span></span>

8. <span data-ttu-id="040fe-115">Wprowadź kwotę w polu Debet.</span><span class="sxs-lookup"><span data-stu-id="040fe-115">Enter an amount in the Debit field.</span></span>
9. <span data-ttu-id="040fe-116">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="040fe-116">Click New.</span></span>
10. <span data-ttu-id="040fe-117">Wprowadź inne konto w polu Typ konta.</span><span class="sxs-lookup"><span data-stu-id="040fe-117">Enter a different account for the Account type.</span></span>
11. <span data-ttu-id="040fe-118">Wypełnij pole Opis.</span><span class="sxs-lookup"><span data-stu-id="040fe-118">In the Description field, type a value.</span></span>
12. <span data-ttu-id="040fe-119">Wprowadź kwotę w polu Debet.</span><span class="sxs-lookup"><span data-stu-id="040fe-119">Enter an amount in the Debit field.</span></span>
13. <span data-ttu-id="040fe-120">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="040fe-120">Click New.</span></span>
14. <span data-ttu-id="040fe-121">W polu Konto podaj żądane wartości.</span><span class="sxs-lookup"><span data-stu-id="040fe-121">In the Account field, specify the desired values.</span></span>
15. <span data-ttu-id="040fe-122">W polu Opis wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="040fe-122">In the Description field, type a value.</span></span>
16. <span data-ttu-id="040fe-123">Wprowadź kwotę w polu Kredyt, aby zbilansować załącznik.</span><span class="sxs-lookup"><span data-stu-id="040fe-123">Enter an amount in the Credit field to balance the voucher.</span></span>

18. <span data-ttu-id="040fe-125">Kliknij przycisk Funkcje.</span><span class="sxs-lookup"><span data-stu-id="040fe-125">Click Functions.</span></span>
19. <span data-ttu-id="040fe-126">Kliknij przycisk Zapisz szablon załącznika.</span><span class="sxs-lookup"><span data-stu-id="040fe-126">Click Save voucher template.</span></span>

    * <span data-ttu-id="040fe-129">• Procent: kwoty w załączniku są konwertowane na współczynniki procentowe, co pozwala na zastosowanie wszelkich kwot podczas wybierania szablonu załącznika.</span><span class="sxs-lookup"><span data-stu-id="040fe-129">• Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>  <span data-ttu-id="040fe-130">• Kwota: przechowywane i stosowane będą wartości rzeczywiste.</span><span class="sxs-lookup"><span data-stu-id="040fe-130">• Amount: The actual amounts will be stored and applied.</span></span>  
21. <span data-ttu-id="040fe-131">Kliknij opcję Arkusze finansowe.</span><span class="sxs-lookup"><span data-stu-id="040fe-131">Click General journals.</span></span>
22. <span data-ttu-id="040fe-132">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="040fe-132">Click New.</span></span>
23. <span data-ttu-id="040fe-133">W polu Nazwa kliknij przycisk rozwijany, aby otworzyć wyszukiwanie.</span><span class="sxs-lookup"><span data-stu-id="040fe-133">In the Name field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="040fe-134">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="040fe-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="040fe-135">Kliknij przycisk Wiersze.</span><span class="sxs-lookup"><span data-stu-id="040fe-135">Click Lines.</span></span>
26. <span data-ttu-id="040fe-136">Kliknij przycisk Funkcje.</span><span class="sxs-lookup"><span data-stu-id="040fe-136">Click Functions.</span></span>
27. <span data-ttu-id="040fe-137">Kliknij opcję Wybierz szablon załącznika.</span><span class="sxs-lookup"><span data-stu-id="040fe-137">Click Select voucher template.</span></span>
28. <span data-ttu-id="040fe-138">Znajdź szablon utworzony wcześniej.</span><span class="sxs-lookup"><span data-stu-id="040fe-138">Find the template that you created earlier.</span></span> <span data-ttu-id="040fe-139">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="040fe-139">Click OK.</span></span>
    * <span data-ttu-id="040fe-140">Jeśli istnieją inne szablony, może być konieczne kliknięcie przycisku Poprzedni krok, a następnie wybranie właściwego szablonu.</span><span class="sxs-lookup"><span data-stu-id="040fe-140">You may need to click Previous step and then select the correct template if other templates exist.</span></span>

30. <span data-ttu-id="040fe-143">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="040fe-143">Click OK.</span></span>
