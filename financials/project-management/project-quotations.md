<span data-ttu-id="1b489-105">Ten artykuł zawiera wprowadzenie do koncepcji ofert w ramach projektu, których można używać w celu składania odbiorcom atrakcyjnych ofert, co będzie stanowić pierwszy krok fazy projektu.</span><span class="sxs-lookup"><span data-stu-id="1b489-105">This article introduces the concept of project quotations, which you can use to make an attractive offer to a customer as the first step of the project phase.</span></span> <span data-ttu-id="1b489-106">Oferta w ramach projektu może obejmować oferowane towary i usługi, podstawowe informacje kontaktowe, specjalne umowy handlowe i rabaty oraz możliwe podatki i dopłaty.</span><span class="sxs-lookup"><span data-stu-id="1b489-106">A project quotation might include the items and services that are quoted, basic contact information, special trade agreements and discounts, and possible taxes and surcharges.</span></span> 

<span data-ttu-id="1b489-107">Możliwość monitorowania, przeglądania oraz kontrolowania ofert i zamówień projektu jest ważnym elementem zarządzania projektem.</span><span class="sxs-lookup"><span data-stu-id="1b489-107">The ability to monitor, review, and control the pipeline of project quotations and orders is an important part of project management.</span></span> <span data-ttu-id="1b489-108">Wykonywanie tych zadań mogą ułatwić różne narzędzia dostępne w programie Microsoft Dynamics 365 for Finance and Operations Enterprise Edition.</span><span class="sxs-lookup"><span data-stu-id="1b489-108">Various tools in Microsoft Dynamics 365 for Finance and Operations, Enterprise edition can help with these tasks.</span></span> <span data-ttu-id="1b489-109">Na przykład prawidłowe definicje danych referencyjnych (typy ofert, źródła ofert, a także prognozy i prawdopodobieństwa), ułatwiają analizowanie przebiegu działań.</span><span class="sxs-lookup"><span data-stu-id="1b489-109">For example, correct reference data definitions (quotation types, quotation origin, and prognosis and probability) help you analyze the pipeline.</span></span> <span data-ttu-id="1b489-110">Narzędzia te umożliwiają określanie kategorii przyczyn powodzenia lub niepowodzenia ofert i możliwych wartości ofert.</span><span class="sxs-lookup"><span data-stu-id="1b489-110">You can use these tools to categorize the reasons why a project quotation was won or lost, and to determine the potential value of a quotation.</span></span> 

<span data-ttu-id="1b489-111">Oferta w ramach projektu może obejmować usługi, podstawowe informacje kontaktowe, specjalne umowy handlowe i rabaty oraz możliwe podatki i dopłaty oszacowane dla projektu.</span><span class="sxs-lookup"><span data-stu-id="1b489-111">In a project quotation, you enter the services, basic contact information, special trade agreements and discounts, and estimated taxes and surcharges for a project.</span></span> <span data-ttu-id="1b489-112">Można również zaznaczyć działania lub zadania w projekcie i utworzyć hierarchię zadań i podzadań.</span><span class="sxs-lookup"><span data-stu-id="1b489-112">You can also select the activities or tasks for a project, and create a hierarchy of tasks and subtasks.</span></span> <span data-ttu-id="1b489-113">Dla każdej czynności można wprowadzić szczegóły dotyczące czasu i czas trwania działania, a także umiejętności i doświadczenie wymagane dla pracowników, którzy wykonują działanie.</span><span class="sxs-lookup"><span data-stu-id="1b489-113">For each activity, you can enter details about the timing and duration of the activity, and about the skills and experience that are required for workers who perform the activity.</span></span>

<span data-ttu-id="1b489-116">Jeśli oferta w ramach projektu zostanie zaakceptowana przez nabywcę, można skopiować informacje z oferty projektu do projektu.</span><span class="sxs-lookup"><span data-stu-id="1b489-116">If the customer approves the project quotation, you can copy the information in the project quotation to a project.</span></span> <span data-ttu-id="1b489-117">Można także skopiować informacje o ofercie w ramach projektu do prognozy projektu w tym samym czasie.</span><span class="sxs-lookup"><span data-stu-id="1b489-117">You can also copy the project quotation information to a project forecast at the same time.</span></span>
