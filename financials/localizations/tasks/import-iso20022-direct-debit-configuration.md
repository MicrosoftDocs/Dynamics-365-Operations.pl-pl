<span data-ttu-id="c2ebd-106">Procedura została utworzona przy użyciu danych firmy demonstracyjnej DEMF, ale do tego celu można użyć danych dowolnej firmy demonstracyjnej.</span><span class="sxs-lookup"><span data-stu-id="c2ebd-106">This procedure was created using the demo data company DEMF but you can use any demo data company for this purpose.</span></span>
