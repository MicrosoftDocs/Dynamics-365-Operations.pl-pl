# <a name="report-progress-on-a-mobile-job-device"></a><span data-ttu-id="243cb-103">Zgłaszanie postępu za pomocą urządzenia przenośnego zadania</span><span class="sxs-lookup"><span data-stu-id="243cb-103">Report progress on a mobile job device</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="243cb-104">W tej procedurze pokazano sposób uruchamiania i raportowania o postępie zadania produkcyjnego w formularzu rejestracji urządzenia obsługującego zadania.</span><span class="sxs-lookup"><span data-stu-id="243cb-104">This procedure shows you how to start and report progress on a production job in the job device registration form.</span></span>



<span data-ttu-id="243cb-105">Aby można było uruchomić tę procedurę, musisz mieć rolę Administrator systemu lub Operator maszyny skojarzoną ze swoim kontem użytkownika.</span><span class="sxs-lookup"><span data-stu-id="243cb-105">To be able to run this procedure you must have the System administator or Machine Operator role associated with the user account.</span></span>

1. <span data-ttu-id="243cb-106">Wybierz kolejno opcje Kontrola produkcji > Wykonywanie produkcji > Urządzenie karty zadań.</span><span class="sxs-lookup"><span data-stu-id="243cb-106">Go to Production control > Manufacturing execution > Job card device.</span></span>

3. <span data-ttu-id="243cb-109">Kliknij przycisk Zaloguj się.</span><span class="sxs-lookup"><span data-stu-id="243cb-109">Click Log in.</span></span>
4. <span data-ttu-id="243cb-110">Kliknij przycisk Filtr.</span><span class="sxs-lookup"><span data-stu-id="243cb-110">Click the Filter button.</span></span>
5. <span data-ttu-id="243cb-111">Zaznacz pole wyboru Zastosuj filtr konfiguracji lub usuń jego zaznaczenie.</span><span class="sxs-lookup"><span data-stu-id="243cb-111">Check or uncheck the Apply configuration filter check box.</span></span> <span data-ttu-id="243cb-112">Jeśli ustawisz filtr, można użyć jednostki produkcyjnej 110 z firmy USMF.</span><span class="sxs-lookup"><span data-stu-id="243cb-112">If you set a filter you can use production unit 110 in USMF.</span></span>

7. <span data-ttu-id="243cb-114">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="243cb-114">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="243cb-115">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="243cb-115">Click OK.</span></span>
9. <span data-ttu-id="243cb-116">Kliknij przycisk Uruchom zadanie.</span><span class="sxs-lookup"><span data-stu-id="243cb-116">Click the Start job button.</span></span>
10. <span data-ttu-id="243cb-117">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="243cb-117">Click OK.</span></span>
11. <span data-ttu-id="243cb-118">Kliknij przycisk Zgłaszanie postępu.</span><span class="sxs-lookup"><span data-stu-id="243cb-118">Click the Report progress button.</span></span>
12. <span data-ttu-id="243cb-119">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="243cb-119">Click OK.</span></span>
13. <span data-ttu-id="243cb-120">Kliknij przycisk Następne zadanie.</span><span class="sxs-lookup"><span data-stu-id="243cb-120">Click the Next job button.</span></span>

16. <span data-ttu-id="243cb-123">Kliknij przycisk Przerwa.</span><span class="sxs-lookup"><span data-stu-id="243cb-123">Click the Break button.</span></span>

18. <span data-ttu-id="243cb-125">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="243cb-125">Click OK.</span></span>
19. <span data-ttu-id="243cb-126">Kliknij przycisk Opuszczanie.</span><span class="sxs-lookup"><span data-stu-id="243cb-126">Click the Leaving button.</span></span>

21. <span data-ttu-id="243cb-128">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="243cb-128">Click OK.</span></span>
22. <span data-ttu-id="243cb-129">W polu WorkerTextField zaloguj się ponownie.</span><span class="sxs-lookup"><span data-stu-id="243cb-129">In the WorkerTextField field, log in again.</span></span> <span data-ttu-id="243cb-130">Można wybrać pracownika „123” z danych firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="243cb-130">You can select worker '123' in USMF demo data.</span></span>
23. <span data-ttu-id="243cb-131">Kliknij przycisk Zaloguj się.</span><span class="sxs-lookup"><span data-stu-id="243cb-131">Click Log in.</span></span>
24. <span data-ttu-id="243cb-132">Kliknij przycisk Zatrzymaj przerwę.</span><span class="sxs-lookup"><span data-stu-id="243cb-132">Click Stop break.</span></span>
25. <span data-ttu-id="243cb-133">Kliknij przycisk Działanie.</span><span class="sxs-lookup"><span data-stu-id="243cb-133">Click the Activity button.</span></span>
26. <span data-ttu-id="243cb-134">Kliknij przycisk Anuluj.</span><span class="sxs-lookup"><span data-stu-id="243cb-134">Click Cancel.</span></span>

28. <span data-ttu-id="243cb-136">Kliknij przycisk wyrejestrowania.</span><span class="sxs-lookup"><span data-stu-id="243cb-136">Select to clock out.</span></span>
