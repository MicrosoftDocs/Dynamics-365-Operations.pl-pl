# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="8205e-103">Wyświetlanie wpisów w arkuszu lub transakcji</span><span class="sxs-lookup"><span data-stu-id="8205e-103">View journal entries or transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8205e-104">W tej procedurze pokazano sposób używania zapytań o transakcje w załącznikach w celu wyszukiwania wpisów w arkuszu lub transakcji.</span><span class="sxs-lookup"><span data-stu-id="8205e-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="8205e-105">Wybierz kolejno opcje Księga główna > Zapytania i raporty > Transakcje na załączniku.</span><span class="sxs-lookup"><span data-stu-id="8205e-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="8205e-106">Zaznacz pole, dla którego chcesz zdefiniować kryteria filtrowania.</span><span class="sxs-lookup"><span data-stu-id="8205e-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="8205e-107">Wprowadź kryteria filtrowania zaznaczonego pola.</span><span class="sxs-lookup"><span data-stu-id="8205e-107">Enter your filter criteria for the selected field.</span></span>
    * <span data-ttu-id="8205e-108">Można filtrować według jednej wartości lub zakresu.</span><span class="sxs-lookup"><span data-stu-id="8205e-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="8205e-109">Podczas definiowania zakresu upewnij się, że jest używana poprawna składnia.</span><span class="sxs-lookup"><span data-stu-id="8205e-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="8205e-110">Wartości powinny być rozdzielone dwoma kropkami (..).</span><span class="sxs-lookup"><span data-stu-id="8205e-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="8205e-111">Kliknij kartę Sprzężenia, aby dodać więcej tabel, z których można filtrować.</span><span class="sxs-lookup"><span data-stu-id="8205e-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="8205e-112">W drzewie zaznacz pozycję „Tabele\Wpis w arkuszu finansowym”.</span><span class="sxs-lookup"><span data-stu-id="8205e-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="8205e-113">Kliknij opcję Dodaj sprzężenie tabeli.</span><span class="sxs-lookup"><span data-stu-id="8205e-113">Click Add table join.</span></span>
7. <span data-ttu-id="8205e-114">Kliknij przycisk Anuluj, jeśli nie chcesz dodawać kolejnej tabeli.</span><span class="sxs-lookup"><span data-stu-id="8205e-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="8205e-115">Kliknij kartę Zakres.</span><span class="sxs-lookup"><span data-stu-id="8205e-115">Click the Range tab.</span></span>
9. <span data-ttu-id="8205e-116">Aby uaktywnić kwerendę, kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="8205e-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="8205e-117">Kliknij opcję Podstawa transakcji.</span><span class="sxs-lookup"><span data-stu-id="8205e-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="8205e-118">Różne przyciski wokół siatki mogą służyć do szukania dodatkowych informacji o wybranym rekordzie załącznika.</span><span class="sxs-lookup"><span data-stu-id="8205e-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="8205e-119">Niektóre przyciski mogą być niedostępne, w zależności od typu i cech transakcji.</span><span class="sxs-lookup"><span data-stu-id="8205e-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="8205e-120">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="8205e-120">Close the page.</span></span>
12. <span data-ttu-id="8205e-121">Kliknij opcję Oryginalny dokument.</span><span class="sxs-lookup"><span data-stu-id="8205e-121">Click Original document.</span></span>
13. <span data-ttu-id="8205e-122">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="8205e-122">Close the page.</span></span>
