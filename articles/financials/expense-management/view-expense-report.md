# <a name="view-an-expense-report"></a><span data-ttu-id="89d17-103">Wyświetl raport z wydatków</span><span class="sxs-lookup"><span data-stu-id="89d17-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="89d17-104">Podczas wykonywania zapytania mającego wyświetlić transakcji z księgi głównej lub modułu Zaopatrzenie i sourcing może zostać zwrócona zbyt duża ilość informacji.</span><span class="sxs-lookup"><span data-stu-id="89d17-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="89d17-105">W związku warto zawęzić kryteria wyszukiwania.</span><span class="sxs-lookup"><span data-stu-id="89d17-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="89d17-106">Aby uzyskać tylko potrzebne informacje, można przejść przez kolejne poziomy szczegółów aż do transakcji, a zobaczysz wtedy raport z wydatków, w którym transakcja została pierwotnie zaraportowana.</span><span class="sxs-lookup"><span data-stu-id="89d17-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="89d17-107">W przypadku raportów z wydatków można przejść od transakcji do oryginalnego raportu zapisanego w module Zarządzanie wydatkami.</span><span class="sxs-lookup"><span data-stu-id="89d17-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="89d17-108">Oprócz łącznej kwoty wydatków można również wyświetlać poszczególne transakcje, transakcji kwalifikujące się do odzyskania podatku od towarów i usług (VAT) oraz paragony dołączone do wydatków.</span><span class="sxs-lookup"><span data-stu-id="89d17-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="89d17-109">Aby przejść przez kolejne poziomy szczegółów do oryginalnego raportu z wydatków zawierającego transakcję, zaznacz transakcję, a następnie wybierz opcję **Dokument oryginalny**.</span><span class="sxs-lookup"><span data-stu-id="89d17-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="89d17-110">Podczas wyświetlenia indywidualnych transakcji można również obejrzeć wszelkie dodatkowe dane dołączone do transakcji.</span><span class="sxs-lookup"><span data-stu-id="89d17-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="89d17-111">Na przykład dla wyszczególnionego wydatku za hotel można wyświetlić dokładną listę wszystkich kosztów, takich jak dostęp do Internetu, rozmowy telefoniczne i serwis pokojowy.</span><span class="sxs-lookup"><span data-stu-id="89d17-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
