    * <span data-ttu-id="51958-108">Zaznacz wartość 200140.</span><span class="sxs-lookup"><span data-stu-id="51958-108">Select the value 200140.</span></span>  
3. <span data-ttu-id="51958-109">W polu Do konta wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="51958-109">In the To account field, enter or select a value.</span></span>
    * <span data-ttu-id="51958-110">Zaznacz wartość 600180.</span><span class="sxs-lookup"><span data-stu-id="51958-110">Select the value 600180.</span></span>

    * <span data-ttu-id="51958-112">Zaznacz wartość Pon.</span><span class="sxs-lookup"><span data-stu-id="51958-112">Select the value Mon.</span></span>  
5. <span data-ttu-id="51958-113">W polu Od dnia wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="51958-113">In the From date field, enter a date.</span></span>
