[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="811bf-104">W tej procedurze pokazano sposób likwidacji środka trwałego przy użyciu faktury niezależnej.</span><span class="sxs-lookup"><span data-stu-id="811bf-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="811bf-105">Wybierz kolejno opcje Rozrachunki z odbiorcami > Faktury > Wszystkie faktury niezależne.</span><span class="sxs-lookup"><span data-stu-id="811bf-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="811bf-106">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="811bf-106">Click New.</span></span>
3. <span data-ttu-id="811bf-107">W polu Konto odbiorcy wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="811bf-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="811bf-108">Sprawdź poprawność domyślnej daty faktury i w razie potrzeby ją zmień.</span><span class="sxs-lookup"><span data-stu-id="811bf-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="811bf-109">Sprawdź poprawność pozostałych domyślnych pól nagłówka, takich jak Waluta, i w razie potrzeby je zmodyfikuj.</span><span class="sxs-lookup"><span data-stu-id="811bf-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="811bf-110">Wprowadź opis w wierszu faktury.</span><span class="sxs-lookup"><span data-stu-id="811bf-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="811bf-111">Wprowadź lub wybierz konto główne dla wiersza faktury.</span><span class="sxs-lookup"><span data-stu-id="811bf-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="811bf-112">Sprawdź poprawność domyślnej grupy podatków i grupy podatków dla pozycji.</span><span class="sxs-lookup"><span data-stu-id="811bf-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="811bf-113">Wprowadź cenę jednostkową lub ilość sprzedawanego środka trwałego.</span><span class="sxs-lookup"><span data-stu-id="811bf-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="811bf-114">Kliknij opcję Szczegóły wiersza.</span><span class="sxs-lookup"><span data-stu-id="811bf-114">Click Line details.</span></span>  
11. <span data-ttu-id="811bf-115">Zaznacz środek trwały przeznaczony do sprzedaży.</span><span class="sxs-lookup"><span data-stu-id="811bf-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="811bf-116">Kliknij przycisk Księguj.</span><span class="sxs-lookup"><span data-stu-id="811bf-116">Click Post.</span></span>
