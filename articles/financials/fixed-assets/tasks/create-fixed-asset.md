# <a name="create-a-fixed-asset"></a><span data-ttu-id="047b8-103">Tworzenie środka trwałego</span><span class="sxs-lookup"><span data-stu-id="047b8-103">Create a fixed asset</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="047b8-104">W tym przewodniku po zadaniach jest wykorzystywana firma demonstracyjna USMF.</span><span class="sxs-lookup"><span data-stu-id="047b8-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="047b8-105">Zostanie w nim utworzony nowy środek trwały przy użyciu strony Lista środków trwałych.</span><span class="sxs-lookup"><span data-stu-id="047b8-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="047b8-106">Przejdź do Środki trwałe > Środki trwałe > Środki trwałe.</span><span class="sxs-lookup"><span data-stu-id="047b8-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>
2. <span data-ttu-id="047b8-107">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="047b8-107">Click New.</span></span>
3. <span data-ttu-id="047b8-108">W polu Grupa środków trwałych wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="047b8-108">In the Fixed asset group field, enter or select a value.</span></span>
    * <span data-ttu-id="047b8-109">Wartość w polu Numer będzie ustawiana domyślnie, jeśli w oknie Parametry środków trwałych włączono funkcję Automatyczna numeracja środków trwałych oraz włączono funkcję Grupa środków trwałych.</span><span class="sxs-lookup"><span data-stu-id="047b8-109">The Number will default if you have enabled Autonumber fixed assets functionality in the Fixed assets parameters and the Fixed asset group.</span></span>  <span data-ttu-id="047b8-110">Jeśli nie, samodzielnie wprowadź unikatowy numer identyfikujący środek trwały.</span><span class="sxs-lookup"><span data-stu-id="047b8-110">If not, you must enter a unique number to identify the fixed asset.</span></span>  
4. <span data-ttu-id="047b8-111">W polu Nazwa wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="047b8-111">In the Name field, type a value.</span></span>
    * <span data-ttu-id="047b8-112">Wprowadź dodatkowe informacje, jakich firma potrzebuje o tym składniku aktywów.</span><span class="sxs-lookup"><span data-stu-id="047b8-112">Enter the additional information that your business needs for this asset.</span></span>  
5. <span data-ttu-id="047b8-113">Kliknij opcję Księgi.</span><span class="sxs-lookup"><span data-stu-id="047b8-113">Click Books.</span></span>
6. <span data-ttu-id="047b8-114">W polu Data nabycia wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="047b8-114">In the Acquisition date field, enter a date.</span></span>
7. <span data-ttu-id="047b8-115">W polu Cena nabycia wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="047b8-115">In the Acquisition price field, enter a number.</span></span>
    * <span data-ttu-id="047b8-116">Wprowadź dodatkowe informacje, jakich firma potrzebuje o tej księdze.</span><span class="sxs-lookup"><span data-stu-id="047b8-116">Enter the additional information that your business needs for this book.</span></span>  
    * <span data-ttu-id="047b8-117">Wprowadź dodatkowe informacje, jakich firma potrzebuje o pozostałych księgach.</span><span class="sxs-lookup"><span data-stu-id="047b8-117">Enter the additional information that your business needs for the remaining books.</span></span>  
8. <span data-ttu-id="047b8-118">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="047b8-118">Close the page.</span></span>
