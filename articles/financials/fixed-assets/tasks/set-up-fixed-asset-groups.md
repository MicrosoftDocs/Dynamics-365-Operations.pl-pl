# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="9377b-103">Ustawianie grup środków trwałych</span><span class="sxs-lookup"><span data-stu-id="9377b-103">Set up fixed asset groups</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="9377b-104">W tej procedurze pokazano sposób tworzenia nowej grupy środków trwałych.</span><span class="sxs-lookup"><span data-stu-id="9377b-104">This procedure shows how to create a new fixed asset group.</span></span> <span data-ttu-id="9377b-105">Przewodnik korzysta z roli Księgowy i danych firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="9377b-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="9377b-106">Wybierz kolejno opcje Środki trwałe > Ustawienia > Grupy środków trwałych.</span><span class="sxs-lookup"><span data-stu-id="9377b-106">Go to Fixed assets > Setup > Fixed asset groups.</span></span>
2. <span data-ttu-id="9377b-107">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="9377b-107">Click New.</span></span>
3. <span data-ttu-id="9377b-108">W polu Grupa środków trwałych wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="9377b-108">In the Fixed asset group field, type a value.</span></span>
4. <span data-ttu-id="9377b-109">W polu Nazwa wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="9377b-109">In the Name field, type a value.</span></span>
    * <span data-ttu-id="9377b-110">Ustawienia Automatyczna numeracja środków trwałych i Kod sekwencji numerów w oknie Grupa środków trwałych zastąpią ustawienia w oknie Parametry środków trwałych.</span><span class="sxs-lookup"><span data-stu-id="9377b-110">Autonumber fixed assets and Number sequence code on the Fixed asset group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="9377b-111">Można zmienić to w tym oknie, jeśli składniki aktywów z tej grupy środków trwałych będą miały inną numerację niż pozostałe grupy.</span><span class="sxs-lookup"><span data-stu-id="9377b-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="9377b-112">Kliknij opcję Księgi.</span><span class="sxs-lookup"><span data-stu-id="9377b-112">Click Books.</span></span>
6. <span data-ttu-id="9377b-113">Wprowadź lub wybierz wartość w polu Księga.</span><span class="sxs-lookup"><span data-stu-id="9377b-113">In the Book field, enter or select a value.</span></span>
    * <span data-ttu-id="9377b-114">Pole Oblicz amortyzację ma zaznaczoną opcję Tak, dlatego ewidencja środków trwałych będzie uwzględniana w propozycjach amortyzacji.</span><span class="sxs-lookup"><span data-stu-id="9377b-114">The Calculate depreciation field is set to Yes, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="9377b-115">Jeśli ustawienie Oblicz amortyzację ma wartość Nie, środek trwały nie będzie automatycznie amortyzowany.</span><span class="sxs-lookup"><span data-stu-id="9377b-115">If Calculate depreciation is set to No, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="9377b-116">Ustaw okres użytkowania składnika aktywów w latach.</span><span class="sxs-lookup"><span data-stu-id="9377b-116">Set the Service life of the asset, in years.</span></span>
    * <span data-ttu-id="9377b-117">Należy zauważyć, że wartość pola Okresy amortyzacji jest obliczana po ustawieniu okresu użytkowania.</span><span class="sxs-lookup"><span data-stu-id="9377b-117">Note that the Depreciation periods field value is calculated after setting the Service life.</span></span>
