# <a name="configure-the-global-address-book"></a><span data-ttu-id="80a32-103">Konfigurowanie globalnej książki adresowej</span><span class="sxs-lookup"><span data-stu-id="80a32-103">Configure the global address book</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="80a32-104">Ta procedura służy do ustawiania domyślnych wartości i zasad zabezpieczeń dla globalnej książki adresowej.</span><span class="sxs-lookup"><span data-stu-id="80a32-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="80a32-105">Dane wykorzystane do stworzenia tego zadania pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="80a32-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="80a32-106">Zadanie jest przeznaczone dla zespołu ds. planowania i konfiguracji.</span><span class="sxs-lookup"><span data-stu-id="80a32-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="80a32-107">Wybierz kolejno opcje Administrowanie organizacją > Globalna książka adresowa > Parametry globalnej książki adresowej.</span><span class="sxs-lookup"><span data-stu-id="80a32-107">Go to Organization administration > Global address book > Global address book parameters.</span></span>
2. <span data-ttu-id="80a32-108">W polu Kolejność w nazwisku wybierz, jak mają być pokazywane imiona i nazwiska.</span><span class="sxs-lookup"><span data-stu-id="80a32-108">In the Name sequence field, select how names should be shown.</span></span>
3. <span data-ttu-id="80a32-109">Określ, czy mają być usuwane strony, którym nie przypisano ról.</span><span class="sxs-lookup"><span data-stu-id="80a32-109">Select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="80a32-110">Określ, czy ma być sprawdzane istnienie duplikatów rekordów.</span><span class="sxs-lookup"><span data-stu-id="80a32-110">Select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="80a32-111">Określ, czy w adresach mają być wyświetlane numery DUNS.</span><span class="sxs-lookup"><span data-stu-id="80a32-111">Select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="80a32-112">Określ, czy ma być sprawdzane istnienie unikatowych numerów DUNS.</span><span class="sxs-lookup"><span data-stu-id="80a32-112">Select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="80a32-113">W polu Strona wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="80a32-113">In the Party field, select an option.</span></span>
8. <span data-ttu-id="80a32-114">W polu Odbiorca wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="80a32-114">In the Customer field, select an option.</span></span>
9. <span data-ttu-id="80a32-115">W polu Dostawca wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="80a32-115">In the Vendor field, select an option.</span></span>
10. <span data-ttu-id="80a32-116">W polu Potencjalny klient wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="80a32-116">In the Prospect field, select an option.</span></span>
11. <span data-ttu-id="80a32-117">W polu Konkurent wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="80a32-117">In the Competitor field, select an option.</span></span>
12. <span data-ttu-id="80a32-118">Kliknij kartę Zabezpieczenia lokalizacji prywatnych.</span><span class="sxs-lookup"><span data-stu-id="80a32-118">Click the Private location security tab.</span></span>
13. <span data-ttu-id="80a32-119">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="80a32-119">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="80a32-120">Naciśnij klawisz Shift, aby zaznaczyć wiele ról, które mają zostać dodane do okienka Wybrane role, a następnie kliknij strzałkę, aby dodać wybrane role.</span><span class="sxs-lookup"><span data-stu-id="80a32-120">Press the Shift key to select multiple roles to add to the Selected roles pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="80a32-121">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="80a32-121">Click Save.</span></span>
