<span data-ttu-id="1489a-104">Można skonfigurować system, aby wysyłał wiadomości e-mail do użytkowników w przypadku wystąpienia zdarzeń związanych z przepływem pracy.</span><span class="sxs-lookup"><span data-stu-id="1489a-104">You can configure the system to send email messages to users when workflow-related events occur.</span></span> <span data-ttu-id="1489a-105">Na przykład, wiadomości e-mail mogą być wysyłane do użytkowników po przypisaniu do nich dokumentów do zatwierdzenia.</span><span class="sxs-lookup"><span data-stu-id="1489a-105">For example, email messages can be sent to users when documents are assigned to them for approval.</span></span> <span data-ttu-id="1489a-106">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="1489a-106">The demo data company used to create this procedure is USMF.</span></span>

4. <span data-ttu-id="1489a-110">Kliknij kartę Przepływ pracy.</span><span class="sxs-lookup"><span data-stu-id="1489a-110">Click the Workflow tab.</span></span>

    * <span data-ttu-id="1489a-116">Jeśli chcesz, aby użytkownik otrzymywał powiadomienia na kliencie, zaznacz pole wyboru Wysyłaj powiadomienia pocztą e-mail.</span><span class="sxs-lookup"><span data-stu-id="1489a-116">If you want the user to receive notifications in the client, select the Send notifications in email check box.</span></span>
