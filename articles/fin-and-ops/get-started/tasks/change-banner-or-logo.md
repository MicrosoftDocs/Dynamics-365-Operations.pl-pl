# <a name="change-the-banner-or-logo"></a><span data-ttu-id="1489f-103">Zmienianie transparentu lub logo</span><span class="sxs-lookup"><span data-stu-id="1489f-103">Change the banner or logo</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="1489f-104">Następująca procedura wymienia kroki, jakie administratorzy systemu mogą stosować do aktualizowania obrazu transparentu lub logo wyświetlanego dla firmy.</span><span class="sxs-lookup"><span data-stu-id="1489f-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="1489f-105">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="1489f-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="1489f-106">Wybierz kolejno opcje Administrowanie organizacją > Organizacje > Firmy.</span><span class="sxs-lookup"><span data-stu-id="1489f-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="1489f-107">Na liście po lewej stronie wybierz firmę, dla której chcesz zaktualizować transparent lub logo.</span><span class="sxs-lookup"><span data-stu-id="1489f-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="1489f-108">Jeśli firma jest zaznaczona, przejdź do następnego kroku.</span><span class="sxs-lookup"><span data-stu-id="1489f-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="1489f-109">Rozwiń sekcję Obrazy.</span><span class="sxs-lookup"><span data-stu-id="1489f-109">Expand the Images section.</span></span> <span data-ttu-id="1489f-110">Jeśli sekcja jest już rozwinięta, przejdź do następnego kroku.</span><span class="sxs-lookup"><span data-stu-id="1489f-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="1489f-111">Kliknij przycisk Edytuj.</span><span class="sxs-lookup"><span data-stu-id="1489f-111">Click Edit.</span></span>
5. <span data-ttu-id="1489f-112">W polu Typ obrazu firmy na pulpit nawigacyjny wybierz opcję „Transparent”.</span><span class="sxs-lookup"><span data-stu-id="1489f-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="1489f-113">Kliknij przycisk Zmień.</span><span class="sxs-lookup"><span data-stu-id="1489f-113">Click Change.</span></span>
    * <span data-ttu-id="1489f-114">Porada: Idealną rozdzielczością obrazu transparentu 1920 x 281 pikseli.</span><span class="sxs-lookup"><span data-stu-id="1489f-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="1489f-115">Idealna szerokość logo wynosi 350 pikseli.</span><span class="sxs-lookup"><span data-stu-id="1489f-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="1489f-116">Po zakończeniu przekazywania nowy transparent lub logo powinno być widoczne w sekcji Obraz na pulpit nawigacyjny.</span><span class="sxs-lookup"><span data-stu-id="1489f-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="1489f-117">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="1489f-117">Click Save.</span></span>
8. <span data-ttu-id="1489f-118">Wybierz kolejno opcje Wspólne > Wspólne > Domyślny pulpit nawigacyjny.</span><span class="sxs-lookup"><span data-stu-id="1489f-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="1489f-119">Nowy transparent lub logo powinno być widoczne na pulpicie nawigacyjnym.</span><span class="sxs-lookup"><span data-stu-id="1489f-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="1489f-120">Może być konieczna zmiana firmy, za pomocą wybieraka firmy, na firmę, do której przekazano transparent.</span><span class="sxs-lookup"><span data-stu-id="1489f-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
