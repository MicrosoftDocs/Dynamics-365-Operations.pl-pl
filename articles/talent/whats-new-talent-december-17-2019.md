# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Nowości i zmiany w rozwiązaniu Dynamics 365 Talent (17 grudnia 2019 r.)

W tym artykule opisano nowe oraz zmienione funkcje dostępne w Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Zmiany w Attract

Ta wersja zawiera drobne poprawki błędów dla rozwiązania Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Zmiany w Onboard

Ta wersja zawiera drobne poprawki błędów dla rozwiązania Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Zmiany w Core HR

Zmiany opisane w tej części dotyczą kompilacji 8.1.2674. Liczby w nawiasach w niektórych nagłówkach odnoszą się do numerów pomocy w usługach Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Drukowanie przeglądów wydajności

Drukowanie przeglądów wydajności jest teraz dostępne we wszystkich środowiskach. Aby włączyć tę funkcję, należy użyć obszaru roboczego **Zarządzanie funkcjami**. Aby uzyskać więcej informacji o zarządzaniu funkcjami funkcji, zapoznaj się z [Zarządzanie funkcjami — omówienie](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Aby uzyskać więcej informacji na temat drukowania przeglądów wydajności zobacz [Przeglądy wydajności](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Wycofana pozycja z bezpośrednimi raportami „pływa” po schemacie organizacyjnym jako pozycja otwarta (385436)

W wersji z tego tygodnia skorygowano sposób wyświetlania wycofanych pozycji z raportami bezpośrednimi w hierarchii raportowania.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>System nie może wygenerować dni kalendarzowych (381195) 

Zmiany z tego tygodnia rozwiązują problem polegający na tym, że nie można wygenerować dni kalendarzowych.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Aktualny raport salda urlopów PBI podwaja saldo pracowników (385581)

Ta zmiana koryguje bieżący raport salda w tym scenariuszu.
