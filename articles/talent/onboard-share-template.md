[!include [banner](includes/banner.md)]

<span data-ttu-id="9b3a8-104">Microsoft Dynamics 365 Talent: Onboard pozwala udostępniać przewodniki wdrażania do pracy i szablony z innymi współautorami, tak aby mogli przeglądać dokumenty i wprowadzać zmiany.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="9b3a8-105">Udostępnianie przewodnika wdrażania do pracy innym współautorom</span><span class="sxs-lookup"><span data-stu-id="9b3a8-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="9b3a8-106">W menu po lewej stronie wybierz **Przewodniki**, a następnie wybierz przewodnik, który chcesz udostępnić.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="9b3a8-107">Wybierz przycisk **Więcej** (symbol trzech kropek w pionie) w prawym górnym rogu strony, a następnie wybierz opcję **Udostępnij**.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="9b3a8-108">Udostępnianie przewodnika wdrażania do pracy innym współautorom</span><span class="sxs-lookup"><span data-stu-id="9b3a8-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="9b3a8-109">W obszarze **Udostępnij**, jeśli odbiorcy powinni mieć możliwość zmiany przewodnika, ustaw te osoby, ustaw opcję **Czy te osoby mogą wprowadzać zmiany?** na **Włączona**.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="9b3a8-110">Jeśli odbiorcy powinni zostać powiadomieni pocztą elektroniczną, że udostępniłeś im przewodnik , ustaw opcję **Wyślij powiadomienie e-mail do adresatów** na **Włączona**.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="9b3a8-111">Ustawianie dostępu i preferencji e-mail dla przewodnika wdrażania do pracy</span><span class="sxs-lookup"><span data-stu-id="9b3a8-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="9b3a8-112">Wprowadź nazwę lub adres e-mail i wybierz odbiorcę z listy rozwijanej.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="9b3a8-113">Na karcie **Zarządzaj dostępem** można przejrzeć lub zmienić, co odbiorcy mogą zrobić z przewodnikiem.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="9b3a8-114">Aby zmienić dostęp odbiorcy, na liście rozwijanej obok nazwy adresata wybierz opcję **Można wyświetlić**, **Można edytować** lub **Usuń**, stosownie do potrzeb.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="9b3a8-115">Po zakończeniu dodawania lub modyfikowania adresatów wybierz pozycję **Aktualizuj**.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

## <a name="share-an-onboarding-template-with-other-contributors"></a><span data-ttu-id="9b3a8-116">Udostępnianie szablonu wdrażania do pracy innym współautorom</span><span class="sxs-lookup"><span data-stu-id="9b3a8-116">Share an onboarding template with other contributors</span></span>

2. <span data-ttu-id="9b3a8-118">W obszarze **Moje szablony**, wybierz przycisk **Więcej** (wielokropkek \[**...**\]) w prawym górnym rogu szablonu, który chcesz udostępnić, a następnie wybierz pozycję **Udostępnij**.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-118">Under **My templates**, select the **More** button (the ellipsis \[**...**\]) in the upper-right corner of the template that you want to share, and then select **Share**.</span></span>

    [!Ustawianie dostępu i preferencji e-mail dla szablonu]<span data-ttu-id="9b3a8-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="9b3a8-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="9b3a8-123">Wprowadź nazwę lub adres e-mail i wybierz odbiorcę z listy rozwijanej.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="9b3a8-124">Na karcie **Zarządzaj dostępem** można przejrzeć lub zmienić, co odbiorcy mogą zrobić z szablonem.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-124">On the **Manage access** tab, you can review or change what recipients can do with the template.</span></span> <span data-ttu-id="9b3a8-125">Aby zmienić dostęp odbiorcy, na liście rozwijanej obok nazwy adresata wybierz opcję **Można wyświetlić**, **Można edytować** lub **Usuń**, stosownie do potrzeb.</span><span class="sxs-lookup"><span data-stu-id="9b3a8-125">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
