# <a name="get-the-dynamics-365-talent---onboard-app"></a>Pobieranie aplikacji Dynamics 365 Talent - Onboard

[!include [banner](includes/banner.md)]

Możesz zobaczyć demo i wypróbować aplikację Microsoft Dynamics 365 Talent: Onboard za darmo ze strony [Strona produktu Onboard](https://dynamics.microsoft.com/talent/onboard/).

> [!NOTE]
> Bezpłatna wersja próbna wymaga biznesowego konta e-mail.

Możesz zakupić subskrypcję Onboard jako samodzielna aplikacja lub część Dynamics 365 Talent. Talent to kompleksowy system zarządzania kapitałem ludzkim (HCM), który obejmuje Dynamics 365 Talent: Attract, Onboard i Core HR. Aby uzyskać więcej informacji na temat zakupu Onboard, zobacz [Strona produktu Onboard](https://dynamics.microsoft.com/talent/onboard/).

Podczas okresu testowego lub zakupu zostanie skonfigurowany adres e-mail i hasło Microsoft 365. Pamiętaj, aby zanotować te wartości.

> [!WARNING]
> Nie można migrować danych z wersji próbnej do płatnego środowiska subskrypcji. <!--Reviewers: please verify.-->

Aby uzyskać więcej informacji o nowych funkcjach w aplikacji Talent, zobacz [Nowości i zmiany w aplikacji Dynamics 365 Talent](./whats-new.md) oraz [Informacje o nowej wersji Dynamics 365 i Power Platform](https://docs.microsoft.com/business-applications-release-notes/index). Jeśli chcesz wyświetlić podgląd nowych funkcji w Onboard, zobacz [Dostęp do funkcji w wersji zapoznawczej w Microsoft Dynamics 365 Talent](./access-preview-feature.md).

Jeśli jesteś specjalistą IT i chcesz dowiedzieć się więcej o tym, jak dwie wersje aplikacji Onboard są udostępniane, zobacz [Inicjowanie obsługi administracyjnej dla aplikacji Dynamics 365 Talent - Onboard](./modular-app-tech-faq.md).

## <a name="get-started-with-onboard"></a>Rozpoczęcie korzystania z rozwiązania Onboard

Kiedy otwierasz Onboard po raz pierwszy, jesteś zaproszony do rozpoczęcia zwiedzania centrum administracyjnego Microsoft 365. Centrum administracyjne to miejsce, w którym można skonfigurować organizację, zarządzać użytkownikami i zarządzać subskrypcjami. (Jedną z tych subskrypcji jest subskrypcja Onboard.) Aby uzyskać więcej informacji na temat centrum administracyjnego Microsoft 365, zobacz [Informacje o centrum administracyjnym Microsoft 365](https://docs.microsoft.com/office365/admin/admin-overview/about-the-admin-center?view=o365-worldwide).

Aby dostać się do aplikacji Onboard, wykonaj następujące kroki.

1. Otwórz [stronę logowania firmy Microsoft](https://portal.office.com/).
2. Gdy zostanie wyświetlony monit, wprowadź adres e-mail i hasło do Microsoft 365.
3. Wybierz program uruchamiający aplikacje w lewym górnym rogu, a następnie wybierz pozycję **Dynamics 365**.

    [![Uzyskiwanie dostępu do Dynamics 365](./media/onboard-start-dynamics365.png)](./media/onboard-start-dynamics365.png)

4. Wybierz płytkę **Talent: Onboard**.

    [![Otwieranie Onboard](./media/onboard-start-onboard.png)](./media/onboard-start-onboard.png)

Pierwsze logowanie może potrwać kilka minut, ponieważ środowisko musi być zainicjowane.

## <a name="try-the-walkthrough"></a>Wypróbuj przewodnik

Po pierwszym otwarciu Onboard, można wybrać **Uruchom przewodnik**, aby rozpocząć pracę z szablonem roboczym.

Jeśli pominiesz przewodnik, można uzyskać do niego dostęp później, wybierając przycisk **Pomoc** (**?**), a następnie wybierając **Rozpoczynanie pracy**.

![[Rozpoczęcie przewodnika Onboard](./media/onboard-start-walkthrough.png)](./media/onboard-start-walkthrough.png)

## <a name="change-the-domain-name"></a>Zmienianie nazwy domeny

Jeśli zaakceptowano domyślną nazwę domeny po zalogowaniu się w Onboard, można ją później zmienić na inną domenę. (Domyślna nazwa domeny kończy się w **onmicrosoft.com**.)

1. Otwórz [stronę logowania firmy Microsoft](https://portal.office.com/).
2. Gdy zostanie wyświetlony monit, wprowadź adres e-mail i hasło do Microsoft 365.
3. Jeśli zostanie wyświetlona rekomendacja, aby dodać własną domenę w obszarze **Zalecane dla Ciebie**, wybierz pozycję **Wyświetl rekomendację** i postępuj zgodnie z monitami. Jeśli nie widzisz rekomendacji, wybierz pozycję **Pokaż wszystko** w menu po lewej stronie, wybierz pozycję **Ustawienia**, wybierz pozycję **Domeny**, a następnie wybierz pozycję **Dodaj domenę** lub **Kup domenę**. Następnie postępuj zgodnie z wyświetlanymi monitami.
