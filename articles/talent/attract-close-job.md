## <a name="cancel-a-job"></a><span data-ttu-id="46a82-107">Anuluj pracę</span><span class="sxs-lookup"><span data-stu-id="46a82-107">Cancel a job</span></span>

<span data-ttu-id="46a82-108">Ogłoszenie o pracę w Attract można usunąć w dowolnym momencie.</span><span class="sxs-lookup"><span data-stu-id="46a82-108">You can cancel a job in Attract at any time.</span></span>

1. <span data-ttu-id="46a82-109">Na karcie **Prace** w Attract wybierz ofertę pracy, którą chcesz anulować.</span><span class="sxs-lookup"><span data-stu-id="46a82-109">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

   ![Zamykanie pracy w Attract](./media/attract-close-job.png)

3. <span data-ttu-id="46a82-112">W oknie dialogowym **Zamknij tę pracę** zadanie wybierz pozycję **Praca anulowana**, dodaj komentarze, a następnie wybierz pozycję **Zamknij pracę**.</span><span class="sxs-lookup"><span data-stu-id="46a82-112">In the **Close this job** dialog box, select **Job cancelled**, add any comments, and then select **Close job**.</span></span>

   ![Zamknij pracę jako Stanowisko obsadzone](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a><span data-ttu-id="46a82-114">Zamknij pracę jako Stanowisko obsadzone</span><span class="sxs-lookup"><span data-stu-id="46a82-114">Close a job as Position filled</span></span>

<span data-ttu-id="46a82-115">Można zamknąć pracę jako **Stanowisko obsadzone** po ukończeniu przez co najmniej jedną relację biznesową etapu oferty.</span><span class="sxs-lookup"><span data-stu-id="46a82-115">You can close a job as **Position filled** after at least one prospect has completed the Offer stage.</span></span>

2. <span data-ttu-id="46a82-117">Kliknij przycisk **Zamknij pracę**.</span><span class="sxs-lookup"><span data-stu-id="46a82-117">Select **Close job**.</span></span>

   ![Zamykanie pracy w Attract](./media/attract-close-job.png)

3. <span data-ttu-id="46a82-119">W oknie dialogowym **Zamknij tę pracę** wybierz **Stanowisko obsadzone**, wybierz kandydata, który zaakceptował ofertę, Dodaj komentarze, a następnie wybierz pozycję **Zamknij pracę.**</span><span class="sxs-lookup"><span data-stu-id="46a82-119">In the **Close this job** dialog box, select **Position filled**, select the candidate who accepted the offer, add any comments, and then select **Close job**.</span></span>

   > [!NOTE]
   > <span data-ttu-id="46a82-121">Można zamknąć pracę jako **Stanowisko obsadzone** tylko po ukończeniu przez co najmniej jedną relację biznesową etapu oferty.</span><span class="sxs-lookup"><span data-stu-id="46a82-121">You can only close a job as **Position filled** when at least one prospect has completed the Offer stage.</span></span> <span data-ttu-id="46a82-122">Jeśli dla podania o pracę nie wybrano **Oferta zaakceptowana** lub jeśli w aplikacji nie jest wyświetlany stan **Zaakceptowana** w ofercie, opcja **Stanowisko obsadzone** nie będzie dostępna w liście rozwijanej **Powód**.</span><span class="sxs-lookup"><span data-stu-id="46a82-122">If the job application doesn't have **Offer accepted** selected, or if the application doesn't display a status of **Accepted** in Offer, the **Position filled** option won't display in the **Reason** dropdown.</span></span>
