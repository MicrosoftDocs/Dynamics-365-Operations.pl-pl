# <a name="welcome-to-dynamics-365-talent"></a>Dynamics 365 Talent — Zapraszamy!

[!include [banner](includes/banner.md)]

Rozwiązanie Talent zawiera aplikację Attract, która pomaga wyszukiwać i zatrudniać osoby o odpowiednich kwalifikacjach dla Twojej organizacji. W procesie zatrudniania pomaga aplikacja Onboard, dzięki której łatwiej wprowadzisz nowych pracowników etatowych do organizacji. Pozwala wyznaczyć precyzyjne cele, przekazywać niezbędne informacje, zapewnia łączność z innymi pracownikami i jest znakomitym punktem wejścia do nowej pracy.  

- [Microsoft Dynamics 365 Talent — funkcje i możliwości aplikacji Attract](attract-overview.md)
- [Rozpoczęcie korzystania z rozwiązania Onboard](create-onboarding-experience.md)

> [!IMPORTANT]
> Ta funkcja w aplikacji Dynamics 365 Talent: Core HR jest obecnie częścią aplikacji Dynamics 365 Human Resources. Możesz przejrzeć dokumentację aplikacji Human Resources pod adresem [Dynamics 365 Human Resources — Zapraszamy!](https://docs.microsoft.com/dynamics365/human-resources/hr-welcome).
