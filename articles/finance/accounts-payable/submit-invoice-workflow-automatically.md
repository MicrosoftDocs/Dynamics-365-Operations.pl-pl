<span data-ttu-id="1304b-119">Proces dopasowania będzie działał, dopóki dopasowana ilość z dokumentu przyjęcia produktów nie będzie równa ilości fakturowanej.</span><span class="sxs-lookup"><span data-stu-id="1304b-119">The matching process will run until the matched product receipt quantity equals the invoice quantity.</span></span> <span data-ttu-id="1304b-120">Jeśli jednak istnieje wiele dokumentów przyjęcia produktów dla jednego wiersza faktury, musisz kilka razy uruchomić ten proces, aby uzyskać pełne dopasowanie ilości.</span><span class="sxs-lookup"><span data-stu-id="1304b-120">However, if there are multiple product receipts for a single invoice line, you’ll need to run the process multiple times to achieve the full quantity match.</span></span> <span data-ttu-id="1304b-121">Można określić maksymalną liczbę prób dopasowania przez systemu przyjęcia produktów do wiersza faktury przed poinformowaniem o niepowodzeniu procesu.</span><span class="sxs-lookup"><span data-stu-id="1304b-121">You can specify the maximum number of times that the system should try to match product receipts to an invoice line before it concludes that the process failed.</span></span> <span data-ttu-id="1304b-122">Proces będzie uruchamiany w tle, co godzinę lub codziennie.</span><span class="sxs-lookup"><span data-stu-id="1304b-122">The process will run in the background, either hourly or daily.</span></span> 

<span data-ttu-id="1304b-123">Proces automatycznego dopasowywania można uruchomić jako część procesu przesyłania faktur do systemu przepływu pracy.</span><span class="sxs-lookup"><span data-stu-id="1304b-123">You can run the automated matching process as part of the process for submitting invoices to the workflow system.</span></span> <span data-ttu-id="1304b-124">Alternatywnie można uruchomić go jako proces autonomiczny.</span><span class="sxs-lookup"><span data-stu-id="1304b-124">Alternatively, you can run it as a standalone process.</span></span> <span data-ttu-id="1304b-125">Ustawienia dopasowywania przyjęć produktów do wierszy faktur można skonfigurować na karcie **Automatyzacja faktur od dostawcy** strony **Parametry rozrachunków z dostawcami** (**Rozrachunki z dostawcami \> Konfiguracja \> Parametry rozrachunków z dostawcami**).</span><span class="sxs-lookup"><span data-stu-id="1304b-125">The settings for the match-product-receipts-to-invoice-lines process are configured on the **Vendor invoice automation** tab of the **Accounts payable parameters** page (**Accounts payable \> Setup \> Accounts payable parameters**).</span></span>

<span data-ttu-id="1304b-131">Wiersz faktury zostanie wyłączony z automatycznego przetwarzania jeśli spełniony jest dowolny z następujących warunków:</span><span class="sxs-lookup"><span data-stu-id="1304b-131">An invoice line will be excluded from the automated processing if any of the following conditions are met:</span></span>

- <span data-ttu-id="1304b-134">Faktura znajduje się w systemie przepływu pracy.</span><span class="sxs-lookup"><span data-stu-id="1304b-134">The invoice is in the workflow system.</span></span>
