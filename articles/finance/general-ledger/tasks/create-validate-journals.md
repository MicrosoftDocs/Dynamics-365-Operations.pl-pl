[!include [banner](../../includes/banner.md)]

<span data-ttu-id="75afa-104">W tej procedurze zostaną utworzone i zweryfikowane arkusze i wiersze arkuszy.</span><span class="sxs-lookup"><span data-stu-id="75afa-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="75afa-105">Procedurę można wykonać przy użyciu danych firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="75afa-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="75afa-106">Wybierz kolejno opcje **Księga główna > Wpisy w arkuszu > Arkusze finansowe**.</span><span class="sxs-lookup"><span data-stu-id="75afa-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="75afa-107">Kliknij przycisk **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="75afa-107">Click **New**.</span></span>
3. <span data-ttu-id="75afa-108">W polu **Nazwa** kliknij rozwijany przycisk, aby otworzyć wyszukiwanie.</span><span class="sxs-lookup"><span data-stu-id="75afa-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="75afa-109">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="75afa-109">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="75afa-110">Kliknij przycisk **Wiersze**.</span><span class="sxs-lookup"><span data-stu-id="75afa-110">Click **Lines**.</span></span>
6. <span data-ttu-id="75afa-111">W polu **Konto** wprowadź odpowiednie konto zgodnie z typem konta.</span><span class="sxs-lookup"><span data-stu-id="75afa-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>
7. <span data-ttu-id="75afa-112">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="75afa-112">In the **Description** field, type a value.</span></span>

10. <span data-ttu-id="75afa-115">Kliknij **Potwierdź**.</span><span class="sxs-lookup"><span data-stu-id="75afa-115">Click **Validate**.</span></span>
11. <span data-ttu-id="75afa-116">Kliknij **Potwierdź**.</span><span class="sxs-lookup"><span data-stu-id="75afa-116">Click **Validate**.</span></span>
12. <span data-ttu-id="75afa-117">Kliknij przycisk **Księguj**.</span><span class="sxs-lookup"><span data-stu-id="75afa-117">Click **Post**.</span></span>
13. <span data-ttu-id="75afa-118">Kliknij opcję **Załącznik**.</span><span class="sxs-lookup"><span data-stu-id="75afa-118">Click **Voucher**.</span></span>
