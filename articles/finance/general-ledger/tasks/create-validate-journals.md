# <a name="create-and-validate-journals"></a><span data-ttu-id="865c8-103">Tworzenie i sprawdzanie arkuszy</span><span class="sxs-lookup"><span data-stu-id="865c8-103">Create and validate journals</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="865c8-104">W tej procedurze zostaną utworzone i zweryfikowane arkusze i wiersze arkuszy.</span><span class="sxs-lookup"><span data-stu-id="865c8-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="865c8-105">Procedurę można wykonać przy użyciu danych firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="865c8-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="865c8-106">Wybierz kolejno opcje **Księga główna > Wpisy w arkuszu > Arkusze finansowe**.</span><span class="sxs-lookup"><span data-stu-id="865c8-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="865c8-107">Kliknij przycisk **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="865c8-107">Click **New**.</span></span>
3. <span data-ttu-id="865c8-108">W polu **Nazwa** kliknij rozwijany przycisk, aby otworzyć wyszukiwanie.</span><span class="sxs-lookup"><span data-stu-id="865c8-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="865c8-109">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="865c8-109">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="865c8-110">Kliknij przycisk **Wiersze**.</span><span class="sxs-lookup"><span data-stu-id="865c8-110">Click **Lines**.</span></span>
6. <span data-ttu-id="865c8-111">W polu **Konto** wprowadź odpowiednie konto zgodnie z typem konta.</span><span class="sxs-lookup"><span data-stu-id="865c8-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>
7. <span data-ttu-id="865c8-112">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="865c8-112">In the **Description** field, type a value.</span></span>
8. <span data-ttu-id="865c8-113">Wprowadź kwotę na koncie po stronie **Debet** lub **Kredyt**.</span><span class="sxs-lookup"><span data-stu-id="865c8-113">Enter an amount for the account as either a **Debit** or **Credit**.</span></span> 
9. <span data-ttu-id="865c8-114">W polu **Konto przeciwstawne** wprowadź odpowiednie konto zgodnie z typem konta przeciwstawnego.</span><span class="sxs-lookup"><span data-stu-id="865c8-114">In the **Offset account** field, enter an appropriate account based on the Offset account type.</span></span>
10. <span data-ttu-id="865c8-115">Kliknij **Potwierdź**.</span><span class="sxs-lookup"><span data-stu-id="865c8-115">Click **Validate**.</span></span>
11. <span data-ttu-id="865c8-116">Kliknij **Potwierdź**.</span><span class="sxs-lookup"><span data-stu-id="865c8-116">Click **Validate**.</span></span>
12. <span data-ttu-id="865c8-117">Kliknij przycisk **Księguj**.</span><span class="sxs-lookup"><span data-stu-id="865c8-117">Click **Post**.</span></span>
13. <span data-ttu-id="865c8-118">Kliknij opcję **Załącznik**.</span><span class="sxs-lookup"><span data-stu-id="865c8-118">Click **Voucher**.</span></span>
