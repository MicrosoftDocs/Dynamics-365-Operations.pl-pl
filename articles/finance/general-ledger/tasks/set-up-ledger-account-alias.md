# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="eddd7-103">Konfigurowanie aliasu konta księgowego</span><span class="sxs-lookup"><span data-stu-id="eddd7-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="eddd7-104">W tej procedurze pokazano, jak utworzyć alias konta, który zawiera skrót do wprowadzania numeru konta.</span><span class="sxs-lookup"><span data-stu-id="eddd7-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="eddd7-105">Procedura wykorzystuje dane firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="eddd7-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="eddd7-106">Wybierz kolejno opcje Księga główna > Plan kont > Konta > Alias konta księgowego.</span><span class="sxs-lookup"><span data-stu-id="eddd7-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="eddd7-107">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="eddd7-107">Click New.</span></span>
3. <span data-ttu-id="eddd7-108">W polu Alias konta księgowego wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="eddd7-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="eddd7-109">W polu Struktura konta wybierz strukturę, do której należy konto i wymiary.</span><span class="sxs-lookup"><span data-stu-id="eddd7-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="eddd7-110">W polu Firma kliknij przycisk rozwijany, aby otworzyć wyszukiwanie.</span><span class="sxs-lookup"><span data-stu-id="eddd7-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="eddd7-111">Na liście odszukaj i zaznacz firmę, do której stosuje się alias.</span><span class="sxs-lookup"><span data-stu-id="eddd7-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="eddd7-112">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="eddd7-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="eddd7-113">W polu Definicja aliasu konta księgowego określ konto i wymiary.</span><span class="sxs-lookup"><span data-stu-id="eddd7-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="eddd7-114">Użycie skrótu spowoduje wypełnienie pól danych konta i wymiarów.</span><span class="sxs-lookup"><span data-stu-id="eddd7-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="eddd7-115">W polu Początkowy fokus wybierz wymiar, który otrzyma fokus, jeśli jest używany alias.</span><span class="sxs-lookup"><span data-stu-id="eddd7-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="eddd7-116">Po wpisaniu skrótu oraz wypełnieniu pól danych konta i wymiarów kursor znajdzie się w polu Początkowy fokus.</span><span class="sxs-lookup"><span data-stu-id="eddd7-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
