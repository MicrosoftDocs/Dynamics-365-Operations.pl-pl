# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="fe610-103">Obsługa domyślnych szablonów wymiarów finansowych</span><span class="sxs-lookup"><span data-stu-id="fe610-103">Maintain financial dimension default templates</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="fe610-104">W tym temacie pokazano sposób konfigurowania domyślnych szablonów wymiarów finansowych.</span><span class="sxs-lookup"><span data-stu-id="fe610-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="fe610-105">Szablony mogą służyć do szybkiego rozdziału kwot w dokumentach źródłowych.</span><span class="sxs-lookup"><span data-stu-id="fe610-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="fe610-106">W okienku nawigacji wybierz kolejno opcje **Moduły > Księga główna > Plan kont > Wymiary > Domyślne szablony wymiaru finansowego**.</span><span class="sxs-lookup"><span data-stu-id="fe610-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>
2. <span data-ttu-id="fe610-107">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="fe610-107">Select **New**.</span></span>
3. <span data-ttu-id="fe610-108">W polu **Identyfikator szablonu** wprowadź identyfikator szablonu.</span><span class="sxs-lookup"><span data-stu-id="fe610-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="fe610-109">Wprowadź wartość procentową dla wartości pierwszego wymiaru.</span><span class="sxs-lookup"><span data-stu-id="fe610-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="fe610-110">Wprowadź wartości wymiarów finansowych odpowiadające wprowadzonej wartości procentowej.</span><span class="sxs-lookup"><span data-stu-id="fe610-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="fe610-111">Wybierz opcję **Dodaj**.</span><span class="sxs-lookup"><span data-stu-id="fe610-111">Select **Add**.</span></span>
7. <span data-ttu-id="fe610-112">Wprowadź kolejną wartość procentową dla wartości następnego wymiaru.</span><span class="sxs-lookup"><span data-stu-id="fe610-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="fe610-113">W tym przykładzie wprowadź wartość procentową, tak aby mieć przydzielone niecałe 100%.</span><span class="sxs-lookup"><span data-stu-id="fe610-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="fe610-114">Wprowadź wartości wymiarów finansowych odpowiadające wprowadzonej wartości procentowej.</span><span class="sxs-lookup"><span data-stu-id="fe610-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="fe610-115">Wybierz opcję **Dodaj**.</span><span class="sxs-lookup"><span data-stu-id="fe610-115">Select **Add**.</span></span>
10. <span data-ttu-id="fe610-116">Wprowadź inną wartość procentową, która sprawi, że całkowita wartość procentowa będzie równa 100%.</span><span class="sxs-lookup"><span data-stu-id="fe610-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="fe610-117">Wprowadź wartości wymiarów finansowych odpowiadające wprowadzonej wartości procentowej.</span><span class="sxs-lookup"><span data-stu-id="fe610-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="fe610-118">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="fe610-118">Select **Save**.</span></span>
