# <a name="create-a-main-account"></a><span data-ttu-id="2b2f7-103">Tworzenie konta głównego</span><span class="sxs-lookup"><span data-stu-id="2b2f7-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2b2f7-104">To zadanie prowadzi przez proces dodawania konta głównego do istniejącego planu kont.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="2b2f7-105">To nagranie wykorzystuje firmę demonstracyjną USMF.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="2b2f7-106">Wybierz kolejno **Okienko nawigacji > Moduły > Księga główna > Plan kont > Konta > Konta główne**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="2b2f7-107">Kliknij przycisk **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-107">Click **New**.</span></span>
3. <span data-ttu-id="2b2f7-108">W polu **Konto główne** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="2b2f7-109">W polu **Nazwa** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="2b2f7-110">W polu **Typ konta głównego** wybierz typ, który najlepiej odzwierciedla salda i lokalizacje kont w sprawozdaniach finansowych.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="2b2f7-111">Na liście zaznacz kategorię kont, do której należy konto główne.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="2b2f7-112">Kategoria konta jest używana w domyślnych raportach finansowych i zawartości pulpitu nawigacyjnego narzędzia Power BI.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="2b2f7-113">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="2b2f7-114">Zmień domyślne saldo debetowe lub kredytowe.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="2b2f7-115">W polu **Waluta domyślna** wybierz wartość z listy walut.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="2b2f7-116">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="2b2f7-117">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="2b2f7-118">Przełącz rozwinięcie sekcji **Firma zastępuje**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="2b2f7-119">Kliknij przycisk **Dodaj** i wybierz firmę.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="2b2f7-120">Na liście zaznacz firmę.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="2b2f7-121">Kliknij przycisk **Dodaj**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-121">Click **Add**.</span></span>
15. <span data-ttu-id="2b2f7-122">Na liście oznacz wybrany wiersz.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="2b2f7-123">Zaznacz pole wyboru **Zawieszone** lub usuń jego zaznaczenie.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="2b2f7-124">Rozwiń sekcję **Raportowanie finansowe**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="2b2f7-125">W polu **Typ kursu wymiany** kliknij przycisk rozwijany, aby otworzyć wyszukiwanie.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="2b2f7-126">Na liście wybierz **Typ kursu wymiany dla konta**.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="2b2f7-127">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="2b2f7-128">W polu **Typ przeliczania waluty** wybierz metodę obliczania kursów wymiany dla konta.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="2b2f7-129">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="2b2f7-129">Close the page.</span></span>
