# <a name="create-a-cost-accounting-ledger"></a><span data-ttu-id="d2855-103">Tworzenie księgi rachunku kosztów</span><span class="sxs-lookup"><span data-stu-id="d2855-103">Create a cost accounting ledger</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="d2855-104">Księga rachunku kosztów reprezentuje całościowo jednostkę raportowania.</span><span class="sxs-lookup"><span data-stu-id="d2855-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="d2855-105">Jest ona definiowana przez wymiar składnika kosztów, wymiar statystyczny, kalendarz obrachunkowy i walutę.</span><span class="sxs-lookup"><span data-stu-id="d2855-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="d2855-106">Nie rozpoznaje ona obiektów firm.</span><span class="sxs-lookup"><span data-stu-id="d2855-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="d2855-107">Firma i jej dane mogą być skojarzone z wieloma księgami rachunku kosztów.</span><span class="sxs-lookup"><span data-stu-id="d2855-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="d2855-108">Nagranie wykorzystuje dane firmy demonstracyjnej USP2.</span><span class="sxs-lookup"><span data-stu-id="d2855-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="d2855-109">Wybierz kolejno opcje Rachunek kosztów > Ustawienia księgi > Księgi rachunku kosztów.</span><span class="sxs-lookup"><span data-stu-id="d2855-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="d2855-110">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="d2855-110">Click New.</span></span>
3. <span data-ttu-id="d2855-111">W polu Nazwa wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="d2855-112">W polu Wymiar składnika kosztu wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="d2855-113">W polu Kalendarz obrachunkowy wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="d2855-114">W polu Waluta rozliczeniowa wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="d2855-115">W polu Typ kursu wymiany wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="d2855-116">W polu Wymiar statystyczny wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="d2855-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="d2855-117">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="d2855-117">Click Save.</span></span>
