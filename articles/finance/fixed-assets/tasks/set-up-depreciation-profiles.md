# <a name="set-up-and-create-depreciation-profiles"></a>Konfigurowanie i tworzenie profili amortyzacji

[!include [banner](../../includes/banner.md)]

Profile amortyzacji określają typ i częstotliwość amortyzacji danego środka trwałego.


## <a name="create-a-depreciation-profile"></a>Tworzenie profilu amortyzacji
1. W okienku nawigacji przejdź do **Moduły > Środki trwałe > Ustawienia > Profile amortyzacji**.
2. Kliknij przycisk **Nowy**.
3. W polu **Profil amortyzacji** wpisz wartość.
4. W polu **Nazwa** wpisz wartość.
5. W polu **Metoda** wybierz opcję. W przypadku wybrania opcji degresywna, należy wprowadzić wartość procentową w polu **Procent**.  
6. W polu **Rok amortyzacji** wybierz opcję.
7. W polu **Częstotliwość okresu** wybierz opcję.
8. Zamknij formularz.
