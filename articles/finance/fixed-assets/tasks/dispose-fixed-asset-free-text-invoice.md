# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="aa8dd-103">Likwidacja środka trwałego z wykorzystaniem faktury niezależnej</span><span class="sxs-lookup"><span data-stu-id="aa8dd-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="aa8dd-104">W tym temacie pokazano sposób likwidacji środka trwałego przy użyciu faktury niezależnej.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="aa8dd-105">W okienku nawigacji otwórz **Moduły > Rozrachunki z odbiorcami > Faktury > Wszystkie faktury niezależne**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="aa8dd-106">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-106">Select **New**.</span></span>
3. <span data-ttu-id="aa8dd-107">W polu **Konto odbiorcy** wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="aa8dd-108">Sprawdź poprawność domyślnej daty **Faktury** i w razie potrzeby ją zmień.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="aa8dd-109">Sprawdź poprawność pozostałych domyślnych pól nagłówka, takich jak **Waluta**, i w razie potrzeby je zmodyfikuj.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="aa8dd-110">W sekcji **Wiersze faktury** sprawdź poprawność pól **Opis** i **Konto główne**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="aa8dd-111">Sprawdź poprawność pól **domyślnej grupy podatków** i **grupy podatków dla pozycji**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="aa8dd-112">Wprowadź **cenę jednostkową** lub **Ilość** sprzedawanego środka trwałego.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="aa8dd-113">Wybierz sekcję **Szczegóły wiersza**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="aa8dd-114">W polu **Środek trwały** wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="aa8dd-115">Wybierz opcję **Zaksięguj**.</span><span class="sxs-lookup"><span data-stu-id="aa8dd-115">Select **Post**.</span></span>
