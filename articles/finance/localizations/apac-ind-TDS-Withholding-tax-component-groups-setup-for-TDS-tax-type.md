# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="c5b2a-103">Skonfiguruj grupy składników podatku u źródła dla typu podatku TDS</span><span class="sxs-lookup"><span data-stu-id="c5b2a-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c5b2a-104">W tym temacie wyjaśniono, jak skonfigurować grupy składników podatku u źródła, takie jak **czynsz** i **wykonawca**, dla typu podatku odliczanego u źródła (TDS).</span><span class="sxs-lookup"><span data-stu-id="c5b2a-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="c5b2a-105">Wybierz kolejno opcje **Podatek \> Konfiguracja \> Potrącona zaliczka na podatek \> Grupy potrąconych zaliczek na podatek**.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="c5b2a-106">[![Strona grup składników potrąconej zaliczki na podatek](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="c5b2a-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="c5b2a-107">W polu **Typ podatku** wybierz **TDS**, aby skonfigurować grupy składników podatku u źródła dla typu podatku potrącanego u źródła.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="c5b2a-108">W okienku akcji wybierz opcję **Nowy**, aby utworzyć wiersz.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="c5b2a-109">W polu **Grupa składników potrąconej zaliczki na podatek** wprowadź nazwę grupy składników TDS.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="c5b2a-110">W polu **Opis wprowadź** opis.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="c5b2a-111">Na skróconej karcie **Ogólne**, w polu **Stan** wybierz stan zamieszkania grupy składników TDS.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="c5b2a-112">Dostępne opcje to **Rezydent** i **Nierezydent**.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="c5b2a-113">W **polu Kod sekcji** wprowadź kod sekcji stosowany do grupy składników TDS.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="c5b2a-114">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="c5b2a-114">Close the page.</span></span>
