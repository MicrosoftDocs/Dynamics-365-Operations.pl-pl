## <a name="set-up-competencies-certificates"></a><span data-ttu-id="8c4f6-112">Konfigurowanie kompetencji: Certyfikaty</span><span class="sxs-lookup"><span data-stu-id="8c4f6-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="8c4f6-113">Na stronie listy **Pracownicy** wybierz wiersz pracownika, dla którego chcesz dodać informacje o zaświadczeniach.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="8c4f6-114">W okienku akcji na karcie **Pracownik** w grupie **Kompetencje** wybierz opcję **Certyfikaty**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="8c4f6-115">Wybierz opcję **Nowy**, a następnie w polu **Typ certyfikatu** wybierz opcję **PMP**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="8c4f6-116">W polu **Data rozpoczęcia** wybierz **1/10/2015**, a następnie wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="8c4f6-117">Konfigurowanie kompetencji: Umiejętności</span><span class="sxs-lookup"><span data-stu-id="8c4f6-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="8c4f6-118">Na stronie listy **Pracownicy** upewnij się, że pracownik użyty w poprzedniej procedurze nadal pozostaje zaznaczony.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="8c4f6-119">Następnie w okienku akcji na karcie **Pracownik** w grupie **Kompetencje** wybierz opcję **Umiejętności**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="8c4f6-120">Wybierz **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-120">Select **New**.</span></span>
3. <span data-ttu-id="8c4f6-121">W polu **Umiejętności** zaznacz opcję **Zarządzanie projektami**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="8c4f6-122">W polu **Poziom** zaznacz opcję **5 Ekspert**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="8c4f6-123">W polu **Data poziomu** zaznacz wartość **14.1.2014**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="8c4f6-124">W polu **Lata doświadczenia** wprowadź wartość **10**.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="8c4f6-125">Wybierz przycisk **Zapisz** i zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="8c4f6-125">Select **Save**, and then close the page.</span></span>
