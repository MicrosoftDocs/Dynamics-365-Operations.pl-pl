# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="303e4-103">Tworzenie faktury korygującej w fakturach projektu</span><span class="sxs-lookup"><span data-stu-id="303e4-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="303e4-104">Wybierz kolejno opcje Zarządzanie projektami i ich księgowanie> Projekty > Wszystkie projekty.</span><span class="sxs-lookup"><span data-stu-id="303e4-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="303e4-105">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="303e4-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="303e4-106">Wybierz identyfikator projektu używany dla faktury korygującej.</span><span class="sxs-lookup"><span data-stu-id="303e4-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="303e4-107">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="303e4-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="303e4-108">W okienku akcji kliknij pozycję Zarządzaj.</span><span class="sxs-lookup"><span data-stu-id="303e4-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="303e4-109">Kliknij opcję Arkusze faktur.</span><span class="sxs-lookup"><span data-stu-id="303e4-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="303e4-110">Wybierz transakcję dla faktury korygującej.</span><span class="sxs-lookup"><span data-stu-id="303e4-110">Select the transaction for credit note.</span></span> <span data-ttu-id="303e4-111">Na liście oznacz wybrany wiersz.</span><span class="sxs-lookup"><span data-stu-id="303e4-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="303e4-112">Kliknij przycisk Funkcje.</span><span class="sxs-lookup"><span data-stu-id="303e4-112">Click Functions.</span></span> 
8. <span data-ttu-id="303e4-113">Kliknij opcję Wybierz do faktury korygującej.</span><span class="sxs-lookup"><span data-stu-id="303e4-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="303e4-114">W celu wybrania transakcji do faktury korygującej należy zaznaczyć pole wyboru.</span><span class="sxs-lookup"><span data-stu-id="303e4-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="303e4-115">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="303e4-115">Click OK.</span></span> 

<span data-ttu-id="303e4-116">Po wykonaniu tego korku faktura korygująca będzie dostępna jako propozycja faktury.</span><span class="sxs-lookup"><span data-stu-id="303e4-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
