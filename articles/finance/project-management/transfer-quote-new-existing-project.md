# <a name="transfer-a-quotation-to-a-project"></a><span data-ttu-id="c07db-103">Przeniesienie oferty do projektu</span><span class="sxs-lookup"><span data-stu-id="c07db-103">Transfer a quotation to a project</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c07db-104">Jeśli oferta projektu ma stan **Potwierdzone**, można przenieść niektóre lub wszystkie wiersze oferty do nowego lub istniejącego projektu za pomocą kreatora **Przenieś do projektu**.</span><span class="sxs-lookup"><span data-stu-id="c07db-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="c07db-105">Aby użyć kreatora, należy wykonać następujące czynności.</span><span class="sxs-lookup"><span data-stu-id="c07db-105">To use the wizard, follow these steps.</span></span>

1. <span data-ttu-id="c07db-106">Wybierz kolejno opcje **Zarządzanie projektami i ich księgowanie** > **Oferty** > **Oferty w ramach projektu**.</span><span class="sxs-lookup"><span data-stu-id="c07db-106">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="c07db-107">Wybierz ofertę, która ma zostać przeniesiona, a następnie na karcie **Monituj** w grupie **Modyfikuj** wybierz opcję **Przenieś do projektu**.</span><span class="sxs-lookup"><span data-stu-id="c07db-107">Select the quotation you want to transfer and on the **Follow up** tab, in the **Modify** group, select **Transfer to project**.</span></span>
3. <span data-ttu-id="c07db-108">Aby skojarzyć i przenieść ofertę z projektem, należy postępować zgodnie z kreatorem **Przenieś ofertę do projektu**.</span><span class="sxs-lookup"><span data-stu-id="c07db-108">Follow the **Transfer quotation to project** wizard to associate and transfer the quotation with a project.</span></span>
