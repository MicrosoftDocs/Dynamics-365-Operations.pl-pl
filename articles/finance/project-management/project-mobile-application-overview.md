# <a name="project-mobile-applications-overview"></a>Przegląd aplikacji mobilnych do obsługi Project

[!include [banner](../includes/banner.md)]

Dostępne są trzy aplikacje mobilne związane z czasem projektu dla Microsoft Dynamics 365 Project Timesheet, Wprowadzanie czasu projektu i Moje karty czasu pracy/Karty czasu pracy do przeglądu przeze mnie (zoptymalizowane dla urządzeń przenośnych).

## <a name="project-timesheet-mobile-app"></a>Aplikacja mobilna Project Timesheet

Aplikacja mobilna Dynamics 365 Project Timesheet umożliwia użytkownikom przesyłanie i zatwierdzanie kart czasu dla projektów na urządzeniach mobilnych. Tej aplikacja mobilna zapewnia dostęp do funkcji karty czasu pracy w obszarze zarządzania projektami i księgowania w Dynamics 365 Finance. Pomaga poprawiać wydajność pracy użytkownika, a także pomaga w punktualnym wypełnianiu i zatwierdzaniu kart czasu pracy dla projektu.

## <a name="project-time-entry-workspace"></a>Obszar roboczy Wpis czasu projektu

Obszaru Wpis czasu projektu służy do wprowadzania i zapisywania czasu dla projektu. Jednak nie pozwala na przesyłanie ani zatwierdzanie kart czasu pracy dla projektu. Można go używać tylko do wprowadzania czasu. Ten mobilny obszar roboczy jest przeznaczony do używania razem z aplikacją mobilną Finance and Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Moje karty czasu pracy/Karty czasu pracy do przeglądu przeze mnie

Moje karty czasu pracy (zoptymalizowana dla urządzeń przenośnych) i Karty czasu pracy do przeglądu przeze mnie (zoptymalizowana dla urządzeń przenośnych) to rozwiązanie, które pozwala pracownikom tworzyć, przesyłać i zatwierdzać karty czasu pracy dla projektu na urządzeniach mobilnych. Pracownicy mają dostęp do funkcji Finance za pomocą adresu URL dla aplikacji. To rozwiązanie nie wymaga aplikacji mobilnej Finance and Operations.
