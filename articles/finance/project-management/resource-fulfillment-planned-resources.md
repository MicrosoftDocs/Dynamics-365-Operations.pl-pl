# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="018df-103">Wykorzystanie planowanych zasobów</span><span class="sxs-lookup"><span data-stu-id="018df-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="018df-104">Kierownik projektu może zaplanować role wymaganych zasobów w projekcie.</span><span class="sxs-lookup"><span data-stu-id="018df-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="018df-105">Menedżer zasobów będzie widział te zaplanowane zasoby jako żądania na stronie **Wykorzystanie zasobu** i może wtedy przypisywać faktyczne zasoby.</span><span class="sxs-lookup"><span data-stu-id="018df-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="018df-106">Na stronie **Wszystkie projekty** wybierz projekt **Uaktualnianie XYZ faza 2**.</span><span class="sxs-lookup"><span data-stu-id="018df-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="018df-107">Wybierz opcję **Projekt**, a następnie opcję **Edytuj**.</span><span class="sxs-lookup"><span data-stu-id="018df-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="018df-108">Na karcie **Zespół projektu i planowanie** wybierz przycisk **Dodaj**.</span><span class="sxs-lookup"><span data-stu-id="018df-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="018df-109">W oknie dialogowym **Dodaj role** wybierz rolę **Programista**.</span><span class="sxs-lookup"><span data-stu-id="018df-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="018df-110">Wybierz przycisk **Utwórz** i zamknij stronę projektu.</span><span class="sxs-lookup"><span data-stu-id="018df-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="018df-111">Na stronie **Wykorzystanie zasobu** wybierz pozycję **Programista 1** dla projektu **Uaktualnianie XYZ faza 2**.</span><span class="sxs-lookup"><span data-stu-id="018df-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="018df-112">Wybierz pracownika, a następnie wybierz przycisk **Przypisz**.</span><span class="sxs-lookup"><span data-stu-id="018df-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="018df-113">Upewnij się, że wiersz **Programista 1** został usunięte dla projektu **Uaktualnianie XYZ faza 2**.</span><span class="sxs-lookup"><span data-stu-id="018df-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="018df-114">Na karcie **Zespół projektu i planowanie** w projekcie **Uaktualnianie XYZ faza 2** upewnij się, że pracownik wybrany w poprzednim kroku został dodany jako **Programista**.</span><span class="sxs-lookup"><span data-stu-id="018df-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
