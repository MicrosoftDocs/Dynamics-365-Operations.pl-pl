4. <span data-ttu-id="745b6-110">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="745b6-110">In the **Description** field, type a value.</span></span>  <span data-ttu-id="745b6-111">Warunki płatności są opcjonalne.</span><span class="sxs-lookup"><span data-stu-id="745b6-111">The terms of payment is optional.</span></span> <span data-ttu-id="745b6-112">Jeśli wprowadzisz wartość w tym polu, faktura za opłaty z ponaglenia będzie używała tych warunków płatności zamiast warunków płatności zapisanych dla odbiorcy.</span><span class="sxs-lookup"><span data-stu-id="745b6-112">If you enter a value here, the collection letter fee invoice will use these terms of payment instead of the terms of payment stored with the customer.</span></span>

6. <span data-ttu-id="745b6-115">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="745b6-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="745b6-116">Domyślnie walutą opłaty będzie waluta odbiorcy.</span><span class="sxs-lookup"><span data-stu-id="745b6-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="745b6-117">Ten kod waluty może się różnić od kodu waluty faktury.</span><span class="sxs-lookup"><span data-stu-id="745b6-117">This currency code can be different than the invoice currency.</span></span>

9. <span data-ttu-id="745b6-122">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="745b6-122">In the **Description** field, type a value.</span></span>
10. <span data-ttu-id="745b6-123">W polu **Konto główne** wybierz konto przychodów, które będzie używane dla opłat.</span><span class="sxs-lookup"><span data-stu-id="745b6-123">In the **Main account** field, select the revenue account that will be used for fees.</span></span>

13. <span data-ttu-id="745b6-127">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="745b6-127">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="745b6-128">W polu **Minimalne saldo zaległości** wprowadź Minimalne saldo zaległości wymagane przed wysłaniem ponaglenia.</span><span class="sxs-lookup"><span data-stu-id="745b6-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>

    - <span data-ttu-id="745b6-133">Okres prolongaty ponaglenia 2 i kolejnych jest względny wobec daty zaksięgowania lub wydrukowania poprzedniego ponaglenia, zależnie od opcji wybranej w polu Aktualizacja kodu ponaglenia na stronie Parametry modułu rozrachunków z odbiorcami.</span><span class="sxs-lookup"><span data-stu-id="745b6-133">The grace period for collection letters 2 and higher is relative to the date that the previous collection letter is posted or printed, depending on the selection in the Update collection letter code field in the Accounts receivable parameters page.</span></span>

18. <span data-ttu-id="745b6-137">W polu **Opis** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="745b6-137">In the **Description** field, type a value.</span></span>

22. <span data-ttu-id="745b6-141">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="745b6-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="745b6-142">W polu **Minimalne saldo zaległości** wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="745b6-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="745b6-143">W polu **Dni** wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="745b6-143">In the **Days** field, enter a number.</span></span>
25. <span data-ttu-id="745b6-144">Zaznacz pole wyboru **Blokuj**, aby zablokować możliwość tworzenia kolejnych dostaw i faktur dla danego odbiorcy.</span><span class="sxs-lookup"><span data-stu-id="745b6-144">Select the **Block** check box to stop the customer from additional deliveries and invoicing.</span></span> <span data-ttu-id="745b6-145">Aby odblokować konto, wybierz **Nie** w Fakturowanie i dostawy zablokowane na stronie Klientów.</span><span class="sxs-lookup"><span data-stu-id="745b6-145">To unblock the account, select **No** in the Invoicing and delivery on hold field in the Customers page.</span></span>
