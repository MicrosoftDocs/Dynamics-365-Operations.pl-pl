[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano zestaw opcji stanu planu świadczeń pracowniczych dla aplikacji Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Zestaw opcji: typ procesu

Nazwa fizyczna: mshr_hrmCompProcessType

To wyliczenie udostępnia zestaw opcji dla typu procesu w encjach wirtualnych zasobów ludzkich.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | None | Brak. |
| 200000001 | Wybrana | Wybrano. |
| 200000002 | Uchylone | Uchylone. |
