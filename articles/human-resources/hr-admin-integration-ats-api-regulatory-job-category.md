# <a name="regulatory-job-category"></a>Kategoria pracy wymaganej przepisami prawa


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano zestaw opcji opcję Kategoria pracy wymaganej przepisami prawa dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_hcmregulatoryjobcategory

To wyliczenie określa zestaw opcji dla wartości zestawu opcji kategorii zadań prawnych. Te informacje są dostępne w istniejącym zestawie opcji mshr_hcmregulatoryjobcategory.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | None | Brak. |
| 200000001 | Wykonawczy | Kierownictwo/Starszy ranga członków i menedżerów. |
| 200000002 | Menedżer | Pierwszy/środkowy urzędnik i menedżer. |
| 200000003 | Profesjonalista | Specjaliści. |
| 200000004 | Technik | Technicy. |
| 200000005 | Sprzedawca | Pracownicy działu sprzedaży. |
| 200000006 | Administracyjne | Pracownicy pomocy technicznej administracyjnej. |
| 200000007 | Worker ( praca) | Rzemieślnicy. |
| 200000008 | Operacyjny | Operacyjny. |
| 200000009 | Robotnik | Pracownicy/pomocnicy. |
| 200000010 | ServiceWorker | Pracownicy serwisu. |
