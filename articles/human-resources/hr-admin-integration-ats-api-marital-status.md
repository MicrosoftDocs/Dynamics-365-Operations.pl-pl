# <a name="marital-status"></a>Stan cywilny


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym temacie opisano opcję Stan cywilny ustawioną dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_dirpersonmaritalstatus

To wyliczenie zapewnia opcję ustawioną dla stanu cywilnego akt osobowych.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | None | Nie wybrano żadnej wartości.
| 200000001 | Niebędący/niebędąca w związku | Niebędący/niebędąca w związku. |
| 200000002 | Żonaty/mężatka | Żonaty/mężatka. |
| 200000003 | Rozwiedziony/rozwiedziona | Rozwiedziony/rozwiedziona. |
| 200000004 | Wdowiec/wdowa | Wdowieństwo. |
