<span data-ttu-id="cf0d2-104">Jeśli użytkownik ma ochotników lub innych pracowników, może chcieć zmienić nazwę obszaru roboczego **Samoobsługi pracownika etatowego**.</span><span class="sxs-lookup"><span data-stu-id="cf0d2-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="cf0d2-105">Możesz zmienić ten obszar roboczy na **Samoobsługowe**.</span><span class="sxs-lookup"><span data-stu-id="cf0d2-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="cf0d2-106">Zmiana nazwy obszaru roboczego **Samoobsługi pracownika** powoduje również zmianę elementu menu używanego wewnętrznie przez Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="cf0d2-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="cf0d2-107">Jeśli wcześniej zastosowano dostosowania zabezpieczeń elementu menu **HcmEmployeeSelfServiceWorkspace**, zaleca się zastosowanie tych samych zmian do **HcmSelfServiceWorkspace** w celu zachowania parzystości.</span><span class="sxs-lookup"><span data-stu-id="cf0d2-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>
