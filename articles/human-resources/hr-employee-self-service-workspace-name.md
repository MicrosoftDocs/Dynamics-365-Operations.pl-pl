[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="d73d4-104">Jeśli użytkownik ma ochotników lub innych pracowników, może chcieć zmienić nazwę obszaru roboczego **Samoobsługi pracownika etatowego**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="d73d4-105">Możesz zmienić ten obszar roboczy na **Samoobsługowe**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="d73d4-106">Zmiana nazwy obszaru roboczego **Samoobsługi pracownika** powoduje również zmianę elementu menu używanego wewnętrznie przez Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="d73d4-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="d73d4-107">Jeśli wcześniej zastosowano dostosowania zabezpieczeń elementu menu **HcmEmployeeSelfServiceWorkspace**, zaleca się zastosowanie tych samych zmian do **HcmSelfServiceWorkspace** w celu zachowania parzystości.</span><span class="sxs-lookup"><span data-stu-id="d73d4-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="d73d4-108">W Human Resources wybierz opcję **Zarządzanie kadrami**, wybierz opcję **Łącza**, a następnie wybierz **Parametry Human Resources**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="d73d4-109">Wybierz kartę **Samoobsługa pracownika etatowego**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="d73d4-110">W obszarze **Nazwa wyświetlana** wybierz opcję **Samoobsługa**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-110">Under **Display name**, select **Self service**.</span></span>

   ![Zmień nazwę obszaru roboczego pracownika etatowego na samoobsługa](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="d73d4-112">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="d73d4-112">Select **Save**.</span></span>
