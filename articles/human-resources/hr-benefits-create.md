    * <span data-ttu-id="488ca-113">Aby ograniczyć pracownikom możliwość rejestracji w wielu planach medycznych, zaznacz opcję Jedna rejestracja na typ.</span><span class="sxs-lookup"><span data-stu-id="488ca-113">To restrict employees' ability to enroll in multiple medical plans, select One enrollment per type.</span></span>

7. <span data-ttu-id="488ca-115">Kliknij kartę Plany.</span><span class="sxs-lookup"><span data-stu-id="488ca-115">Click the Plans tab.</span></span>
8. <span data-ttu-id="488ca-116">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="488ca-116">Click New.</span></span>
9. <span data-ttu-id="488ca-117">W polu Plan wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="488ca-117">In the Plan field, type a value.</span></span>
10. <span data-ttu-id="488ca-118">Wypełnij pole Opis.</span><span class="sxs-lookup"><span data-stu-id="488ca-118">In the Description field, type a value.</span></span>
11. <span data-ttu-id="488ca-119">W polu Typ wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="488ca-119">In the Type field, enter or select a value.</span></span>
