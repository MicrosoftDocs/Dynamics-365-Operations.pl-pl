# <a name="set-up-rounding-rules"></a><span data-ttu-id="eb490-102">Konfigurowanie reguł zaokrąglania</span><span class="sxs-lookup"><span data-stu-id="eb490-102">Set up rounding rules</span></span>

[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="eb490-103">W programie Dynamics 365 Human Resources istnieje możliwość skonfigurowania reguł zaokrąglania spełniających konkretne potrzeby organizacji.</span><span class="sxs-lookup"><span data-stu-id="eb490-103">You can set up rounding rules in Dynamics 365 Human Resources to meet the needs of your organization.</span></span>

1. <span data-ttu-id="eb490-104">W obszarze roboczym **Zarządzanie świadczeniami** w sekcji **Konfiguracja** wybierz opcję **Reguły zaokrąglania**.</span><span class="sxs-lookup"><span data-stu-id="eb490-104">In the **Benefits management** workspace, under **Setup**, select **Rounding rules**.</span></span>

2. <span data-ttu-id="eb490-105">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="eb490-105">Select **New**.</span></span>

3. <span data-ttu-id="eb490-106">Wypełnij pola **Zaokrąglanie do najbliższej**, **Scenariusz** i **Metoda zaokrąglenia**.</span><span class="sxs-lookup"><span data-stu-id="eb490-106">Specify values for **Round to the nearest**, **Scenario**, and **Rounding form**.</span></span>

4. <span data-ttu-id="eb490-107">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="eb490-107">Select **Save**.</span></span>
