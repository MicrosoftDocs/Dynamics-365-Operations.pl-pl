3. <span data-ttu-id="471d1-108">W razie potrzeby dostosuj następujące ustawienia:</span><span class="sxs-lookup"><span data-stu-id="471d1-108">Adjust the following settings as appropriate:</span></span>

   - <span data-ttu-id="471d1-111">**Data końcowa**</span><span class="sxs-lookup"><span data-stu-id="471d1-111">**End date**</span></span>
   - <span data-ttu-id="471d1-112">**Podstawa warstwy**</span><span class="sxs-lookup"><span data-stu-id="471d1-112">**Tier basis**</span></span>

4. <span data-ttu-id="471d1-113">W obszarze **Pracownicy do rejestracji** zaznacz wszystkich pracowników, których chcesz przypisać do planu.</span><span class="sxs-lookup"><span data-stu-id="471d1-113">Under **Workers to enroll**, select all the works to assign to the plan.</span></span>
