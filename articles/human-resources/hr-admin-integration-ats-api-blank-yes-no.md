# <a name="blank-yes-no"></a>Puste Tak Nie

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym temacie opisano opcję Puste Tak Nie ustawioną dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_hrmblankyesno

To wyliczenie zapewnia zestaw opcji dla właściwości tak / nie, które również zezwalają na puste miejsca.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | Pusty | Nie wybrano żadnej wartości. |
| 200000001 | Tak | Tak. |
| 200000002 | Nr | Nr |
