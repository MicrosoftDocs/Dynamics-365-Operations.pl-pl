[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="cde36-104">Można wstrzymać urlop pracownika, aby przerwać przetwarzanie naliczeń dla wybranych typów urlopów.</span><span class="sxs-lookup"><span data-stu-id="cde36-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="cde36-105">Zawieszanie urlopów i nieobecności dla pracownika etatowego</span><span class="sxs-lookup"><span data-stu-id="cde36-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="cde36-106">W rekordzie pracownika wybierz pozycję **Urlop**.</span><span class="sxs-lookup"><span data-stu-id="cde36-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="cde36-107">Wybierz **Wstrzymaj urlop**.</span><span class="sxs-lookup"><span data-stu-id="cde36-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="cde36-108">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="cde36-108">Select **New**.</span></span>

4. <span data-ttu-id="cde36-109">W oknie dialogowym **Wstrzymaj naliczanie urlopu** wybierz **Typ urlopu** wraz z **Datą rozpoczęcia** i **Datą zakończenia** zawieszenia.</span><span class="sxs-lookup"><span data-stu-id="cde36-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

<span data-ttu-id="cde36-111">Jeśli naliczenia są przetwarzane podczas zawieszenia urlopu pracownika, nie zostaną nałożone żadne naliczenie dla typów wstrzymania urlopów.</span><span class="sxs-lookup"><span data-stu-id="cde36-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
