# <a name="education-credit-basis"></a>Podstawa kredytu dla wykształcenia


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym temacie opisano opcję Podstawę kredytową dla edukacji ustawioną dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_hcmeducationcreditbasis

To wyliczenie zawiera zestaw wartości opcji dla podstawy punktów edukacyjnych dla historii wykształcenia kandydata.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | Pusty | Nie wybrano żadnej wartości. |
| 200000001 | Semestr | Semestr. |
| 200000002 | Kwartał | Kwartał. |
| 200000003 | Trymestr | Trymestr. |
| 200000004 | Okres | Okres. |
| 200000005 | Inna | Inne. |
