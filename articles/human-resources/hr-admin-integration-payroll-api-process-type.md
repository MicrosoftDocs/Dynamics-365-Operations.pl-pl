## <a name="option-set-process-type"></a>Zestaw opcji: typ procesu

Nazwa fizyczna: mshr_hrmCompProcessType

To wyliczenie udostępnia zestaw opcji dla typu procesu w encjach wirtualnych zasobów ludzkich.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | None | Brak. |
| 200000001 | Co rok | Co rok. |
| 200000002 | Rocznica | Rocznica. |
| 200000003 | Niecyklicznie | Niecyklicznie. |
| 200000004 | ½ rok | Co pół roku. |
| 200000005 | Co kwartał | Co kwartał. |
| 200000006 | Miesięczne | Miesięczne. |
