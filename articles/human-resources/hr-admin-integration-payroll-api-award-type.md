# <a name="award-type"></a>Typ nagrody


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano zestaw opcji Typ nagrody dla Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Zestaw opcji: typ nagrody

Nazwa fizyczna: mshr_HrmCompVarAwardEmplType

To wyliczenie udostępnia opcję ustawioną dla typu wynagrodzenia o zmiennej wysokości w wirtualnych jednostkach Human Resources.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | Nagroda ogólna | Nagroda ogólna. |
| 200000001 | Stałe wynagrodzenie ryczałtowe | Stałe wynagrodzenie ryczałtowe. |
