# <a name="payroll---employment-type-option-set"></a>Lista płac — zestaw opcji typu zatrudnienia

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano opcję Typ zatrudnienia ustawioną dla aplikacji Dynamics 365 Human Resources.

## <a name="option-set-employment-type"></a>Zestaw opcji: typ zatrudnienia

To wyliczenie zawiera zestaw opcji typu zatrudnienia dla pracownika etatowego. Jest to dostępne w zestawie opcji mshr_hcmemploymenttype.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | Pracownik | Pracownik |
| 200000001 | Zleceniobiorca | Zleceniobiorca |
