# <a name="screening-frequency"></a>Częstotliwość kontroli

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym temacie opisano częstotliwość kontroli na podstawie zestawu opcji dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_hcmfrequencyunit

To wyliczenie zapewnia zestaw wartości opcji dla częstotliwości przesiewania. 

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 Tylko jeden raz | Kontrola jest wymagana tylko raz. |
| 200000001 codziennie | Częstotliwość kontroli jest obliczana w dniach. |
| 200000002 Tygodniowa | Częstotliwość kontroli jest obliczana w tygodniach. |
| 200000003 Miesięczne | Częstotliwość kontroli jest obliczana w miesiącach. |
| 200000004 Co rok | Częstotliwość kontroli jest obliczana w latach. |
