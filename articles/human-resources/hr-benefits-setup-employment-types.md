# <a name="set-up-employment-types"></a>Konfigurowanie typów zatrudnienia

[!include [banner](includes/preview-feature.md)]

W programie Dynamics 365 Human Resources ustaw typy zatrudnienia pasujące do pracowników w Twojej organizacji.

1. W obszarze roboczym **Zarządzanie świadczeniami** w sekcji **Konfiguracja** wybierz opcję **Typy zatrudnienia**.

2. Wybierz pozycję **Nowy**.

3. Określ wartości dla następujących pól:

   | Pole | Opis |
   | --- | --- |
   | Typ zatrudnienia | Unikatowa identyfikacja typu zatrudnienia. |
   | Opis | Opis typu zatrudnienia. |

4. Wybierz opcję **Zapisz**.
