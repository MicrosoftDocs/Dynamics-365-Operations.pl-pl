# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Usunięte lub przestarzałe funkcje w rozwiązaniu Dynamics 365 Human resources

W tym artykule opisano funkcje, które zostały usunięte lub są przeznaczone do usunięcia z aplikacji Microsoft Dynamics 365 Human resources.

- *Usunięta* funkcja nie jest już dostępna w produkcie.
- *Przestarzała* funkcja nie jest aktywnie tworzona i może zostać usunięta w przyszłej aktualizacji.

Ta lista ma na celu ułatwienie uwzględnienia usuniętych i przestarzałych funkcji we własnym planowaniu.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Funkcje usunięte lub przestarzałe w Zasobach ludzkich 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Tworzenie nowego projektu LCS w dziale zasobów ludzkich (samodzielny) 

[Aprowizowanie rozwiązania Human Resources](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Przyczyna wycofania/usunięcia** | Zastąpiono tworzenie projektów dotyczących infrastruktury finansowej i operacyjnej. Aby dowiedzieć się więcej o łączeniu infrastruktury, zobacz [Dynamics 365 – Łączenie infrastruktury zasobów ludzkich FAQ](/hr-infrastructure-merge-faq). |
| **Zamieniona przez inną funkcję?**   | Tak |
| **Powiązane obszary produktów**         | Zgłoszenie |
| **Opcja wdrażania**              | Wszystko |
| **Stan**                         | Przestarzałe: Od lipca 2022 roku nie wspieramy już tworzenia nowych projektów w Zasobach ludzkich (samodzielnych). |

|------------|--------------------|

| **Zamieniona przez inną funkcję?**   | Tak |
| **Powiązane obszary produktów**         | Zgłoszenie |

| **Stan**                         | Przestarzałe: Od lipca 2022 roku nie obsługujemy już dostarczania środowisk produkcyjnych HR na stronie Zasoby ludzkie (samodzielne). |
