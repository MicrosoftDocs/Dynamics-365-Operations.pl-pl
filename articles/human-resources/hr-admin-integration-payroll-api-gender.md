# <a name="payroll---gender-option-set"></a>Lista płac - Zestaw opcji: Płeć


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano opcję Płeć ustawioną dla aplikacji Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Zestaw opcji: Płeć

To wyliczenie zawiera zestaw opcji wartości płeć dla pracowników. Jest to dostępne w zestawie opcji mshr_hcmpersongender.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | None | Nie określono płci. |
| 200000001 | Mężczyzna | Mężczyzna. |
| 200000002 | Kobieta | Kobieta. |
| 200000003 | NonSpecific | Wybór dla nieokreślonej płci. |
