# <a name="run-the-compensation-process"></a>Uruchamianie procesu związanego z wynagrodzeniem

**Wydaj**

Klient próbuje uruchomić proces kompensacji w Microsoft Dynamics 365 Human Resources, ale nie udaje się go zakończyć pomyślnie.

**Przyczyna**

Klient potrzebuje dokumentacji.

**Rozdzielczość**

Poniżej przedstawiono dokumentację dla kompensacji:

- Przezbrajanie:

    - [Plany wynagrodzeń](https://docs.microsoft.com/dynamics365/unified-operations/talent/compensation-plans)
    - [Tworzenie planów stałych wynagrodzeń](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-fixed-compensation-plans)
    - [Tworzenie planów wynagrodzeń o zmiennej wysokości](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-variable-compensation-plans)

- Proces:

    - [Przetwarzanie wynagrodzenia](https://docs.microsoft.com/dynamics365/unified-operations/talent/process-compensation)
