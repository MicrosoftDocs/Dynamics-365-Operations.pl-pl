# <a name="skill-level-type"></a>Typ poziomu kwalifikacji


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

W tym artykule opisano zestaw opcji Typ poziomu kwalifikacji dla Dynamics 365 Human Resources.

Nazwa fizyczna: mshr_hrmskillleveltype

To wyliczenie zapewnia zestaw wartości opcji, które kategoryzują poziom przypisany do umiejętności kandydata.

| Wartość | Etykiety | opis |
| --- | --- | --- |
| 200000000 | Wartość rzeczywista | Rzeczywisty poziom kwalifikacji. |
| 200000001 | Wartość docelowa | Wartość docelowa dla poziomu umiejętności. |
