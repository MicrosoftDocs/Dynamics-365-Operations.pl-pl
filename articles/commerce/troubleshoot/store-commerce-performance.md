# <a name="troubleshoot-store-commerce-performance-issues"></a>Rozwiązywanie problemów z wydajnością Store Commerce

[!include [banner](../includes/banner.md)]

W tym artykule opisano sposób rozwiązywania problemów z wydajnością aplikacji Microsoft Dynamics 365 Commerce Store Commerce.

- W przypadku rozwiązywania problemów z wydajnością Store Commerce należy unikać zdalnego logowania. Zamiast tego zaloguj się bezpośrednio na urządzeniu Store Commerce.
- Jeśli aplikacja Store Commerce działa wolno, sprawdź użycie procesora CPU, pamięci, sieci i dysku, używając Menedżera zadań lub Monitora zasobów. Upewnij się, że zasoby urządzeń nie są znaczące.
- Sprawdź stan łączności sieciowej i potwierdź, że żądania/odpowiedzi HTTPS nie trwa dłużej niż kilka sekund.
