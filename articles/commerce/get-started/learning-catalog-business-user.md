# <a name="learning-catalog-for-users-of-dynamics-365-commerce"></a>Katalog szkoleń dla użytkowników Dynamics 365 Commerce

Czy po prostu dostajesz nową aplikację do użycia?

Poniższy katalog jest zorganizowany od podstawowej bazy wiedzy do konkretnych domen oraz od najbardziej do najbardziej zaawansowanego. Jeśli zawartość istnieje w wielu formatach, poinformujemy Cię, aby można było wybrać format szkoleń najlepiej odpowiadający Twoim potrzebom.

Aby uzyskać informacje dotyczące funkcji udostępnianych innym aplikacjom finansowym i operacyjnym, zapoznaj się z [Katalogiem szkoleń Dynamics 365 Finance](../../finance/get-started/learning-catalog-business-user.md).
