# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Wymagania przeglądarki dotyczące witryn Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

Ten temat zawiera wymagania przeglądarki dotyczące witryn Microsoft Dynamics 365 Commerce.

Witryny Dynamics 365 Commerce są najlepiej obsługiwane przez następujące przeglądarki internetowe. Aby znaleźć najnowszą wersję dla każdej przeglądarki, przejdź do witryny producenta oprogramowania.

- [Microsoft Edge](https://www.microsoft.com/edge) (do bieżącej wersji minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (do bieżącej wersji minus 1)
- [Google Chrome](https://www.google.com/chrome) (do bieżącej wersji minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (do bieżącej wersji minus 1)

> [!NOTE]
> Internet Explorer nie jest obsługiwana. W przypadku korzystania z Internet Explorer niektóre funkcje mogą nie działać, a użytkownicy mogą mieć problemy z obsługą.
