[!include [banner](../includes/banner.md)]

<span data-ttu-id="76fca-104">Ta procedura pokazuje, jak skonfigurować ustawienia profilu funkcji sklepu, które mają zastosowanie do przedstawicieli handlowych za pomocą danych demonstracyjnych firmy USRT.</span><span class="sxs-lookup"><span data-stu-id="76fca-104">This procedure demonstrates how to configure a store's functionality profile settings that apply to sales representatives using the USRT demo data company.</span></span>

1. <span data-ttu-id="76fca-105">Wybierz kolejno opcje Retail i Commerce > Ustawienia kanału > Ustawienia punktu sprzedaży > Profile punktów sprzedaży > Profile funkcji.</span><span class="sxs-lookup"><span data-stu-id="76fca-105">Go to Retail and Commerce > Channel setup > POS setup > POS profiles > Functionality profiles.</span></span>
2. <span data-ttu-id="76fca-106">Kliknij przycisk Edytuj.</span><span class="sxs-lookup"><span data-stu-id="76fca-106">Click Edit.</span></span>
3. <span data-ttu-id="76fca-107">Rozwiń sekcję Funkcje.</span><span class="sxs-lookup"><span data-stu-id="76fca-107">Expand the Functions section.</span></span>
    * <span data-ttu-id="76fca-108">Funkcja ustawień profilu umożliwia skonfigurowania punktu sprzedaży, aby automatycznie dodawał domyślną grupę sprzedaży kasjera, monitował o podanie grup sprzedaży i wymagał grup sprzedaży.</span><span class="sxs-lookup"><span data-stu-id="76fca-108">You can use the functionality profile settings to configure POS to automatically add the cashier's default sales group, to prompt for sales groups, and to require sales groups.</span></span>
