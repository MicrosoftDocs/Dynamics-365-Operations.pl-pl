# <a name="crop-images"></a><span data-ttu-id="6af76-103">Przycinanie obrazów</span><span class="sxs-lookup"><span data-stu-id="6af76-103">Crop images</span></span>

<span data-ttu-id="6af76-105">Biblioteka multimediów kreatora witryn Commerce umożliwia kadrowanie obrazów w celu ich zoptymalizowania dla różnych typów modułów i okienka ekranu.</span><span class="sxs-lookup"><span data-stu-id="6af76-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="6af76-106">Przytnij obraz</span><span class="sxs-lookup"><span data-stu-id="6af76-106">Crop an image</span></span>

<span data-ttu-id="6af76-107">Aby przyciąć obraz w konstruktorze witryn, wykonaj następujące kroki.</span><span class="sxs-lookup"><span data-stu-id="6af76-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="6af76-108">W lewym okienku nawigacyjnym w module kreatora witryn Commerce wybierz **bibliotekę multimediów**.</span><span class="sxs-lookup"><span data-stu-id="6af76-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="6af76-109">W oknie głównym wybierz obraz, który chcesz zmodyfikować.</span><span class="sxs-lookup"><span data-stu-id="6af76-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="6af76-110">Na pasku poleceń wybierz opcję **Edytuj**.</span><span class="sxs-lookup"><span data-stu-id="6af76-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="6af76-111">Wybierz obraz, by wejść w **tryb edycji**.</span><span class="sxs-lookup"><span data-stu-id="6af76-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="6af76-112">W obszarze **tryb edycji** wybierz opcję **Edytuj widok według modułów**.</span><span class="sxs-lookup"><span data-stu-id="6af76-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="6af76-113">Z menu rozwijanego **moduł** wybierz typ modułu.</span><span class="sxs-lookup"><span data-stu-id="6af76-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="6af76-114">Z menu rozwijanego **Wyświetl typ** wybierz typ wyświetlania.</span><span class="sxs-lookup"><span data-stu-id="6af76-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="6af76-115">Z menu rozwijanego **Umieszczenie** wybierz umieszczenie obrazu.</span><span class="sxs-lookup"><span data-stu-id="6af76-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="6af76-116">Z menu rozwijanego **Okienko ekranu** wybierz typ rozmiar okienka.</span><span class="sxs-lookup"><span data-stu-id="6af76-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="6af76-117">Obraz jest nałożony na obszar odpowiadający regionowi przycinania.</span><span class="sxs-lookup"><span data-stu-id="6af76-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="6af76-118">W razie potrzeby Przenieś obszar kadrowania i zmień jego rozmiar.</span><span class="sxs-lookup"><span data-stu-id="6af76-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="6af76-119">Współczynnik proporcji będzie obsługiwany automatycznie.</span><span class="sxs-lookup"><span data-stu-id="6af76-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="6af76-120">Po zakończeniu pracy na pasku poleceń wybierz opcję **Zapisz**, a następnie wybierz opcję **Zakończ edycję**.</span><span class="sxs-lookup"><span data-stu-id="6af76-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="6af76-121">Po zakończeniu przycinania niestandardowego modyfikacje obrazów zaczną obowiązywać niemal natychmiast.</span><span class="sxs-lookup"><span data-stu-id="6af76-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
