# <a name="change-the-banner-or-logo"></a><span data-ttu-id="812d7-103">Zmienianie transparentu lub logo</span><span class="sxs-lookup"><span data-stu-id="812d7-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="812d7-104">Następująca procedura wymienia kroki, jakie administratorzy systemu mogą stosować do aktualizowania obrazu transparentu lub logo wyświetlanego dla firmy.</span><span class="sxs-lookup"><span data-stu-id="812d7-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="812d7-105">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="812d7-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="812d7-106">Otwórz **Okienko nawigacji > Moduły > Organizacja systemu > Organizacje > Firmy**.</span><span class="sxs-lookup"><span data-stu-id="812d7-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="812d7-107">Na liście po lewej stronie wybierz firmę, dla której chcesz zaktualizować transparent lub logo.</span><span class="sxs-lookup"><span data-stu-id="812d7-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="812d7-108">Jeśli firma jest zaznaczona, przejdź do następnego kroku.</span><span class="sxs-lookup"><span data-stu-id="812d7-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="812d7-109">Rozwiń kartę **Obraz na pulpicie nawigacyjnym**.</span><span class="sxs-lookup"><span data-stu-id="812d7-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="812d7-110">Wybierz **Zmień**.</span><span class="sxs-lookup"><span data-stu-id="812d7-110">Select **Change**.</span></span>
    
    <span data-ttu-id="812d7-111">Idealna rozdzielczość obrazu banera to 1920 x 281 pikseli.</span><span class="sxs-lookup"><span data-stu-id="812d7-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="812d7-112">Idealna szerokość logo wynosi 350 pikseli.</span><span class="sxs-lookup"><span data-stu-id="812d7-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="812d7-113">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="812d7-113">Select **Save**.</span></span>
6. <span data-ttu-id="812d7-114">Otwórz **Okienko nawigacji > Moduły > Wspólne > Wspólne > Domyślny pulpit nawigacyjny**.</span><span class="sxs-lookup"><span data-stu-id="812d7-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="812d7-115">Nowy transparent lub logo powinno być widoczne na pulpicie nawigacyjnym.</span><span class="sxs-lookup"><span data-stu-id="812d7-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="812d7-116">Może być konieczna zmiana firmy, za pomocą wybieraka firmy, na firmę, do której przekazano transparent.</span><span class="sxs-lookup"><span data-stu-id="812d7-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
