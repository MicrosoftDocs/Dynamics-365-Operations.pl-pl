# <a name="configure-the-global-address-book"></a><span data-ttu-id="f7f56-103">Konfigurowanie globalnej książki adresowej</span><span class="sxs-lookup"><span data-stu-id="f7f56-103">Configure the global address book</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f7f56-104">Ta procedura służy do ustawiania domyślnych wartości i zasad zabezpieczeń dla globalnej książki adresowej.</span><span class="sxs-lookup"><span data-stu-id="f7f56-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="f7f56-105">Dane wykorzystane do stworzenia tego zadania pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="f7f56-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="f7f56-106">Zadanie jest przeznaczone dla zespołu ds. planowania i konfiguracji.</span><span class="sxs-lookup"><span data-stu-id="f7f56-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="f7f56-107">W okienku nawigacji przejdź do **Moduły > Administrowanie organizacją > Globalna książka adresowa > Parametry globalnej książki adresowej**.</span><span class="sxs-lookup"><span data-stu-id="f7f56-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="f7f56-108">W polu **Kolejność w nazwisku** wybierz, jak mają być pokazywane imiona i nazwiska.</span><span class="sxs-lookup"><span data-stu-id="f7f56-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="f7f56-109">W **Usuń strony bez ról**, wybierz, czy chcesz usunąć strony, którym nie przypisano roli.</span><span class="sxs-lookup"><span data-stu-id="f7f56-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="f7f56-110">W **Użyj funkcji sprawdzania duplikatów** wybierz, czy należy sprawdzić, czy są zduplikowane rekordy.</span><span class="sxs-lookup"><span data-stu-id="f7f56-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="f7f56-111">W **Przeglądanie numeru DUNS adresów** wybierz, czy chcesz wyświetlić numer DUNS dla adresów.</span><span class="sxs-lookup"><span data-stu-id="f7f56-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="f7f56-112">W **Kontrola unikatowości numeru DUNS** wybierz, czy chcesz sprawdzić, czy są niepowtarzalne numery DUNS.</span><span class="sxs-lookup"><span data-stu-id="f7f56-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="f7f56-113">W polu **Strona** wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="f7f56-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="f7f56-114">W polu **Odbiorca** wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="f7f56-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="f7f56-115">W polu **Dostawca** wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="f7f56-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="f7f56-116">W polu **Potencjalny klient** wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="f7f56-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="f7f56-117">W polu **Konkurent** wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="f7f56-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="f7f56-118">Kliknij kartę **Zabezpieczenia lokalizacji prywatnych**.</span><span class="sxs-lookup"><span data-stu-id="f7f56-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="f7f56-119">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="f7f56-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="f7f56-120">Naciśnij klawisz Shift, aby zaznaczyć wiele ról, które mają zostać dodane do okienka **Wybrane role**, a następnie kliknij strzałkę, aby dodać wybrane role.</span><span class="sxs-lookup"><span data-stu-id="f7f56-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="f7f56-121">Kliknij przycisk **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="f7f56-121">Click **Save**.</span></span>
