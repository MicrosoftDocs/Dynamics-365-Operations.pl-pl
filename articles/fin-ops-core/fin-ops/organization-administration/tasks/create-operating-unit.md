# <a name="create-an-operating-unit"></a><span data-ttu-id="74c0a-103">Tworzenie jednostki operacyjnej</span><span class="sxs-lookup"><span data-stu-id="74c0a-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="74c0a-104">Jednostka operacyjna to organizacja używana do dzielenia formantu zasobów ekonomicznych i procesów operacyjnych w firmie.</span><span class="sxs-lookup"><span data-stu-id="74c0a-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="74c0a-105">Osoby w jednostce operacyjnej mają obowiązek maksymalizować wykorzystanie ograniczonych zasobów i konta dla zapewnienia ich wydajności.</span><span class="sxs-lookup"><span data-stu-id="74c0a-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="74c0a-106">Typy jednostek operacyjnych obejmują centra kosztów, jednostki biznesowe, działy i strumienie wartości.</span><span class="sxs-lookup"><span data-stu-id="74c0a-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="74c0a-107">Aby jednostkę operacyjną, należy wykonać następującą procedurę.</span><span class="sxs-lookup"><span data-stu-id="74c0a-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="74c0a-108">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="74c0a-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="74c0a-109">Otwórz **Okienko nawigacji > Moduły > Organizacja systemu > Organizacje > Firmy > Jednostki operacyjne.**</span><span class="sxs-lookup"><span data-stu-id="74c0a-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="74c0a-110">Kliknij przycisk **Nowy** aby otworzyć rozwijane okno dialogowe.</span><span class="sxs-lookup"><span data-stu-id="74c0a-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="74c0a-111">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="74c0a-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="74c0a-112">Wybierz typu jednostki operacyjnej, który ma zostać utworzony.</span><span class="sxs-lookup"><span data-stu-id="74c0a-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="74c0a-113">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="74c0a-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="74c0a-114">W polu **Nazwa** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="74c0a-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="74c0a-115">W razie potrzeby rozwiń sekcję **Ogólne**.</span><span class="sxs-lookup"><span data-stu-id="74c0a-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="74c0a-116">Podaj ogólne informacje o jednostce operacyjnej, takie jak na przykład menedżer, numer identyfikacyjny i identyfikator DUNS.</span><span class="sxs-lookup"><span data-stu-id="74c0a-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="74c0a-117">W razie potrzeby rozwiń sekcję **Adresy**.</span><span class="sxs-lookup"><span data-stu-id="74c0a-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="74c0a-118">Wprowadź informacje adresowe, takie jak nazwa ulicy i numer budynku, kod pocztowy oraz miasto.</span><span class="sxs-lookup"><span data-stu-id="74c0a-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="74c0a-119">Kliknij przycisk **Dodaj**, aby wprowadzić nowy rekord adresu, lub kliknij przycisk Edytuj, aby zmodyfikować istniejący rekord adresu.</span><span class="sxs-lookup"><span data-stu-id="74c0a-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="74c0a-120">W razie potrzeby rozwiń sekcję **Informacje kontaktowe**.</span><span class="sxs-lookup"><span data-stu-id="74c0a-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="74c0a-121">Wprowadź informacje dotyczące metod komunikacji, takie jak adresy e-mail, adresy URL i numery telefonów.</span><span class="sxs-lookup"><span data-stu-id="74c0a-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="74c0a-122">Aby wprowadzić nowy rekord komunikacji, kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="74c0a-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="74c0a-123">Aby zmodyfikować istniejący rekord komunikacji, wybierz kolejno opcje **Więcej opcji > Zaawansowane**.</span><span class="sxs-lookup"><span data-stu-id="74c0a-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="74c0a-124">Opcjonalnie zmień **Numer jednostki operacyjnej** w razie potrzeby.</span><span class="sxs-lookup"><span data-stu-id="74c0a-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="74c0a-125">Należy zauważyć, że ten numer jest unikatowym identyfiktorem dla pasującego rekordu **Jednostki** i nie może być taki sam jak jakakolwiek inna jednostka operacyjna.</span><span class="sxs-lookup"><span data-stu-id="74c0a-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="74c0a-126">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="74c0a-126">Select **Save**.</span></span>
