6. <span data-ttu-id="4c9b7-118">Rozwiń sekcję **Segmenty**.</span><span class="sxs-lookup"><span data-stu-id="4c9b7-118">Expand the **Segments** section.</span></span>

    - <span data-ttu-id="4c9b7-120">Sekwencje identyfikatorów wszystkich zakresów mogą zawierać *segmenty stałe* i *segmenty alfanumeryczne*.</span><span class="sxs-lookup"><span alfanumerycznych, które się nie zmieniają.</span><span class="sxs-lookup"><span data-stu-id="4c9b7-121">Constant segments contain a set of alphanumeric characters that do not change.</span></span> <span data-ttu-id="4c9b7-122">Tego typu segmentu należy używać, aby dodać łącznik lub inne separatory między segmentami sekwencji identyfikatorów.</span><span class="sxs-lookup"><span data-stu-id="4c9b7-122">Use this segment type to add a hyphen or other separators between number sequence segments.</span></span> <span data-ttu-id="4c9b7-123">Segmenty alfanumeryczne zawierają class="sxs-lookup"><span data-stu-id="4c9b7-124">These characters represent letters and numbers that increment every time that a number from the sequence is used.</span></span> <span
