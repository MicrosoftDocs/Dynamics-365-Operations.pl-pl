# <a name="removed-or-deprecated-features-in-lifecycle-services-lcs"></a>Usunięte lub przestarzałe funkcje usługi Lifecycle Services (LCS)

[!include[banner](../includes/banner.md)]

W tym temacie opisano funkcje, które zostały usunięte lub uznane za przestarzałe w usłudze Microsoft Dynamics Lifecycle Services (LCS).

- *Usunięta* funkcja nie jest już dostępna w usłudze.
- *Przestarzała* funkcja nie jest aktywnie tworzona i może zostać usunięta w przyszłej aktualizacji.

Ta lista jest udostępniana, aby umożliwić Ci uwzględnianie usuniętych i przestarzałych funkcji we własnym planowaniu.
