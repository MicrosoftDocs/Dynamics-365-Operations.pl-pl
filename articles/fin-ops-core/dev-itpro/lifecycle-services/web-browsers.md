# <a name="supported-web-browsers-for-lifecycle-services"></a>Obsługiwane przeglądarki internetowe dla Lifecycle Services (LCS) 

[!include[banner](../includes/banner.md)]

Użytkownicy mogą uzyskać dostęp do usług Microsoft Dynamics Lifecyle Services (LCS) za pomocą najnowszych wersji tych popularnych przeglądarki: 

- Microsoft Edge (zalecane: [Edge oparte na Chromium](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Aby znaleźć najnowszą wersję dla każdej przeglądarki, przejdź do witryny producenta oprogramowania. W celu uzyskania optymalnej wydajności i optymalnego działania zaleca się, aby używać najnowszej wersji nowoczesnej przeglądarki, w szczególności Microsoft Edge.
