# <a name="screen-destination"></a>Miejsce docelowe — ekran

[!include [banner](../includes/banner.md)]

Możesz konfigurować lokalizacje docelową ekran dla poszczególnych składników FOLDER lub FILE formatu sprawozdawczości elektronicznej (ER) skonfigurowanych do generowania dokumentów wychodzących. Na podstawie ustawienia miejsca docelowego wygenerowany dokument jest otwarty do przejrzenia w oddzielnym oknie przeglądarki.

Jeśli ustawisz w opcji **Włączone** wartość **Tak**, będzie tworzony podgląd danych wyjściowych. Niektóre typy plików, takie jak XML, TXT i PDF, można wyświetlać bezpośrednio w oknie przeglądarki internetowej. Dla innych typów plików, takich jak pliki programów Microsoft Excel lub Word, jest używana usługa Microsoft Office Online.
