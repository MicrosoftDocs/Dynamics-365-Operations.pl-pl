# <a name="list-of-er-functions-in-the-record-category"></a>Lista funkcji modułu ER w kategorii rekordów

[!include [banner](../includes/banner.md)]

Funkcje rekordów w module raportowania elektronicznego (ER) mogą być używane do wyodrębniania informacji z i wykonywania operacji na źródłach danych o typie danych *Kontener (rekordu)*. Ten temat zawiera podsumowanie tych funkcji.

## <a name="list-of-supported-functions"></a>Lista obsługiwanych funkcji

| Funkcja | Opis |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Ta funkcja zwraca pustą wartość *Kontener (rekord)*, która ma tę samą strukturę, co wybrana lista rekordów lub rekord. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Ta funkcja zwraca pustą wartość *Kontener (rekord)*, która ma tę samą strukturę, co wybrana lista rekordów lub rekord. |
