# <a name="list-of-er-functions-in-the-mathematical-category"></a>Lista funkcji modułu ER w kategorii funkcji matematycznych

[!include [banner](../includes/banner.md)]

Funkcje matematyczne w module raportowania elektronicznego (ER) mogą służyć do wykonywania wielu typowych obliczeń matematycznych. Ten temat zawiera podsumowanie tych funkcji.

## <a name="list-of-supported-functions"></a>Lista obsługiwanych funkcji

| Funkcja | Opis |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Ta funkcja zwraca wartość bezwzględną (moduł) określonej liczby jako wartość *rzeczywistą*. Innymi słowy zwraca liczbę bez znaku. |
| [Moc](er-functions-mathematical-power.md)         | Ta funkcja zwraca wartość *rzeczywistą* reprezentującą wynik podniesienia określonej liczby dodatniej do określonej potęgi. |
| [Round](er-functions-mathematical-round.md)         | Ta funkcja zwraca podaną liczbę jako wartość *rzeczywistą* po zaokrągleniu jej do określonej liczby miejsc dziesiętnych. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Ta funkcja zwraca podaną liczbę jako wartość *rzeczywistą* po zaokrągleniu jej w dół do określonej liczby miejsc dziesiętnych. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Ta funkcja zwraca podaną liczbę jako wartość *rzeczywistą* po zaokrągleniu jej w górę do określonej liczby miejsc dziesiętnych. |
