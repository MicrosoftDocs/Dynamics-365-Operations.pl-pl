# <a name="file-destination"></a>Aplikacja docelowa pliku

[!include [banner](../includes/banner.md)]

Możesz konfigurować lokalizacje docelową pliku dla poszczególnych składników FOLDER lub FILE formatu sprawozdawczości elektronicznej (ER) skonfigurowanych do generowania dokumentów wychodzących. Na podstawie ustawienia miejsca docelowego wygenerowany dokument jest dostępny do pobrania z przeglądarki sieci Web.

Jeśli w **Ustawieniach miejsca docelowego** nadano parametrowi **Włączone** wartość **Tak**, po zakończeniu sesji konfiguracji otworzy się okno dialogowe otwierania lub zapisywania.
