# <a name="len-er-function"></a>LEN, funkcja ER

[!include [banner](../includes/banner.md)]

Funkcja `LEN` zwraca określoną liczbę znaków w określonym ciągu jako wartość *Liczba całkowita*.

## <a name="syntax"></a>Składnia

```vb
LEN (text)
```

## <a name="arguments"></a>Argumenty

`text`: *Ciąg*

Wartość typu *Ciąg* określająca tekst.

## <a name="return-values"></a>Wartości zwracane

*Wartość całkowita*

Wynikowa wartość numeryczna.

## <a name="example"></a>Przykład

Funkcja `LEN ("Sample")` zwraca wartość **6**.
