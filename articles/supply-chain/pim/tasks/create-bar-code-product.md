# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="14557-103">Tworzenie kodu kreskowego dla produktu</span><span class="sxs-lookup"><span data-stu-id="14557-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="14557-104">W tym temacie pokazano sposób ręcznego tworzenia kodu kreskowego przy użyciu przykładowego numeru towaru M0001.</span><span class="sxs-lookup"><span data-stu-id="14557-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="14557-105">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="14557-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="14557-106">Przejdź do **Zarządzanie informacjami o produktach\> Produkty \> Zwolnione produkty**.</span><span class="sxs-lookup"><span data-stu-id="14557-106">Go to **Product information management \> Products \> Released products**.</span></span>
1. <span data-ttu-id="14557-107">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="14557-107">In the list, find and select the desired record.</span></span> <span data-ttu-id="14557-108">W tym przykładzie zaznacz numer towaru **M0001**.</span><span class="sxs-lookup"><span data-stu-id="14557-108">For this example, select item number **M0001**.</span></span>
1. <span data-ttu-id="14557-109">W okienku akcji wybierz pozycję **Zarządzaj zapasami**.</span><span class="sxs-lookup"><span data-stu-id="14557-109">On the Action Pane, select **Manage inventory**.</span></span>
1. <span data-ttu-id="14557-110">Wybierz pozycję **Kody kreskowe**.</span><span class="sxs-lookup"><span data-stu-id="14557-110">Select **Bar codes**.</span></span>
1. <span data-ttu-id="14557-111">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="14557-111">Select **New**.</span></span>
1. <span data-ttu-id="14557-112">Zaznacz wybrany wiersz, który został utworzony na poniższej liście.</span><span class="sxs-lookup"><span data-stu-id="14557-112">Mark the selected row that is created in the list below.</span></span>
1. <span data-ttu-id="14557-113">W polu **Konfiguracja kodów** kreskowych wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="14557-113">In the **Barcode setup** field, enter or select a value.</span></span>
1. <span data-ttu-id="14557-114">W polu **Kod kreskowy** wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="14557-114">In the **Bar code** field, enter or select a value.</span></span>
1. <span data-ttu-id="14557-115">W polu **Kod kreskowy** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="14557-115">In the **Bar code** field, type a value.</span></span>  
1. <span data-ttu-id="14557-116">W polu **Ilość** wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="14557-116">In the **Quantity** field, enter a number.</span></span>
1. <span data-ttu-id="14557-117">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="14557-117">Select **Save**.</span></span>
1. <span data-ttu-id="14557-118">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="14557-118">Close the page.</span></span>
