# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="3aa52-103">Tworzenie kodu kreskowego dla produktu</span><span class="sxs-lookup"><span data-stu-id="3aa52-103">Create a bar code for a product</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="3aa52-104">W tym temacie pokazano sposób ręcznego tworzenia kodu kreskowego przy użyciu przykładowego numeru towaru M0001.</span><span class="sxs-lookup"><span data-stu-id="3aa52-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="3aa52-105">Dane wykorzystane do stworzenia tej procedury pochodzą z firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="3aa52-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="3aa52-106">Wybierz opcję **Obsługa zwolnionego produktu** na stronie głównej.</span><span class="sxs-lookup"><span data-stu-id="3aa52-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="3aa52-107">Przejdź do menu **Produkty > Zwolnione produkty** w sekcji **Linki**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="3aa52-108">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="3aa52-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="3aa52-109">W tym przykładzie zaznacz numer towaru **M0001**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="3aa52-110">W okienku akcji wybierz pozycję **Zarządzaj zapasami**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="3aa52-111">Wybierz pozycję **Kody kreskowe**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="3aa52-112">Wybierz pozycję **Nowy**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-112">Select **New**.</span></span>
7. <span data-ttu-id="3aa52-113">Zaznacz wybrany wiersz, który został utworzony na poniższej liście.</span><span class="sxs-lookup"><span data-stu-id="3aa52-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="3aa52-114">W polu **Konfiguracja kodów** kreskowych wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="3aa52-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="3aa52-115">W polu **Kod kreskowy** wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="3aa52-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="3aa52-116">W polu **Kod kreskowy** wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="3aa52-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="3aa52-117">W polu **Ilość** wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="3aa52-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="3aa52-118">Wybierz opcję **Zapisz**.</span><span class="sxs-lookup"><span data-stu-id="3aa52-118">Select **Save**.</span></span>
13. <span data-ttu-id="3aa52-119">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="3aa52-119">Close the page.</span></span>
