# <a name="create-a-value-stream"></a>Tworzenie strumienia wartości

[!include [banner](../../includes/banner.md)]

Strumienie wartości są jednostkami operacyjnymi, które grupują jeden lub wiele przepływów produkcji. Aby zdefiniować przepływ produkcji, należy najpierw zdefiniować strumień wartości.


## <a name="open-the-form"></a>Otwarcie formularza
1. Wybierz kolejno opcje Kontrola produkcji > Ustawienia > Przepływ produkcji oszczędnej > Strumienie wartości.

## <a name="create-a-new-value-stream"></a>Tworzenie nowego strumienia wartości
1. Kliknij przycisk Nowy, aby otworzyć rozwijane okno dialogowe.
2. W polu Nazwa wpisz wartość.

## <a name="select-a-value-stream-manager"></a>Wybór menedżera strumienia wartości
1. W polu Menedżer kliknij przycisk rozwijany, aby otworzyć wyszukiwanie.
    * Menedżera strumienia wartości wyznacza się w celu dokumentowania organizacji.  
2. Na liście znajdź i zaznacz odpowiedni rekord.
3. Na liście kliknij łącze w wybranym wierszu.

## <a name="save-and-close"></a>Zapisz i zamknij
1. Kliknij przycisk Zapisz.
2. Zamknij stronę.
3. Odśwież stronę.
