# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Tworzenie formuły przez kopiowanie z istniejącej formuły

[!include [task guide banner](../../includes/task-guide-banner.md)]

W tej procedurze pokazano, jak szybko utworzyć nową formułę przez skopiowanie z istniejącej formuły. Do utworzenia tej procedury można użyć danych firmy demonstracyjnej USMF, USP2 lub USPI.


## <a name="create-a-new-formula"></a>Tworzenie nowej formuły
1. Przejdź do okna Formuły.
2. Kliknij przycisk Nowy.
3. W polu Formuła wpisz wartość.
4. W polu Nazwa wpisz wartość.
5. W polu Oddział wprowadź lub wybierz wartość.
6. W polu Grupa towarów wprowadź lub wybierz wartość.
7. Kliknij przycisk Zapisz.

## <a name="copy-from-an-existing-formula"></a>Kopiowanie z istniejącej formuły
1. Kliknij opcję Kopiuj.
2. W polu Numer towaru wprowadź lub wybierz wartość.
3. W polu Wersja formuły wprowadź lub wybierz wartość.
4. Kliknij przycisk OK.

## <a name="revise-the-copied-formula"></a>Poprawianie skopiowanej formuły
1. Na liście znajdź i zaznacz odpowiedni rekord.
2. Kliknij przycisk Usuń.
3. Kliknij przycisk Tak.

## <a name="approve-formula"></a>Zatwierdź formułę
1. Kliknij przycisk Zatwierdź formułę.
2. W polu Zatwierdzone przez wprowadź lub wybierz wartość.
3. Kliknij przycisk OK.
4. Zamknij stronę.
