# <a name="not-enough-capacity-could-be-found"></a>Brak wystarczających zdolności produkcyjnych

## <a name="symptoms"></a>Objawy

Po uruchomieniu planowania może pojawić się następujący komunikat o błędzie:

> Nie można zaplanować zlecenia produkcyjnego %1. Brak wystarczających zdolności produkcyjnych.

Istnieje kilka przyczyn awarii aparatu planowania i wystawienia tego komunikatu o błędzie. Ten temat zawiera informacje, które pomogą w znalezieniu przyczyny głównej błędu, a następnie zmniejszeniu związanego z nim ryzyka.

## <a name="resolution"></a>Rozwiązanie

Istnieje kilka przyczyn awarii aparatu planowania i wystawienia komunikatu o błędzie. Aby uzyskać wskazówki, które pomogą znaleźć główną przyczynę błędu, a następnie go wyeliminować, zobacz temat [Usuwanie błędu aparatu planowania „Nie znaleziono wystarczających zdolności produkcyjnych”](/dynamics365/supply-chain/master-planning/not-enough-capacity-error-resolution.md).
