# <a name="purchase-orders-dont-show-translated-product-text"></a>W zamówieniach zakupu nie są wyświetlane przetłumaczone teksty produktów

Numer artykułu z bazy wiedzy: 4612098

## <a name="symptoms"></a>Objawy

Zdefiniowano tłumaczenia tekstowe nazw produktów. Jednak nazwy produktów są zawsze wyświetlane w języku oryginalnym, niezależnie od ustawień języka w aplikacji.

## <a name="resolution"></a>Rozdzielczość

Tłumaczenia produktów są wyświetlane tylko na dokumentach zewnętrznych. Nie są one pokazywane użytkownikom wewnętrznym. Aby uzyskać więcej informacji, zobacz [Tłumaczenia informacji dotyczących produktów — często zadawane pytania](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information).
