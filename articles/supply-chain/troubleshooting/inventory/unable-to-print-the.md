# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Raport listy pobrania nie zawiera nowych notatek dołączonych do arkusza

Numer artykułu z bazy wiedzy: 4611660

## <a name="symptoms"></a>Objawy

Raport listy pobrania nie zawiera nowych notatek dołączonych do arkusza pobrania.

## <a name="resolution"></a>Rozdzielczość

Raport listy pobrania zawiera tylko notatki dołączone do wiersza zamówienia sprzedaży i nagłówka zamówienia sprzedaży. Nie są w nim wyświetlane żadne nowe notatki dodane do arkusza faktur.
