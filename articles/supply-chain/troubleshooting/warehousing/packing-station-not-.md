# <a name="packing-station-doesnt-show-product-notes"></a>Stacja pakowania nie zawiera notatek produktu

Numer artykułu z bazy wiedzy: 4614615

## <a name="symptoms"></a>Objawy

Notatki dotyczące pakowania nie są wyświetlane w formularzu pakowania, gdy instrukcje pakowania są dodawane jako załącznik do produktu głównego lub wariantu produktu.

## <a name="resolution"></a>Rozdzielczość

System jest szybując tak jak zaprojektowany.

Bieżąca logika wyświetlania not od opakowań w formularzu pakowania wymaga, aby te notatki była skojarzona z wysyłkami.
