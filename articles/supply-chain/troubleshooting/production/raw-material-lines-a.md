# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Użytkownicy mogą cobrać wiersze materiałów dla produktów, które zostały zgłoszone jako gotowe

Numer artykułu z bazy wiedzy: 4614721

## <a name="symptoms"></a>Objawy

System umożliwia użytkownikom odbieranie linii surowców dla produktów, które zostały zgłoszone jako gotowe.

## <a name="resolution"></a>Rozdzielczość

System jest szybując tak jak zaprojektowany. Zużycie surowców można korygować, dopóki zlecenie produkcyjne nie osiągnie stanu *Zakończone*.
