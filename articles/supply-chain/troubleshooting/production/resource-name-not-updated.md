# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Nazwa zasobu nie została zaktualizowana po zmianie nazwiska pracownika w książce adresowej

## <a name="symptoms"></a>Objawy

Jeśli zmienisz nazwę pracownika w globalnej książce adresowej, pasująca nazwa zasobu nie zostanie zaktualizowana w wzorcu grupy zasobów.

## <a name="resolution"></a>Rozwiązanie

Pasująca nazwa zasobu nie jest obecnie aktualizowana automatycznie. Aby rozwiązać ten problem, należy ręcznie zaktualizować nazwę zasobu.
