# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Nie można zakończyć zlecenia produkcyjnego z powodu błędu wartości consumptionCost BOM

## <a name="symptoms"></a>Objawy

Przy próbie zakończenia zlecenia produkcyjnego pojawia się następujący komunikat o błędzie:

> Obliczanie wartości consumptionCost BOM musi dawać wynik ujemny przy wydaniu z magazynu.

## <a name="resolution"></a>Rozwiązanie

Ten błąd został rozwiązany w wersji 10.0.15.
