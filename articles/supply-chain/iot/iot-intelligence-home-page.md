# <a name="iot-intelligence-now-replaced-by-sensor-data-intelligence"></a>Analizy IoT zostaną teraz zastąpione przez Sensor Data Intelligence

[!include [banner](../../includes/banner.md)]

Funkcja Analiza Internetu rzeczy (IoT) została zaktualizowana do wersji 2.0 i jednocześnie jej nazwa zostanie zmieniona na *Sensor Data Intelligence*. Nowa wersja jest obecnie dostępna w wersji zapoznawczej dla rozwiązania Supply Chain Management w wersji 10.0.30 lub nowszej.

Funkcja Sensor Data Intelligence wprowadza nowy model wdrażania, w którym wymagane składniki usługi Azure są wdrażane we własnej subskrypcji Azure użytkownika, zamiast zarządzania nimi przez firmę Microsoft. Ten nowy model zwiększa możliwości dostosowania funkcji i jej integrację z platformami innych firm. W nowej wersji wprowadzono również obsługę dwóch nowych scenariuszy Internetu rzeczy (IoT) do zarządzania składnikami majątku.

Z powodu tych nadchodzących zmian firma Microsoft nie zaleca rozpoczynania jakichkolwiek nowych projektów w oparciu o istniejącą funkcję Analiza Internetu rzeczy (IoT). Jeśli chcesz dodać do swojego systemu możliwości IoT, zalecamy zamiast tego rozpoczęcie pracy z wersją zapoznawczą funkcji Sensor Data Intelligence. Aby uzyskać więcej informacji na temat funkcji Sensor Data Intelligence oraz jak z nią pracować, zobacz stronę [Sensor Data Intelligence — strona główna](../sensor-data-intelligence/sdi-home-page.md).
