# <a name="update-standard-costs-for-a-new-manufactured-item"></a><span data-ttu-id="31e05-103">Aktualizacja kosztów standardowych nowo produkowanego towaru</span><span class="sxs-lookup"><span data-stu-id="31e05-103">Update standard costs for a new manufactured item</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="31e05-104">Ten artykuł zawiera wskazówki dotyczące aktualizowania kosztów standardowych nowo produkowanego towaru.</span><span class="sxs-lookup"><span data-stu-id="31e05-104">This article provides guidance for updating standard costs for a new manufactured item.</span></span>

-   <span data-ttu-id="31e05-110">Przypisz typ wyceny **Koszt standardowy**.</span><span class="sxs-lookup"><span data-stu-id="31e05-110">Assign a costing type of **Standard cost**.</span></span>
-   <span data-ttu-id="31e05-111">Przypisz sugestywny identyfikator wskazujący zawartość wersji wyceny, np. **2016-AKTUALIZACJE**.</span><span class="sxs-lookup"><span data-stu-id="31e05-111">Assign a significant identifier that indicates the contents of the costing version, such as **2016-UPDATES**.</span></span>
-   <span data-ttu-id="31e05-112">W grupie pól **Zezwól na typy cen** upewnij się, że ustawienie **Koszt własny** ma wartość **Tak**.</span><span class="sxs-lookup"><span data-stu-id="31e05-112">In the **Allow price types** field group, make sure that **Cost price** is set to **Yes**.</span></span>
-   <span data-ttu-id="31e05-113">Pozwól na wprowadzanie rekordów kosztów we wszystkich oddziałach (tzn. pozostaw puste pole **Oddział**).</span><span class="sxs-lookup"><span data-stu-id="31e05-113">Allow cost records to be entered for all sites (that is, leave the **Site** field blank).</span></span> <span data-ttu-id="31e05-114">Jeśli określisz oddział, rekordy kosztów można wprowadzić tylko w tym oddziale.</span><span class="sxs-lookup"><span data-stu-id="31e05-114">If you enter a site, cost records can be entered only for that site.</span></span>

3.  <span data-ttu-id="31e05-123">Oblicz koszt nowo produkowanych towarów, korzystając ze strony **Obliczanie**.</span><span class="sxs-lookup"><span data-stu-id="31e05-123">Calculate the cost of new manufactured items by using the **Calculation** page.</span></span> <span data-ttu-id="31e05-124">Otwórz stronę **Obliczanie** ze strony **Obsługa wersji wyceny**, a następnie wybierz wersję wyceny data-ttu-id="31e05-126">W wielopoziomowej strukturze produktów może być również konieczne określenie dowolnego towaru nadrzędnego, który zawiera nowo produkowany towar jako składnik.</span><span class="sxs-lookup"><span data-stu-id="31e05-126">In a multilevel product structure, you might also have to specify any parent item that contains the new manufactured items as a component.</span></span> <span data-ttu-id="31e05-127">Wprowadź datę początkową dla obliczania listy składowej (BOM) odpowiadającą dacie rozpoczęcia produkcji nowo wytwarzanych towarów.</span><span class="sxs-lookup"><span data-stu-id="31e05-127">Enter a from-date for the bill of materials (BOM) calculation that corresponds to the start of manufacturing for the new manufactured items.</span></span> <span data-ttu-id="31e05-128">Data początkowa powinna się mieścić w zakresie dat obowiązywania wersji BOM towaru i wersji marszruty.</span><span class="sxs-lookup"><span data-stu-id="31e05-128">The from-date should be in the effective dates for the item’s BOM version and route version.</span></span> <span data-ttu-id="31e05-129">**Uwaga:** Brak rekordu kosztów może oznaczać, że jest to nowo produkowany towar.</span><span class="sxs-lookup"><span data-stu-id="31e05-129">**Note:** A missing cost record can indicate a new manufactured item.</span></span> <span have missing costs.</span></span>

5.  <span data-ttu-id="31e05-134">Za pomocą strony **Konfiguracja wersji wyceny** zmień flagę blokowania, aby umożliwić aktywowanie rekordów kosztów oczekujących w drugiej wersji wyceny.</span><span class="sxs-lookup"><span data-stu-id="31e05-134">Use the **Costing version setup** page to change the blocking flag to allow activation of the pending cost records in the second costing version.</span></span>
