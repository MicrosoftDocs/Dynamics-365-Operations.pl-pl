# <a name="set-up-a-transportation-tender"></a>Konfigurowanie metody płatności za transport

[!include [banner](../../includes/banner.md)]

W tej procedurze pokazano sposób konfigurowania metody płatności za transport. Zazwyczaj jest to realizowane przez koordynatora transportu. Procedurę można wykonać przy użyciu danych firmy demonstracyjnej USMF.


## <a name="select-a-route"></a>Wybór trasy
1. Wybierz kolejno opcje Zarządzanie transportem > Planowanie > Warsztat planowania wysyłki ładunku.
2. Wyczyść pole wyboru Ukryj wysłane i odebrane.
3. Zaznacz wiersz z ładunkiem o identyfikatorze 00006.
4. Kliknij opcję Ustawianie stawek i wybór trasy.
5. Kliknij opcję Marszruty.

## <a name="create-the-transportation-tender"></a>Tworzenie metody płatności za transport
1. Kliknij opcję Metody płatności za transport.
2. Kliknij przycisk Nowy.
3. Rozwiń sekcję Ogólne.
4. W polu Wymagane stawki wprowadź liczbę.
5. Kliknij przycisk Zapisz.
6. Kliknij przycisk Aktualizuj stan.
7. Kliknij przycisk Prześlij.
8. Wybierz trasę.
