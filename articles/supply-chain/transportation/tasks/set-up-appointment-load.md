# <a name="set-up-an-appointment-for-a-load"></a>Konfigurowanie terminu dla ładunku

[!include [banner](../../includes/banner.md)]

Ta procedura pokazuje, jak skonfigurować i zaplanować termin dokowania dla ładunku. Zazwyczaj jest to realizowane przez koordynatora transportu. Procedurę można wykonać przy użyciu danych firmy demonstracyjnej USMF.


## <a name="select-the-load"></a>Wybór ładunku
1. Wybierz kolejno opcje Zarządzanie transportem > Planowanie > Warsztat planowania wysyłki ładunku.
2. Wyczyść pole wyboru Ukryj wysłane i odebrane.
3. Na liście zaznacz ładunek, który ma stan Wysłano.
4. Kliknij opcję Transport.
5. Kliknij opcję Planowanie terminów.

## <a name="create-an-appointment"></a>Tworzenie terminu
1. Kliknij przycisk Nowy.
2. Wprowadź lub wybierz wartość w polu Reguła terminów.
3. Kliknij przycisk Zapisz.
4. Kliknij przycisk Aktualizuj stan.
5. Kliknij przycisk Akceptuj.
6. Kliknij przycisk Zapisz.
7. Zamknij stronę.
