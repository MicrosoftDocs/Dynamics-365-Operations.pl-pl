# <a name="configure-wave-processing"></a><span data-ttu-id="882d8-103">Konfigurowanie przetwarzania grupy czynności</span><span class="sxs-lookup"><span data-stu-id="882d8-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="882d8-104">W tym przewodniku opisano sposób konfigurowania kryteriów, które określają, jaka praca jest generowana dla magazynu podczas przetwarzania grupy czynności oraz czy grupy czynności są przetwarzane ręcznie czy automatycznie.</span><span class="sxs-lookup"><span data-stu-id="882d8-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="882d8-105">Kryteria można określić, definiując szablony grupy czynności i kwerendy, które dopasowują grupy czynności do zwolnionych wierszy w zamówieniach sprzedaży, zleceniach produkcyjnych i zleceniach Kanban.</span><span class="sxs-lookup"><span data-stu-id="882d8-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="882d8-106">Przetwarzanie grup czynności jest stosowane w magazynach, które używają funkcji modułu Zarządzanie magazynem, a nie tych, które używają funkcji modułu Zarządzanie zapasami.</span><span class="sxs-lookup"><span data-stu-id="882d8-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="882d8-107">Procedurę można wykonać przy użyciu danych firmy demonstracyjnej USMF.</span><span class="sxs-lookup"><span data-stu-id="882d8-107">You can run this procedure in demo data company USMF.</span></span>

2. <span data-ttu-id="882d8-109">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="882d8-109">Click New.</span></span>

    * <span data-ttu-id="882d8-119">Jeśli używasz firmy USMF, można wybrać magazyn 24.</span><span class="sxs-lookup"><span data-stu-id="882d8-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="882d8-120">W polu Automatyczne tworzenie grupy czynności ustaw wartość Tak.</span><span class="sxs-lookup"><span data-stu-id="882d8-120">Set the Automate wave creation field to Yes.</span></span>
    * <span data-ttu-id="882d8-121">Wybierz tę opcję, aby automatycznie utworzyć grupę czynności po zwolnieniu do magazynu zamówienia sprzedaży, zlecenia produkcyjnego lub karty kanban.</span><span class="sxs-lookup"><span data-stu-id="882d8-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>

    * <span data-ttu-id="882d8-123">Wybierz tę opcję, aby automatycznie przetworzyć grupę czynności i utworzyć pracę po zwolnieniu wiersza do magazynu.</span><span class="sxs-lookup"><span data-stu-id="882d8-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>

    * <span data-ttu-id="882d8-125">Gdy ta opcja jest wybrana, grupy czynności są uwalniane automatycznie.</span><span class="sxs-lookup"><span data-stu-id="882d8-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="882d8-126">Praca pobierania jest tworzona i udostępniana na urządzeniach przenośnych.</span><span class="sxs-lookup"><span data-stu-id="882d8-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="882d8-127">W opcji Przypisz do otwartych grup czynności zaznacz wartość Tak.</span><span class="sxs-lookup"><span data-stu-id="882d8-127">Set the Assign to open waves option to Yes.</span></span>

11. <span data-ttu-id="882d8-129">W opcji Automatycznie przetwórz grupę czynności na progu zaznacz wartość Tak.</span><span class="sxs-lookup"><span data-stu-id="882d8-129">Set the Process wave automatically at threshold option to Yes.</span></span>

    * <span data-ttu-id="882d8-136">Metody szablonów grup czynności pozwalają sterować kolejnością działań wykonywanych na każdej grupie podczas jej przetwarzania.</span><span class="sxs-lookup"><span data-stu-id="882d8-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it’s processed.</span></span> <span data-ttu-id="882d8-137">Na przykład może istnieć metoda uzupełniania zapasów dla grupy czynności.</span><span class="sxs-lookup"><span data-stu-id="882d8-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="882d8-138">Po dodaniu metody jest ona automatycznie wyświetlana w odpowiednim miejscu w sekwencji etapów.</span><span class="sxs-lookup"><span data-stu-id="882d8-138">When you add a method, it’s automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="882d8-139">Jeśli w opcji Automatyczne zwalnianie pracy uzupełniania zapasów zaznaczysz wartość Tak, trzeba dodać tutaj metodę uzupełniania zapasów.</span><span class="sxs-lookup"><span data-stu-id="882d8-139">If you’ve set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>
