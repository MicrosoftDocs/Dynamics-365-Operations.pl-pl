# <a name="troubleshoot-warehouse-setup"></a>Rozwiązywanie problemów z konfiguracją

[!include [banner](../includes/banner.md)]

W tym temacie opisano, jak rozwiązać typowe problemy, które mogą wystąpić podczas konfiguracji magazynów w Microsoft Dynamics 365 Supply Chain Management.

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>Nie można korzystać z żadnej roli z wyjątkiem administratora, aby uzyskać dostęp do emulatora aplikacji urządzenia przenośnego.

### <a name="issue-description"></a>Opis problemu

Użytkownik nie może korzystać z żadnej roli z wyjątkiem administratora, aby uzyskać dostęp do emulatora aplikacji urządzenia przenośnego.

### <a name="issue-resolution"></a>Rozwiązywanie problemów

Emulator aplikacji urządzenia przenośnego jest skonfigurowany do pracy tylko z kontem administratora. W przypadku wszelkiego testowania i procesów działających na żywo zaleca się używanie samej aplikacji Warehouse Management.
