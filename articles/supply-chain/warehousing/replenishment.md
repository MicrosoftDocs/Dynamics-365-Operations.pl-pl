<span data-ttu-id="561eb-104">W tym temacie opisano strategie uzupełniania zapasów, które są dostępne dla magazynów używających funkcji dostępnych w module Zarządzanie magazynem.</span><span class="sxs-lookup"><span data-stu-id="561eb-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="561eb-105">Informacje w tym temacie nie mają zastosowania do aplikacji magazynowej dostępnej w module Zarządzanie zapasami.</span><span class="sxs-lookup"><span data-stu-id="561eb-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

- <span data-ttu-id="561eb-107">**Uzupełnianie popytu grupy czynności** — Ta strategia tworzy pracę uzupełniania dla zamówień lub ładunków wychodzących, gdy zapasy są niedostępne podczas tworzenia pracy przez grupę czynności.</span><span class="sxs-lookup"><span data-stu-id="561eb-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="561eb-108">Na przykład pracę uzupełniania można utworzyć, jeśli ilość wymagana dla zamówienia sprzedaży nie jest dostępna w trakcie przetwarzania grupy czynności.</span><span class="sxs-lookup"><span data-stu-id="561eb-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="561eb-109">**Uzupełnianie według minimalnej/maksymalnej ilości** — Ta strategia wykorzystuje minimalne i maksymalne limity zapasów w celu określenia, kiedy należy uzupełnić lokalizacje.</span><span class="sxs-lookup"><span data-stu-id="561eb-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="561eb-110">Kryteria towaru i lokalizacji definiują zapasy oceniane przy uzupełnianiu.</span><span class="sxs-lookup"><span data-stu-id="561eb-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="561eb-111">Szablony uzupełniania zapasów według ilości minimalnej/maksymalnej są podstawowym mechanizmem utrzymywania optymalnego poziomu zapasów w lokalizacjach pobrania.</span><span class="sxs-lookup"><span data-stu-id="561eb-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="561eb-112">Aby pomóc w zagwarantowaniu, że dostępna ilość rozpakowanych zapasów wystarcza do zaspokojenia popytu grupy czynności, można używać uzupełniania zapasów dla popytu dodatkowo między cyklami uzupełnienia według ilości minimalnej/maksymalnej.</span><span class="sxs-lookup"><span data-stu-id="561eb-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="561eb-113">**Uzupełnianie zapasów popytu ładunku** — Ta strategia sumuje popyt kilku ładunków i tworzy pracę uzupełniania niezbędną w celu dostarczenia zapasów do odpowiednich lokalizacji pobrania.</span><span class="sxs-lookup"><span data-stu-id="561eb-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="561eb-114">Ta strategia pomaga zagwarantować, że tworzone ładunki mogą po zwolnieniu zostać pobrane z magazynu.</span><span class="sxs-lookup"><span data-stu-id="561eb-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="561eb-115">**Natychmiastowe uzupełnienie** — ta strategia powoduje uzupełnienie zapasów przed uruchomieniem grupy czynności, jeżeli alokacja nie powiedzie się dla wiersza dyrektywy lokalizacji zawierającego szablon uzupełnienia zapasów.</span><span class="sxs-lookup"><span data-stu-id="561eb-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="561eb-116">Wszystkie cztery strategie tworzą pracę uzupełniania opartą na szablonie uzupełniania zapasów.</span><span class="sxs-lookup"><span data-stu-id="561eb-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

<span data-ttu-id="561eb-133">Uzupełnienie zapasów popytu jest obsługiwane w przypadku zamówień sprzedaży, zamówień przeniesienia, zleceń produkcyjnych i kart Kanban.</span><span class="sxs-lookup"><span data-stu-id="561eb-133">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span>
