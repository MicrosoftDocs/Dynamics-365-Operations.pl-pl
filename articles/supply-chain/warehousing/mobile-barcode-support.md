[!include [banner](../includes/banner.md)]

<span data-ttu-id="fe876-104">Ponieważ system Android to projekt typu open source, każdy producent sprzętu do skanerów kodów kreskowych w magazynach może opracować urządzenie działające pod kontrolą systemu Android.</span><span class="sxs-lookup"><span data-stu-id="fe876-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="fe876-105">Urządzenie jest zgodne z systemem Android tylko wtedy, gdy umożliwia uruchamianie aplikacji dla środowiska uruchomieniowego systemu Android.</span><span class="sxs-lookup"><span data-stu-id="fe876-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="fe876-106">Dostawca sprzętu może jednak modyfikować system i tworzyć nakładki dla wersji systemu Android działającej na jego sprzęcie.</span><span class="sxs-lookup"><span data-stu-id="fe876-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="fe876-107">Firma Microsoft nie ponosi odpowiedzialności za zapewnienie zgodności mobilnej aplikacji do skanowania kodów kreskowych dla systemu Android z urządzeniem producenta do skanowania kodów kreskowych i działającą na nim wersją systemu Android.</span><span class="sxs-lookup"><span data-stu-id="fe876-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="fe876-110">Zalecamy, aby odbiorcy przetestowali aplikację mobilną do skanowania Magazyn na wybranych urządzeniach przed ich zakupem.</span><span class="sxs-lookup"><span data-stu-id="fe876-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
