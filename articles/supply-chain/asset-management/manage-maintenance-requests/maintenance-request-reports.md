## <a name="create-a-maintenance-request-details-report"></a><span data-ttu-id="c5e0c-106">Tworzenie raportu Szczegóły żądania konserwacji</span><span class="sxs-lookup"><span data-stu-id="c5e0c-106">Create a Maintenance request details report</span></span>

<span data-ttu-id="c5e0c-107">Raport **Szczegóły żądania konserwacji** zawiera różne informacje związane z żądaniami konserwacji.</span><span class="sxs-lookup"><span data-stu-id="c5e0c-107">The **Maintenance request details** report shows various information that is related to maintenance requests.</span></span>

1. <span data-ttu-id="c5e0c-108">Wybierz **Zarządzanie składnikami majątku** \> **Raporty** \> **Żądania konserwacji** \> **Szczegóły żądania konserwacji**.</span><span class="sxs-lookup"><span data-stu-id="c5e0c-108">Select **Asset management** \> **Reports** \> **Maintenance requests** \> **Maintenance request details**.</span></span>
2. <span data-ttu-id="c5e0c-109">Na skróconej karcie **Rekordy do uwzględnienia** można wybrać określone żądania konserwacji do uwzględnienia w raporcie.</span><span class="sxs-lookup"><span data-stu-id="c5e0c-109">On the **Records to include** FastTab, you can select specific maintenance requests to include on the report.</span></span>
3. <span data-ttu-id="c5e0c-110">Na skróconej karcie **Uruchom w tle** można skonfigurować generowanie raportów jako zadanie wsadowe, zgodnie z żądaniem.</span><span class="sxs-lookup"><span data-stu-id="c5e0c-110">On the **Run in the background** FastTab, you can set up report generation as a batch job, as you require.</span></span>
