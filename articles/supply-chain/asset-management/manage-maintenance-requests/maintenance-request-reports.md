# <a name="maintenance-request-reports"></a><span data-ttu-id="32381-103">Raporty żądania konserwacji</span><span class="sxs-lookup"><span data-stu-id="32381-103">Maintenance request reports</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="32381-104">W module Zarządzanie składnikami majątku można wygenerować dwa raporty, które są związane z żądaniami konserwacji.</span><span class="sxs-lookup"><span data-stu-id="32381-104">In Asset Management, you can generate two reports that are related to maintenance requests.</span></span> <span data-ttu-id="32381-105">Jeden raport pokazuje szczegóły, a drugi raport zawiera listę, która może służyć do planowania i prowadzenia kolejnych czynności.</span><span class="sxs-lookup"><span data-stu-id="32381-105">One report shows details, and the other report provides a list that can be used for planning and follow-up.</span></span>

## <a name="create-a-maintenance-request-details-report"></a><span data-ttu-id="32381-106">Tworzenie raportu Szczegóły żądania konserwacji</span><span class="sxs-lookup"><span data-stu-id="32381-106">Create a Maintenance request details report</span></span>

<span data-ttu-id="32381-107">Raport **Szczegóły żądania konserwacji** zawiera różne informacje związane z żądaniami konserwacji.</span><span class="sxs-lookup"><span data-stu-id="32381-107">The **Maintenance request details** report shows various information that is related to maintenance requests.</span></span>

1. <span data-ttu-id="32381-108">Wybierz **Zarządzanie składnikami majątku** \> **Raporty** \> **Żądania konserwacji** \> **Szczegóły żądania konserwacji**.</span><span class="sxs-lookup"><span data-stu-id="32381-108">Select **Asset management** \> **Reports** \> **Maintenance requests** \> **Maintenance request details**.</span></span>
2. <span data-ttu-id="32381-109">Na skróconej karcie **Rekordy do uwzględnienia** można wybrać określone żądania konserwacji do uwzględnienia w raporcie.</span><span class="sxs-lookup"><span data-stu-id="32381-109">On the **Records to include** FastTab, you can select specific maintenance requests to include on the report.</span></span>
3. <span data-ttu-id="32381-110">Na skróconej karcie **Uruchom w tle** można skonfigurować generowanie raportów jako zadanie wsadowe, zgodnie z żądaniem.</span><span class="sxs-lookup"><span data-stu-id="32381-110">On the **Run in the background** FastTab, you can set up report generation as a batch job, as you require.</span></span>

<span data-ttu-id="32381-112">Na poniższej ilustracji pokazano przykład raportu **Szczegóły żądania konserwacji**.</span><span class="sxs-lookup"><span data-stu-id="32381-112">The following illustration shows an example of the **Maintenance request details** report.</span></span>

<span data-ttu-id="32381-120">Na poniższej ilustracji przedstawiono przykładowy raport **Lista żądań konserwacji** dla wszystkich aktywnych żądań konserwacji.</span><span class="sxs-lookup"><span data-stu-id="32381-120">The following illustration shows an example of the **Maintenance request list** report for all active maintenance requests.</span></span>
