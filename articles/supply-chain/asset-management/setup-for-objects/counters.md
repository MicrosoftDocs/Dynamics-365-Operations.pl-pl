## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="97fc8-119">Tworzenie typów liczników dla rejestracji liczników składników majątku</span><span class="sxs-lookup"><span data-stu-id="97fc8-119">Create counter types for asset counter registrations</span></span>

1. <span data-ttu-id="97fc8-120">Wybierz **Zarządzaj składnikiem majątku** > **Konfiguracja** > **Typy składników majątku** > **Liczniki**.</span><span class="sxs-lookup"><span data-stu-id="97fc8-120">Select **Asset management** > **Setup** > **Asset types** > **Counters**.</span></span>

3. <span data-ttu-id="97fc8-122">Wstaw Identyfikator w polu **Licznik** oraz nazwę licznika w polu **Nazwa**.</span><span class="sxs-lookup"><span data-stu-id="97fc8-122">Insert an ID in the **Counter** field, and a counter name in the **Name** field.</span></span>

6. <span data-ttu-id="97fc8-125">Zaznacz opcję „tak” na przycisku przełącznika **Dziedzicz wartości licznika**, jeśli zasoby podrzędne w strukturze zasobów powinny automatycznie dziedziczyć rejestracje miar składnika majątku dokonane w zasobie nadrzędnym.</span><span class="sxs-lookup"><span data-stu-id="97fc8-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit asset measure registrations made on the parent asset.</span></span>
7. <span data-ttu-id="97fc8-126">W polu **Zagregowana suma** wybierz metodę sumowania stosowaną dla miary składnika majątku przy użyciu tego typu miary składnika majątku.</span><span class="sxs-lookup"><span data-stu-id="97fc8-126">In the **Total aggregate** field, select the summation method to be used for an asset measure using this asset measure type.</span></span> <span data-ttu-id="97fc8-127">„Suma” to standardowy wybór używany do ciągłego dodawania zarejestrowanych wartości do wartości całkowitej.</span><span class="sxs-lookup"><span data-stu-id="97fc8-127">"Sum" is the standard selection used to continuously add registered values to the total value.</span></span> <span data-ttu-id="97fc8-128">„Średnia” może być użyta, jeśli miara składnika majątku jest ustawiona pod kątem monitorowania progu, na przykład w odniesieniu do temperatury, drgań lub zużycia danego składnika majątku.</span><span class="sxs-lookup"><span data-stu-id="97fc8-128">"Average" can be used if an asset measure is set up to monitor a threshold, for example, regarding temperature, vibrations, or wear and tear on an asset.</span></span>
