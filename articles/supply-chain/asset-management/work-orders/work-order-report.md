# <a name="work-order-report"></a><span data-ttu-id="3da89-103">Raport zlecenia pracy</span><span class="sxs-lookup"><span data-stu-id="3da89-103">Work order report</span></span>

[!include [banner](../../includes/banner.md)]


<span data-ttu-id="3da89-104">Istnieje możliwość wygenerowania raportu zlecenia pracy, który pokazuje szczegółowe informacje o jednym lub więcej zleniach pracy, które są wybrane.</span><span class="sxs-lookup"><span data-stu-id="3da89-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="3da89-105">Wybierz **Zarządzanie składnikami majątku** > **Raporty** > **Zlecenia pracy** > **Raport zlecenia pracy**.</span><span class="sxs-lookup"><span data-stu-id="3da89-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="3da89-106">Aby określić szczegóły, które są wymagane, należy wybrać opcję „tak"/"nie” na przycisku przełączania i wypełnić pola w razie potrzeby.</span><span class="sxs-lookup"><span data-stu-id="3da89-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="3da89-107">Sekcja **Ustawienia drukowania** umożliwia określenie, czy załączniki mają być uwzględniane w konfiguracji odpowiedniego typu zadania konserwacji na druku.</span><span class="sxs-lookup"><span data-stu-id="3da89-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="3da89-108">Na skróconej karcie **Rekordy do uwzględnienia** można odfiltrować zawartość raportu według **Zlecenia pracy**.</span><span class="sxs-lookup"><span data-stu-id="3da89-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

4. <span data-ttu-id="3da89-109">Na skróconej karcie **Uruchom w tle** można skonfigurować generowanie raportu zlecenia pracy jako zadanie wsadowe, zgodnie z żądaniem.</span><span class="sxs-lookup"><span data-stu-id="3da89-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>

5. <span data-ttu-id="3da89-110">Kliknij przycisk **OK**. Raport zostanie wygenerowany.</span><span class="sxs-lookup"><span data-stu-id="3da89-110">Select **OK** to generate the report.</span></span>
