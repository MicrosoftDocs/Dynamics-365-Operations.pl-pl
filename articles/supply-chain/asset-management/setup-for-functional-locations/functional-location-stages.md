# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="65e79-103">Stany cyklu życia lokalizacji czynności konserwacyjnych</span><span class="sxs-lookup"><span data-stu-id="65e79-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="65e79-104">W tym temacie opisano sposób konfigurowania stanów cyklu życia lokalizacji czynności konserwacyjnych i modeli cyklu życia w module Zarządzanie składnikami majątku.</span><span class="sxs-lookup"><span data-stu-id="65e79-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="65e79-105">Stany cyklu życia lokalizacji czynności konserwacyjnych definiują stany, przez które może przechodzić lokalizacja czynności konserwacyjnych, np. utworzona, aktywna i zakończona.</span><span class="sxs-lookup"><span data-stu-id="65e79-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="65e79-106">Wszystkie lokalizacje czynności konserwacyjnych, niezależnie od ich stanu cyklu życia, można wyświetlić na stronie listy **Wszystkie lokalizacje czynności konserwacyjnych**.</span><span class="sxs-lookup"><span data-stu-id="65e79-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="65e79-107">Stan lokalizacji funkcjonalnej można zmienić, zaznaczając go na stronie listy **Wszystkie lokalizacje czynności konserwacyjnych** i wybierając pozycję **Aktualizuj stan lokalizacji czynności konserwacyjnych**.</span><span class="sxs-lookup"><span data-stu-id="65e79-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="65e79-108">Ustawienia stanów cyklu życia lokalizacji czynności konserwacyjnych</span><span class="sxs-lookup"><span data-stu-id="65e79-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="65e79-109">Wybierz pozycję **Zarządzanie składnikami majątku** > **Ustawienia** > **Lokalizacje czynności konserwacyjnych** > **Stany cyklu życia**.</span><span class="sxs-lookup"><span data-stu-id="65e79-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="65e79-110">Wybierz pozycję **Nowy**, aby utworzyć nowy stan lokalizacji czynności konserwacyjnych.</span><span class="sxs-lookup"><span data-stu-id="65e79-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="65e79-111">Wstaw identyfikator stanu w polu **Stan cyklu życia** i nazwę stanu lokalizacji czynności konserwacyjnych w polu **Nazwa**.</span><span class="sxs-lookup"><span data-stu-id="65e79-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="65e79-112">W polu **Modele cyklu życia** można zobaczyć liczbę modeli cyklu życia lokalizacji czynności konserwacyjnych, które używają stanu lokalizacji czynności konserwacyjnych.</span><span class="sxs-lookup"><span data-stu-id="65e79-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="65e79-113">Na skróconej karcie **Ogólne** wybierz opcję „Tak” na przełączniku **Aktywna**, jeśli lokalizacja czynności konserwacyjnych powinna być aktywna w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="65e79-114">Wybierz „Tak” na przełączniku **Utwórz składniki majątku**, jeśli powinno być możliwe automatyczne tworzenie składników majątku o takiej samej nazwie jak lokalizacja czynności konserwacyjnych i zainstalować go w lokalizacji czynności konserwacyjnych w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="65e79-115">Ten przełącznik odnosi się do pola **Typ składnika majątku** na skróconej karcie **Ogólne** w formularzu **Typu lokalizacji czynności konserwacyjnych** (**Zarządzanie składnikami majątku** > **Ustawienia** > **Lokalizacje czynności konserwacyjnych** > **Typy lokalizacji czynności konserwacyjnych**).</span><span class="sxs-lookup"><span data-stu-id="65e79-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="65e79-116">Wybierz „Tak” na przełączniku **Zmień nazwę lokalizacji**, jeśli powinna być możliwa zmiana nazwy lokalizacji czynności konserwacyjnych w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="65e79-117">Wybierz „Tak” na przełączniku **Nowe lokalizacje podrzędne**, jeśli powinno być możliwe dodawanie nowych lokalizacji podrzędnych do lokalizacji czynności konserwacyjnych w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="65e79-118">Wybierz „Tak” na przełączniku **Zainstaluj składniki majątku**, jeśli powinno być możliwe instalowanie składników majątku w lokalizacji czynności konserwacyjnych w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="65e79-119">Wybierz „Tak” na przełączniku **Usuń lokalizację czynności konserwacyjnych majątku**, jeśli powinno być możliwe usuwanie lokalizacji czynności konserwacyjnych w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="65e79-120">Wybierz stan składnika majątku w polu **Stan cyklu życia**, jeśli chcesz, aby stan cyklu życia składnika majątku dla wszystkich składników majątku zainstalowanych w lokalizacji czynności konserwacyjnych był automatycznie aktualizowany w tym stanie.</span><span class="sxs-lookup"><span data-stu-id="65e79-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="65e79-121">Przykład: jeśli zamkniesz lokalizację czynności konserwacyjnych i ustawisz stan cyklu życia lokalizacji czynności konserwacyjnych na „zakończony”, możesz chcieć automatycznie zmienić stan cyklu życia składnika majątku zainstalowanego w tej lokalizacji czynności konserwacyjnych na „nieużywany”.</span><span class="sxs-lookup"><span data-stu-id="65e79-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="65e79-122">Stany cyklu życia lokalizacji czynności konserwacyjnych, modele cyklu życia i typy są powiązane i używane w taki sam sposób jak stany cyklu życia zlecenia pracy, modele cyklu życia zlecenia pracy i typy zleceń pracy.</span><span class="sxs-lookup"><span data-stu-id="65e79-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="65e79-123">Konfigurowanie modeli cyklu życia lokalizacji czynności konserwacyjnych</span><span class="sxs-lookup"><span data-stu-id="65e79-123">Set up functional location lifecycle models</span></span>

<span data-ttu-id="65e79-124">Po utworzeniu stanów cyklu życia wymaganych dla lokalizacji funkcjonalnych można je podzielić na grupy.</span><span class="sxs-lookup"><span data-stu-id="65e79-124">When you have created the lifecycle states required for your functional locations, they can be divided into groups.</span></span> <span data-ttu-id="65e79-125">Można to zrobić, aby utworzyć przepływ modelu cyklu życia, który może być używany dla różnych typów lokalizacji czynności konserwacyjnych.</span><span class="sxs-lookup"><span data-stu-id="65e79-125">This is done to create the lifecycle model flow that may be used for different types of functional locations.</span></span> <span data-ttu-id="65e79-126">Jako minimum należy utworzyć jeden standardowy model cyklu życia lokalizacji czynności konserwacyjnych.</span><span class="sxs-lookup"><span data-stu-id="65e79-126">As a minimum, one standard functional location lifecycle model should be created.</span></span>

1. <span data-ttu-id="65e79-127">Wybierz pozycję **Zarządzanie składnikami majątku** > **Ustawienia** > **Lokalizacje czynności konserwacyjnych** > **Modele cyklu życia**.</span><span class="sxs-lookup"><span data-stu-id="65e79-127">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle models**.</span></span>
2. <span data-ttu-id="65e79-128">Wybierz pozycję **Nowy**, aby utworzyć nowy model cyklu życia.</span><span class="sxs-lookup"><span data-stu-id="65e79-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="65e79-129">Wstaw identyfikator modelu cyklu życia w polu **Model cyklu życia** i nazwę modelu cyklu życia w polu **Nazwa**.</span><span class="sxs-lookup"><span data-stu-id="65e79-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="65e79-130">W polach **Typy lokalizacji czynności konserwacyjnych** i **Stany cyklów życia** można zobaczyć liczbę typów lokalizacji czynności konserwacyjnych, które używają modelu cyklu życia i liczbę stanów wybranych w modelu cyklu życia.</span><span class="sxs-lookup"><span data-stu-id="65e79-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="65e79-131">Na skróconej karcie **Stany cyklu życia** wybierz stany, które powinny być uwzględnione w modelu.</span><span class="sxs-lookup"><span data-stu-id="65e79-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="65e79-132">W tym celu kliknij stan w sekcji **Pozostałe stany cyklu życia** i kliknij przycisk ![strzałki do przodu](media/02-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="65e79-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="65e79-133">Jeśli chcesz wybrać wszystkie dostępne stany dla modelu, kliknij przycisk ![Wybierz wszystkie dostępne etapy](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="65e79-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="65e79-134">Wszystkie stany są przenoszone do sekcji **Wybrane cykle życia**.</span><span class="sxs-lookup"><span data-stu-id="65e79-134">All states are transferred to the **Lifecycle states selected** section.</span></span>
6. <span data-ttu-id="65e79-135">Jeśli chcesz usunąć wybrany stan z modelu, wybierz stan w sekcji **Wybrane stany cyklu życia**, a następnie wybierz przycisk ![strzałki wstecz](media/04-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="65e79-135">If you want to remove a selected state from the model, select the state in the **Lifecycle states selected** section and then select the ![back arrow](media/04-setup-for-functional-locations.png) button.</span></span>
7. <span data-ttu-id="65e79-136">Wybierz pozycję **Aktualizacje stanu cyklu życia**, aby określić, które stany cyklu życia mogą być zgodne z wybranym stanem.</span><span class="sxs-lookup"><span data-stu-id="65e79-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
