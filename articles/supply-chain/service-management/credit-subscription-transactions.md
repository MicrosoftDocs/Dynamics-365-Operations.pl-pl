## <a name="credit-subscription-transactions"></a><span data-ttu-id="c63ef-104">Transakcje subskrypcji kredytu</span><span class="sxs-lookup"><span data-stu-id="c63ef-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="c63ef-105">Kliknij kolejno opcje **Zarządzanie serwisem** \> **Wspólne** \> **Subskrypcje serwisowe** \> **Wszystkie subskrypcje serwisu**.</span><span class="sxs-lookup"><span data-stu-id="c63ef-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="c63ef-106">Zaznacz subskrypcję połączoną z transakcją subskrypcji, dla której ma zostać utworzona faktura korygująca.</span><span class="sxs-lookup"><span data-stu-id="c63ef-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="c63ef-107">Kliknij kartę **Analizuj**, a następnie w okienku akcji kliknij przycisk **Transakcje opłat**.</span><span class="sxs-lookup"><span data-stu-id="c63ef-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="c63ef-108">W formularzu **Transakcje opłat** zaznacz transakcję, dla której ma zostać utworzona faktura korygująca.</span><span class="sxs-lookup"><span data-stu-id="c63ef-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="c63ef-109">Kliknij kolejno opcje **Funkcje** \> **Wybierz do faktury korygującej**</span><span class="sxs-lookup"><span data-stu-id="c63ef-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="c63ef-110">W formularzu **Wybierz do faktury korygującej** wybierz transakcję, dla której chcesz utworzyć uznanie, i kliknij przycisk **OK**.</span><span class="sxs-lookup"><span data-stu-id="c63ef-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="c63ef-111">Podczas tworzenia faktury korygującej należy upewnić się, że wybrano opcję <STRONG>Faktury korygujące</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="c63ef-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="c63ef-112">Znajduje się ona na liście <STRONG>Metoda fakturowania</STRONG> w oknie dialogowym <STRONG>Utwórz fakturę</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="c63ef-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="c63ef-113">Jeśli pole **Wycofaj naliczenia przy korekcie** w formularzu **Parametry modułu Zarządzanie serwisem** ma wartość **Ręcznie**, przed utworzeniem propozycji faktury korygującej dla transakcji należy wycofać poszczególne transakcje z naliczonym przychodem.</span><span class="sxs-lookup"><span data-stu-id="c63ef-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
