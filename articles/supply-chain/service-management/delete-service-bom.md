# <a name="delete-a-service-bom"></a><span data-ttu-id="66866-103">Usuwanie BOM serwisu</span><span class="sxs-lookup"><span data-stu-id="66866-103">Delete a service BOM</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="66866-104">Kliknij kolejno opcje **Zarządzanie serwisem** \> **Wspólne** \> **Umowy serwisowe** \> **Umowy serwisowe**.</span><span class="sxs-lookup"><span data-stu-id="66866-104">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span> <span data-ttu-id="66866-105">Wybierz umowę serwisową, dla której chcesz usunąć pozycje na liście składowej (BOM).</span><span class="sxs-lookup"><span data-stu-id="66866-105">Select the service agreement that you want to delete a service bill of materials (BOM) from.</span></span>

2.  <span data-ttu-id="66866-106">W **okienku akcji** kliknij pozycję **Przedmioty serwisu**.</span><span class="sxs-lookup"><span data-stu-id="66866-106">On the **Action Pane**, click **Service objects**.</span></span>

3.  <span data-ttu-id="66866-107">W formularzu **Przedmioty serwisu** wybierz obiekt serwisowy, z którego chcesz usunąć BOM serwisu.</span><span class="sxs-lookup"><span data-stu-id="66866-107">In the **Service objects** form, select the service object that you want to delete a service BOM from.</span></span> <span data-ttu-id="66866-108">Kliknij opcję **Funkcje** i wybierz opcję **Usuń BOM serwisu**.</span><span class="sxs-lookup"><span data-stu-id="66866-108">Click **Functions**, and then select **Delete service BOM**.</span></span>

4.  <span data-ttu-id="66866-109">Kliknij przycisk **Tak**.</span><span class="sxs-lookup"><span data-stu-id="66866-109">Click **Yes**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="66866-110">Jeśli nie zastosowano szablonu BOM, BOM serwisu można usunąć z formularza <STRONG>Szablony BOM</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="66866-110">If you have not applied a template BOM, the service BOM can be deleted from the <STRONG>Template BOMs</STRONG> form.</span></span> <span data-ttu-id="66866-111">Jeśli jednak zastosowano BOM szablonu, usługi BOM nie można usunąć przy użyciu tej metody.</span><span class="sxs-lookup"><span data-stu-id="66866-111">If you have applied a template BOM, the service BOM cannot be deleted by using this method.</span></span></P>
