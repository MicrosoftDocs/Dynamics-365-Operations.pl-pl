# <a name="set-up-subscription-groups"></a><span data-ttu-id="71304-103">Ustawianie grup subskrypcji</span><span class="sxs-lookup"><span data-stu-id="71304-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="71304-104">Przejdź do **Zarządzanie serwisem** \> **Ustawienia** \> **Subskrypcje serwisowe** \> **Grupy subskrypcji**.</span><span class="sxs-lookup"><span data-stu-id="71304-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="71304-105">Wybierz pozycję **Nowy**, aby utworzyć nową grupę subskrypcji.</span><span class="sxs-lookup"><span data-stu-id="71304-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="71304-106">W polu **Grupa subskrypcji** wprowadź opisową nazwę nowej grupy subskrypcji.</span><span class="sxs-lookup"><span data-stu-id="71304-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="71304-107">W polu **Fakturowanie** wybierz częstotliwość fakturowania w subskrypcji.</span><span class="sxs-lookup"><span data-stu-id="71304-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="71304-108">Częstotliwość fakturowania jest określona przez kod okresu skonfigurowany w formularzu **Typy okresu**.</span><span class="sxs-lookup"><span data-stu-id="71304-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="71304-109">Aby naliczać przychody z transakcji subskrypcji, zaznacz pole wyboru **Nalicz przychód**, a następnie w polu **Naliczenie** wybierz częstotliwość.</span><span class="sxs-lookup"><span data-stu-id="71304-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="71304-110">Częstotliwość naliczania zależy od kodu okresu skonfigurowanego w formularzu **Okresy**.</span><span class="sxs-lookup"><span data-stu-id="71304-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
