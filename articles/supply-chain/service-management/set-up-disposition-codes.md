<span data-ttu-id="3a83b-107">Można także skonfigurować kod przyczyny pomagający w wyjaśnieniu przyczyn zwrócenia towaru.</span><span class="sxs-lookup"><span data-stu-id="3a83b-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="3a83b-108">Aby uzyskać więcej informacji na temat kodów przyczyn, zobacz [Konfigurowanie kodów przyczyny zwrotu](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="3a83b-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="3a83b-109">Przejdź do **Sprzedaż i marketing** \> **Ustawienia** \> **Zamówienia sprzedaży** \> **Zwroty** \> **Kody dyspozycji**.</span><span class="sxs-lookup"><span data-stu-id="3a83b-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

4.  <span data-ttu-id="3a83b-112">W celu skojarzenia z tym kodem dyspozycji żądanych opłat wymaganych od odbiorcy wybierz przycisk **Opłaty**. 5.  <span data-ttu-id="3a83b-113">Aby zdefiniować jakiekolwiek kody zewnętrzne zgodne z firmowymi kodami dyspozycji, wybierz przycisk **Kody zewnętrzne**.
