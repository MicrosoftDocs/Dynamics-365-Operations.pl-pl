[!include [banner](../includes/banner.md)]


<span data-ttu-id="4137f-104">Można utworzyć kody dyspozycji w celu określania sposobu przetwarzania towaru zwracanego od odbiorcy.</span><span class="sxs-lookup"><span data-stu-id="4137f-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="4137f-105">Na przykład, utworzyć kod dyspozycji o nazwie **Naprawa i zwrot**, aby wskazać, że zwrócony towar będzie naprawiony i następnie zwrócony odbiorcy.</span><span class="sxs-lookup"><span data-stu-id="4137f-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="4137f-106">Aby uzyskać więcej przykładów kodów dyspozycji, które są zwykle używane dla towarów zwracanych od odbiorców, zobacz [Określanie sposobu likwidacji zwróconych towarów](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="4137f-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="4137f-107">Można także skonfigurować kod przyczyny pomagający w wyjaśnieniu przyczyn zwrócenia towaru.</span><span class="sxs-lookup"><span data-stu-id="4137f-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="4137f-108">Aby uzyskać więcej informacji na temat kodów przyczyn, zobacz [Konfigurowanie kodów przyczyny zwrotu](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="4137f-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="4137f-109">Kliknij kolejno opcje **Sprzedaż i marketing** \> **Ustawienia** \> **Zamówienia sprzedaży** \> **Zwroty** \> **Kody dyspozycji**.</span><span class="sxs-lookup"><span data-stu-id="4137f-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="4137f-110">Naciśnij kombinację klawiszy CTRL+N lub kliknij przycisk **Nowy**, aby utworzyć nowy kod dyspozycji.</span><span class="sxs-lookup"><span data-stu-id="4137f-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="4137f-111">Wprowadź unikatową nazwę opisową, zaznacz akcję i wpisz opis kodu dyspozycji.</span><span class="sxs-lookup"><span data-stu-id="4137f-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="4137f-112">W celu skojarzenia z tym kodem dyspozycji żądanych opłat wymaganych od odbiorcy kliknij przycisk **Opłaty**. Spowoduje to otwarcie formularza **Ustaw opłaty**.</span><span class="sxs-lookup"><span data-stu-id="4137f-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="4137f-113">Aby zdefiniować jakiekolwiek kody zewnętrzne zgodne z firmowymi kodami dyspozycji, kliknij przycisk **Kody zewnętrzne**. Spowoduje to otwarcie formularza **Kody zewnętrzne**.</span><span class="sxs-lookup"><span data-stu-id="4137f-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
