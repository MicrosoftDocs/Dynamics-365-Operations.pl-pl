[!include [banner](../includes/banner.md)]


<span data-ttu-id="01f63-104">Formularz **Usuwanie zamówień zwrotu** umożliwia usunięcie wielu zamówień zwrotu.</span><span class="sxs-lookup"><span data-stu-id="01f63-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="01f63-105">Kliknij kolejno opcje **Sprzedaż i marketing** \> **Okresowe** \> **Oczyszczanie** \> **Usuwanie zamówień zwrotu**.</span><span class="sxs-lookup"><span data-stu-id="01f63-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

3.  <span data-ttu-id="01f63-107">Określ kryteria, a następnie kliknij przycisk **OK**, aby wrócić do formularza **Usuwanie zamówień zwrotu**.</span><span class="sxs-lookup"><span data-stu-id="01f63-107">Specify the criteria, and then click **OK** to return to the **Delete return order** form.</span></span> <span data-ttu-id="01f63-108">Zamówienia zwrotu utworzone w firmie są wyświetlane na podstawie określonych kryteriów.</span><span class="sxs-lookup"><span data-stu-id="01f63-108">Return orders that have been created in the legal entity are displayed, based on the specified criteria.</span></span>
