# <a name="create-time-windows"></a><span data-ttu-id="c0265-103">Tworzenie okien czasowych</span><span class="sxs-lookup"><span data-stu-id="c0265-103">Create time windows</span></span>   

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c0265-104">Okna czasu umożliwiają optymalizowanie planowania wierszy zleceń serwisowych.</span><span class="sxs-lookup"><span data-stu-id="c0265-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="c0265-105">Kliknij kolejno opcje **Zarządzanie serwisem** \> **Ustawienia** \> **Umowy serwisowe** \> **Okno czasu**.</span><span class="sxs-lookup"><span data-stu-id="c0265-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>
2. <span data-ttu-id="c0265-106">Naciśnij klawisze **Alt+N**, aby utworzyć nowe okno czasu.</span><span class="sxs-lookup"><span data-stu-id="c0265-106">Press **Alt+N** to create a new time window.</span></span>
3. <span data-ttu-id="c0265-107">W polu **Okno czasu** nazwij okno czasu.</span><span class="sxs-lookup"><span data-stu-id="c0265-107">In the **Time window** field, enter a name for the time window.</span></span>
4. <span data-ttu-id="c0265-108">W polu **Opis** wprowadź opis dla okna czasu.</span><span class="sxs-lookup"><span data-stu-id="c0265-108">In the **Description** field, enter a description for the time window.</span></span>
5. <span data-ttu-id="c0265-109">W polu **Metoda** wybierz metodę automatycznego tworzenia.</span><span class="sxs-lookup"><span data-stu-id="c0265-109">In the **Method** field, select the method to be autocreated.</span></span> <span data-ttu-id="c0265-110">Jeśli wybrano metodę typu **Ręcznie**, podaj liczbę dni przed lub po obliczonej dacie określającą możliwy okres przenoszenia wiersza zlecenia serwisowego.</span><span class="sxs-lookup"><span data-stu-id="c0265-110">If the method is of the **Manual** type, specify the number of days before or after the calculated date that the service order line can be moved.</span></span>
