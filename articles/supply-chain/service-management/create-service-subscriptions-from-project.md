# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="b60b1-103">Tworzenie subskrypcji serwisu na podstawie projektu</span><span class="sxs-lookup"><span data-stu-id="b60b1-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="b60b1-104">Można utworzyć subskrypcję serwisu na podstawie projektu w module **Zarządzanie projektami i ich księgowanie**.</span><span class="sxs-lookup"><span data-stu-id="b60b1-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="b60b1-105">Kliknij kolejno opcje **Zarządzanie projektami i ich księgowanie** \> **Wspólne** \> **Projekty** \> **Wszystkie projekty**.</span><span class="sxs-lookup"><span data-stu-id="b60b1-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="b60b1-106">Otwórz projekt, dla którego ma zostać utworzona subskrybcja.</span><span class="sxs-lookup"><span data-stu-id="b60b1-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="b60b1-107">W formularzu **Projekty** w **okienku akcji** na karcie **Zarządzaj** kliknij kolejno opcje **Serwis** \> **Subskrypcja serwisu**.</span><span class="sxs-lookup"><span data-stu-id="b60b1-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="b60b1-108">W formularzu **Subskrypcja** w polu **Subskrypcja serwisu** wprowadź unikatowy numer identyfikacyjny.</span><span class="sxs-lookup"><span data-stu-id="b60b1-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="b60b1-109">Numer projektu zostanie automatycznie wprowadzony do subskrypcji dla wybranego projektu.</span><span class="sxs-lookup"><span data-stu-id="b60b1-109">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="b60b1-110">Wprowadź pozostałe informacje w grupie formularzu .</span><span class="sxs-lookup"><span data-stu-id="b60b1-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="b60b1-111">Teraz możesz tworzyć transakcje dotyczące subskrypcji.</span><span class="sxs-lookup"><span data-stu-id="b60b1-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="b60b1-112">Jeśli jest skonfigurowana sekwencja numerów dla grupy subskrypcji, identyfikator subskrypcji jest generowany automatycznie w momencie określenia grupy subskrypcji.</span><span class="sxs-lookup"><span data-stu-id="b60b1-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
