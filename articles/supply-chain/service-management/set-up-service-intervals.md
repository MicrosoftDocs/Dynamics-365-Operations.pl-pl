[!include [banner](../includes/banner.md)]

<span data-ttu-id="1dffd-104">Interwał serwisu określa częstotliwość, z jaką są tworzone wiersze umowy serwisowej podczas tworzenia zleceń serwisowych.</span><span class="sxs-lookup"><span data-stu-id="1dffd-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="1dffd-105">Kliknij kolejno opcje **Zarządzanie serwisem** \> **Ustawienia** \> **Umowy serwisowe** \> **Ramy czasowe serwisów**.</span><span class="sxs-lookup"><span data-stu-id="1dffd-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="1dffd-106">Utwórz nowe ramy czasowe serwisu.</span><span class="sxs-lookup"><span data-stu-id="1dffd-106">Create a new service interval.</span></span>
3. <span data-ttu-id="1dffd-107">Wprowadź identyfikator i opis ram czasowych serwisu.</span><span class="sxs-lookup"><span data-stu-id="1dffd-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="1dffd-108">W polu **Zakres** wybierz zakres.</span><span class="sxs-lookup"><span data-stu-id="1dffd-108">In the **Range** field, select the range.</span></span>
