# <a name="set-up-service-intervals"></a><span data-ttu-id="b18f6-103">Konfigurowanie ram czasowych serwisu</span><span class="sxs-lookup"><span data-stu-id="b18f6-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b18f6-104">Interwał serwisu określa częstotliwość, z jaką są tworzone wiersze umowy serwisowej podczas tworzenia zleceń serwisowych.</span><span class="sxs-lookup"><span data-stu-id="b18f6-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="b18f6-105">Kliknij kolejno opcje **Zarządzanie serwisem** \> **Ustawienia** \> **Umowy serwisowe** \> **Ramy czasowe serwisów**.</span><span class="sxs-lookup"><span data-stu-id="b18f6-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="b18f6-106">Utwórz nowe ramy czasowe serwisu.</span><span class="sxs-lookup"><span data-stu-id="b18f6-106">Create a new service interval.</span></span>
3. <span data-ttu-id="b18f6-107">Wprowadź identyfikator i opis ram czasowych serwisu.</span><span class="sxs-lookup"><span data-stu-id="b18f6-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="b18f6-108">W polu **Zakres** wybierz zakres.</span><span class="sxs-lookup"><span data-stu-id="b18f6-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="b18f6-109">W polu **Częstotliwość** wpisz częstotliwość.</span><span class="sxs-lookup"><span data-stu-id="b18f6-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="b18f6-110">Częstotliwość to współczynnik, przez który należy pomnożyć zakres w celu uzyskania ram czasowych umowy serwisowej.</span><span class="sxs-lookup"><span data-stu-id="b18f6-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="b18f6-111">Naciśnij klawisze **Alt+S**, aby zapisać ramy czasowe serwisu.</span><span class="sxs-lookup"><span data-stu-id="b18f6-111">Press **Alt+S** to save the service interval.</span></span>
