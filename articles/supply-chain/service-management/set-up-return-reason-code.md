<span data-ttu-id="15339-110">Ta procedura pomaga skonfigurować kody przyczyny zwrotu, których można użyć, aby wskazać, dlaczego produkt został zwrócony przez klienta.</span><span class="sxs-lookup"><span data-stu-id="15339-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>
