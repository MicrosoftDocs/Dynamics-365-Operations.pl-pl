[!include [banner](../includes/banner.md)]

<span data-ttu-id="56af3-107">Aby włączyć zarządzanie niezgodnościami, wykonaj następujące kroki:</span><span class="sxs-lookup"><span data-stu-id="56af3-107">To enable nonconformance management, follow these steps:</span></span>

    -   <span data-ttu-id="56af3-109">W opcji **Korzystaj z funkcji zarządzania jakością** wybierz wartość **Tak**.</span><span class="sxs-lookup"><span data-stu-id="56af3-109">Set the **Use quality management** option to **Yes**.</span></span>
    -   <span data-ttu-id="56af3-110">W polu **Stawka godzinowa** wpisz stawkę godzinową pracy w walucie lokalnej.</span><span class="sxs-lookup"><span data-stu-id="56af3-110">In the **Hourly rate** field, enter an hourly labor rate in the local currency.</span></span> <span data-ttu-id="56af3-111">Stawka godzinowa używana jest do obliczania kosztów operacji związanych z niezgodnością.</span><span class="sxs-lookup"><span data-stu-id="56af3-111">The hourly rate is used to calculate costs for operations that are related to a nonconformance.</span></span> <span data-ttu-id="56af3-112">Stawka godzinowa i obliczone koszty dostarczają informacji pomocniczych o niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-112">The hourly rate and calculated costs provide reference information for a nonconformance.</span></span> <span data-ttu-id="56af3-113">Nie mają one styczności z innymi funkcjami.</span><span class="sxs-lookup"><span data-stu-id="56af3-113">They don't interact with other functionality.</span></span>
    -   <span data-ttu-id="56af3-114">Użyj karty **Zarządzanie jakością** na stronie **Konfiguracja raportu**, aby zdefiniować typ dokumentu do wydrukowania.</span><span class="sxs-lookup"><span data-stu-id="56af3-114">Use the **Quality management** tab on the **Report setup** page to define the type of document to print.</span></span> <span data-ttu-id="56af3-115">Można wydrukować raport o niezgodności, znacznik niezgodności lub raport korekty.</span><span class="sxs-lookup"><span data-stu-id="56af3-115">You can print a nonconformance report, a nonconformance tag, or a correction report.</span></span> <span data-ttu-id="56af3-116">Możesz zdefiniować więcej niż jeden rekord w celu drukowania różnych typów dokumentów w raporcie lub drukowania notatek wewnętrznych i zewnętrznych.</span><span class="sxs-lookup"><span data-stu-id="56af3-116">You can define more than one record to print different document types on a report, or to print internal and external notes.</span></span> <span data-ttu-id="56af3-117">Strona **Typ dokumentu** może być pomocna przy definiowaniu unikatowego typu dokumentu dla niezgodności i unikatowego typu dokumentu dla korekty.</span><span class="sxs-lookup"><span data-stu-id="56af3-117">You might find it helpful to use the **Document type** page to define a unique document type for nonconformances and a unique document type for corrections.</span></span> <span data-ttu-id="56af3-118">Załóżmy, że chcesz wpisywać notatki dotyczące niezgodności za pomocą unikatowego typu dokumentu dla niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-118">For example, want to enter notes about a nonconformance by using the unique document type for nonconformances.</span></span> <span data-ttu-id="56af3-119">W takim przypadku zidentyfikuj unikatowy typ dokumentu w opcjach raportu.</span><span class="sxs-lookup"><span data-stu-id="56af3-119">In this case, identify the unique document type in the report options.</span></span>

3.  <span data-ttu-id="56af3-125">Definiowanie typów problemów, które będą przypisywane do niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-125">Define the problem types that will be assigned to nonconformances.</span></span> <span data-ttu-id="56af3-126">Użyj strony **Typ problemu** do zdefiniowania klasyfikacji problemów z jakością, które występują w różnych typach niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-126">Use the **Problem types** page to define a classification of quality problems that are encountered for the various nonconformance types.</span></span> <span data-ttu-id="56af3-127">Można skonfigurować następujące typy niezgodności: **Wewnętrzny**, **Odbiorca**, **Dostawca**, **Żądanie usługi**, **Produkcji** i **Wytwarzanie produktu towarzyszącego**.</span><span class="sxs-lookup"><span data-stu-id="56af3-127">You can set up the following nonconformance types: **Internal**, **Customer**, **Vendor**, **Service request**, **Production**, and **Co-product production**.</span></span> <span data-ttu-id="56af3-128">Użyj strony **Typy niezgodności**, do autoryzacji typu problemu, który ma być używany do obsługi co najmniej jednego typu niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-128">Use the **Non conformance types** page to authorize a problem type to be used in one or more nonconformance types.</span></span> <span data-ttu-id="56af3-129">Na przykład typ problemu związany z kodem wady może dotyczyć wszystkich typów niezgodności, podczas gdy typ problemu związany ze skargami odbiorców może dotyczyć tylko niezgodności typu **Odbiorca** i **Żądanie usługi**.</span><span class="sxs-lookup"><span data-stu-id="56af3-129">For example, a problem type that is related to a defect code might apply to all nonconformance types, whereas a problem type that is related to customer complaints might apply only to the **Customer** and **Service request** nonconformance types.</span></span>
4.  <span data-ttu-id="56af3-130">Zdefiniuj strefy kwarantanny, aby podać wskazówki dotyczące postępowania z wadliwymi materiałami.</span><span class="sxs-lookup"><span data-stu-id="56af3-130">Define quarantine zones to provide guidance about defective material should be handled.</span></span> <span data-ttu-id="56af3-131">Na stronie **Strefy kwarantanny** zdefiniuj strefy, które można przypisać do niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-131">Use the **Quarantine zones** page to define zones that can be assigned to a nonconformance.</span></span> <span data-ttu-id="56af3-132">Wydrukowany znacznik niezgodności będzie zawierać przypisaną strefę kwarantanny wraz z informacjami użytku jako wskazówkę w postępowaniu z wadliwymi materiałami.</span><span class="sxs-lookup"><span data-stu-id="56af3-132">The printed nonconformance tag will show the assigned quarantine zone and information about usage, to provide guidance about how to handle defective material.</span></span> <span data-ttu-id="56af3-133">Strefy mogą odpowiadać lokalizacjom magazynowym lub zasobom operacyjnym.</span><span class="sxs-lookup"><span data-stu-id="56af3-133">The zones might correspond to inventory locations or operations resources.</span></span>
5.  <span data-ttu-id="56af3-134">Zdefiniuj typy diagnostyki przypisywane do korekty.</span><span class="sxs-lookup"><span data-stu-id="56af3-134">Define the diagnostic types that will be assigned to corrections.</span></span> <span data-ttu-id="56af3-135">Na stronie **Typy diagnostyki** zdefiniuj klasyfikację akcji diagnostycznych.</span><span class="sxs-lookup"><span data-stu-id="56af3-135">Use the **Diagnostic types** page to define a classification of diagnostic actions.</span></span> <span data-ttu-id="56af3-136">Korekta określa jakiego rodzaju działania diagnostyczne należy podjąć w przypadku zatwierdzonej niezgodności, a także osobę, która ma to działanie wykonać.</span><span class="sxs-lookup"><span data-stu-id="56af3-136">A correction specifies what type of diagnostic action should be taken for an approved nonconformance, and who should take that action.</span></span> <span data-ttu-id="56af3-137">Określa również żądaną datę ukończenia żądania planowaną datę ukończenia.</span><span class="sxs-lookup"><span data-stu-id="56af3-137">It also specifies the requested completion date and the planned completion date.</span></span>
6.  <span data-ttu-id="56af3-138">Zdefiniuj operacje pokrewne, które będą przypisywane do niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-138">Define the related operations that will be assigned to nonconformances.</span></span> <span data-ttu-id="56af3-139">Za pomocą strony **Operacje** zdefiniuj klasyfikację pracy, którą można wykonać dla zatwierdzonej niezgodności.</span><span class="sxs-lookup"><span data-stu-id="56af3-139">Use the **Operations** page to define a classification of the work that can be performed for an approved nonconformance.</span></span> <span data-ttu-id="56af3-140">Przypisując operację pokrewną do niezgodności, można również zdefiniować szczegółowe informacje o powiązanych materiałach, godzinach robocizny i opłatach dodatkowych wymaganych do wykonania operacji.</span><span class="sxs-lookup"><span data-stu-id="56af3-140">When you assign a related operation to a nonconformance, you can provide detailed information, such as information about the associated material, labor hours, and miscellaneous charges that are required in order to perform the operation.</span></span> <span data-ttu-id="56af3-141">Ta informacja służy do obliczania szacowanego kosztu operacji.</span><span class="sxs-lookup"><span data-stu-id="56af3-141">This information is used to calculate an estimated cost for the operation.</span></span> <span data-ttu-id="56af3-142">Informacje szczegółowe i szacowane koszty mają charakter odnośnikowy.</span><span class="sxs-lookup"><span data-stu-id="56af3-142">The detailed information and estimated costs are for reference.</span></span> <span data-ttu-id="56af3-143">Operacje pokrewne dla jakości różnią się od operacji, które można zdefiniować dla marszruty produkcji.</span><span class="sxs-lookup"><span data-stu-id="56af3-143">The related operations for quality differ from the operations that can be defined for a production route.</span></span>
