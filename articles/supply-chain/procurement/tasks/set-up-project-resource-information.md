<span data-ttu-id="ce26a-104">W tej procedurze pokazano sposób konfigurowania zasobu projektu.</span><span class="sxs-lookup"><span data-stu-id="ce26a-104">This procedure shows how to set up a project resource.</span></span> <span data-ttu-id="ce26a-105">W tym zadaniu jest wykorzystywany zestaw danych firmy USMF.</span><span class="sxs-lookup"><span data-stu-id="ce26a-105">This task uses the USMF data set.</span></span>

3. <span data-ttu-id="ce26a-108">W polu Pracownik wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="ce26a-108">In the Worker field, enter or select a value.</span></span>

6. <span data-ttu-id="ce26a-111">Kliknij przycisk Utwórz.</span><span class="sxs-lookup"><span data-stu-id="ce26a-111">Click Create.</span></span>
7. <span data-ttu-id="ce26a-112">Kliknij opcję Koszty własne.</span><span class="sxs-lookup"><span data-stu-id="ce26a-112">Click Cost prices.</span></span>
8. <span data-ttu-id="ce26a-113">Kliknij opcję Koszt własny — Godzina.</span><span class="sxs-lookup"><span data-stu-id="ce26a-113">Click Cost price - hour.</span></span>
9. <span data-ttu-id="ce26a-114">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="ce26a-114">Click New.</span></span>
10. <span data-ttu-id="ce26a-115">W polu Data obowiązywania wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="ce26a-115">In the Effective date field, enter a date.</span></span>
11. <span data-ttu-id="ce26a-116">W polu Koszt własny wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="ce26a-116">In the Cost price field, enter a number.</span></span>
12. <span data-ttu-id="ce26a-117">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="ce26a-117">Click Save.</span></span>
13. <span data-ttu-id="ce26a-118">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="ce26a-118">Close the page.</span></span>
14. <span data-ttu-id="ce26a-119">Kliknij opcję Ceny sprzedaży.</span><span class="sxs-lookup"><span data-stu-id="ce26a-119">Click Sales prices.</span></span>
15. <span data-ttu-id="ce26a-120">Kliknij opcję Cena sprzedaży — Godzina.</span><span class="sxs-lookup"><span data-stu-id="ce26a-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="ce26a-121">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="ce26a-121">Click New.</span></span>
17. <span data-ttu-id="ce26a-122">W polu Data obowiązywania wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="ce26a-122">In the Effective date field, enter a date.</span></span>
18. <span data-ttu-id="ce26a-123">W polu Ceny wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="ce26a-123">In the Pricing field, enter a number.</span></span>
19. <span data-ttu-id="ce26a-124">W polu Ważny dla wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="ce26a-124">In the Valid for field, select an option.</span></span>
