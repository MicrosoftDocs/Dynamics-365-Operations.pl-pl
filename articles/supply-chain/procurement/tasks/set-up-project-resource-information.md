<span data-ttu-id="1717e-104">W tej procedurze pokazano sposób konfigurowania zasobu projektu.</span><span class="sxs-lookup"><span data-stu-id="1717e-104">This procedure shows how to set up a project resource.</span></span> <span data-ttu-id="1717e-105">W tym zadaniu jest wykorzystywany zestaw danych firmy USMF.</span><span class="sxs-lookup"><span data-stu-id="1717e-105">This task uses the USMF data set.</span></span>

1. <span data-ttu-id="1717e-106">Wybierz kolejno opcje Zarządzanie projektami i ich księgowanie > Zasoby projektu > Lista zasobów.</span><span class="sxs-lookup"><span data-stu-id="1717e-106">Go to Project management and accounting > Project resources > Resources list.</span></span>
2. <span data-ttu-id="1717e-107">Kliknij opcję Nowy zasób.</span><span class="sxs-lookup"><span data-stu-id="1717e-107">Click New resource.</span></span>
3. <span data-ttu-id="1717e-108">W polu Pracownik wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="1717e-108">In the Worker field, enter or select a value.</span></span>

6. <span data-ttu-id="1717e-111">Kliknij przycisk Utwórz.</span><span class="sxs-lookup"><span data-stu-id="1717e-111">Click Create.</span></span>
7. <span data-ttu-id="1717e-112">Kliknij opcję Koszty własne.</span><span class="sxs-lookup"><span data-stu-id="1717e-112">Click Cost prices.</span></span>
8. <span data-ttu-id="1717e-113">Kliknij opcję Koszt własny — Godzina.</span><span class="sxs-lookup"><span data-stu-id="1717e-113">Click Cost price - hour.</span></span>
9. <span data-ttu-id="1717e-114">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="1717e-114">Click New.</span></span>
10. <span data-ttu-id="1717e-115">W polu Data obowiązywania wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="1717e-115">In the Effective date field, enter a date.</span></span>
11. <span data-ttu-id="1717e-116">W polu Koszt własny wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="1717e-116">In the Cost price field, enter a number.</span></span>
12. <span data-ttu-id="1717e-117">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="1717e-117">Click Save.</span></span>
13. <span data-ttu-id="1717e-118">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="1717e-118">Close the page.</span></span>
14. <span data-ttu-id="1717e-119">Kliknij opcję Ceny sprzedaży.</span><span class="sxs-lookup"><span data-stu-id="1717e-119">Click Sales prices.</span></span>
15. <span data-ttu-id="1717e-120">Kliknij opcję Cena sprzedaży — Godzina.</span><span class="sxs-lookup"><span data-stu-id="1717e-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="1717e-121">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="1717e-121">Click New.</span></span>
17. <span data-ttu-id="1717e-122">W polu Data obowiązywania wprowadź datę.</span><span class="sxs-lookup"><span data-stu-id="1717e-122">In the Effective date field, enter a date.</span></span>
18. <span data-ttu-id="1717e-123">W polu Ceny wpisz liczbę.</span><span class="sxs-lookup"><span data-stu-id="1717e-123">In the Pricing field, enter a number.</span></span>
19. <span data-ttu-id="1717e-124">W polu Ważny dla wybierz opcję.</span><span class="sxs-lookup"><span data-stu-id="1717e-124">In the Valid for field, select an option.</span></span>
20. <span data-ttu-id="1717e-125">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="1717e-125">Click Save.</span></span>
21. <span data-ttu-id="1717e-126">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="1717e-126">Close the page.</span></span>
