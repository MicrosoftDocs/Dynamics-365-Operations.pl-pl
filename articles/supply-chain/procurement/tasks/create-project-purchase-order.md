1. <span data-ttu-id="36433-106">Wybierz kolejno opcje Zarządzanie projektami i ich księgowanie> Projekty > Wszystkie projekty.</span><span class="sxs-lookup"><span data-stu-id="36433-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="36433-107">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="36433-107">In the list, click the link in the selected row.</span></span>

4. <span data-ttu-id="36433-109">Kliknij opcję Zadanie dotyczące towaru.</span><span class="sxs-lookup"><span data-stu-id="36433-109">Click Item task.</span></span>
5. <span data-ttu-id="36433-110">Kliknij opcję Zamówienie zakupu.</span><span class="sxs-lookup"><span data-stu-id="36433-110">Click Purchase order.</span></span>
6. <span data-ttu-id="36433-111">W polu Konto dostawcy wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="36433-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="36433-112">W polu Oddział wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="36433-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="36433-113">Te kroki nie są wymagane, ale upraszczają tworzenie zamówienia zakupu poprzez skonfigurowanie domyślnego oddziału i magazynu dla wierszy zamówienia zakupu.</span><span class="sxs-lookup"><span data-stu-id="36433-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="36433-114">W polu Magazyn wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="36433-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="36433-115">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="36433-115">Click OK.</span></span>
10. <span data-ttu-id="36433-116">Na liście oznacz wybrany wiersz.</span><span class="sxs-lookup"><span data-stu-id="36433-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="36433-117">W polu Numer towaru wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="36433-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="36433-118">Może to być numer towaru lub kategoria zaopatrzenia.</span><span class="sxs-lookup"><span data-stu-id="36433-118">This can be the item number or a procurement category.</span></span>

13. <span data-ttu-id="36433-120">Kliknij kartę Projekt.</span><span class="sxs-lookup"><span data-stu-id="36433-120">Click the Project tab.</span></span>
    * <span data-ttu-id="36433-121">Sprawdź, czy cena sprzedaży i koszt własny są dostępne.</span><span class="sxs-lookup"><span data-stu-id="36433-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="36433-122">Jeśli nie są dostępne, ale konieczne, wprowadź informacje.</span><span class="sxs-lookup"><span data-stu-id="36433-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="36433-123">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="36433-123">Click Save.</span></span>
