[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="50fd9-104">W tej procedurze pokazano sposób tworzenia zamówienia zakupu do projektu.</span><span class="sxs-lookup"><span data-stu-id="50fd9-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="50fd9-105">W tym zadaniu jest wykorzystywany zestaw danych firmy USSI.</span><span class="sxs-lookup"><span data-stu-id="50fd9-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="50fd9-106">Wybierz kolejno opcje Zarządzanie projektami i ich księgowanie> Projekty > Wszystkie projekty.</span><span class="sxs-lookup"><span data-stu-id="50fd9-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="50fd9-107">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="50fd9-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="50fd9-108">W okienku akcji kliknij pozycję Zarządzaj.</span><span class="sxs-lookup"><span data-stu-id="50fd9-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="50fd9-109">Kliknij opcję Zadanie dotyczące towaru.</span><span class="sxs-lookup"><span data-stu-id="50fd9-109">Click Item task.</span></span>
5. <span data-ttu-id="50fd9-110">Kliknij opcję Zamówienie zakupu.</span><span class="sxs-lookup"><span data-stu-id="50fd9-110">Click Purchase order.</span></span>
6. <span data-ttu-id="50fd9-111">W polu Konto dostawcy wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="50fd9-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="50fd9-112">W polu Oddział wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="50fd9-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="50fd9-113">Te kroki nie są wymagane, ale upraszczają tworzenie zamówienia zakupu poprzez skonfigurowanie domyślnego oddziału i magazynu dla wierszy zamówienia zakupu.</span><span class="sxs-lookup"><span data-stu-id="50fd9-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="50fd9-114">W polu Magazyn wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="50fd9-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="50fd9-115">Kliknij przycisk OK.</span><span class="sxs-lookup"><span data-stu-id="50fd9-115">Click OK.</span></span>
10. <span data-ttu-id="50fd9-116">Na liście oznacz wybrany wiersz.</span><span class="sxs-lookup"><span data-stu-id="50fd9-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="50fd9-117">W polu Numer towaru wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="50fd9-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="50fd9-118">Może to być numer towaru lub kategoria zaopatrzenia.</span><span class="sxs-lookup"><span data-stu-id="50fd9-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="50fd9-119">Rozwiń sekcję Szczegóły wiersza.</span><span class="sxs-lookup"><span data-stu-id="50fd9-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="50fd9-120">Kliknij kartę Projekt.</span><span class="sxs-lookup"><span data-stu-id="50fd9-120">Click the Project tab.</span></span>
    * <span data-ttu-id="50fd9-121">Sprawdź, czy cena sprzedaży i koszt własny są dostępne.</span><span class="sxs-lookup"><span data-stu-id="50fd9-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="50fd9-122">Jeśli nie są dostępne, ale konieczne, wprowadź informacje.</span><span class="sxs-lookup"><span data-stu-id="50fd9-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="50fd9-123">Kliknij przycisk Zapisz.</span><span class="sxs-lookup"><span data-stu-id="50fd9-123">Click Save.</span></span>
