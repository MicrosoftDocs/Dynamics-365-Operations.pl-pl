[!include[task guide banner](../includes/task-guide-banner.md)]

<span data-ttu-id="de3b7-104">Ta procedura pokazuje, jak skonfigurować ustawienia profilu funkcji sklepu, które mają zastosowanie do przedstawicieli handlowych.</span><span class="sxs-lookup"><span data-stu-id="de3b7-104">This procedure demonstrates how to configure a store's functionality profile settings that apply to sales representatives.</span></span> <span data-ttu-id="de3b7-105">Procedura wykorzystuje dane firmy demonstracyjnej USRT.</span><span class="sxs-lookup"><span data-stu-id="de3b7-105">This procedures uses the USRT demo data company.</span></span>

1. <span data-ttu-id="de3b7-106">Wybierz kolejno opcje Handel detaliczny i inny > Ustawienia kanału > Ustawienia punktu sprzedaży > Profile punktów sprzedaży > Profile funkcji.</span><span class="sxs-lookup"><span data-stu-id="de3b7-106">Go to Retail and commerce > Channel setup > POS setup > POS profiles > Functionality profiles.</span></span>
2. <span data-ttu-id="de3b7-107">Kliknij przycisk Edytuj.</span><span class="sxs-lookup"><span data-stu-id="de3b7-107">Click Edit.</span></span>
3. <span data-ttu-id="de3b7-108">Rozwiń sekcję Funkcje.</span><span class="sxs-lookup"><span data-stu-id="de3b7-108">Expand the Functions section.</span></span>
    * <span data-ttu-id="de3b7-109">Funkcja ustawień profilu umożliwia skonfigurowania punktu sprzedaży, aby automatycznie dodawał domyślną grupę sprzedaży kasjera, monitował o podanie grup sprzedaży i wymagał grup sprzedaży.</span><span class="sxs-lookup"><span data-stu-id="de3b7-109">You can use the functionality profile settings to configure POS to automatically add the cashier's default sales group, to prompt for sales groups, and to require sales groups.</span></span>

5. <span data-ttu-id="de3b7-111">W polu Wymagaj przedstawiciela handlowego wybierz opcję Tak.</span><span class="sxs-lookup"><span data-stu-id="de3b7-111">Select Yes in the Require sales representative field.</span></span>
