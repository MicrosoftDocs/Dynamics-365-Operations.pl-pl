[!include[task guide banner](../includes/task-guide-banner.md)]

<span data-ttu-id="035da-104">Ta procedura przedstawia sposób wyszukiwania informacji o karcie lojalnościowej i korygowania punktów lojalnościowych.</span><span class="sxs-lookup"><span data-stu-id="035da-104">This procedure demonstrates how to look up loyalty card information and adjust loyalty reward points.</span></span> <span data-ttu-id="035da-105">Dane wykorzystane do stworzenia tego zadania pochodzą z firmy demonstracyjnej USRT.</span><span class="sxs-lookup"><span data-stu-id="035da-105">The demo data company used to create this task is USRT.</span></span> <span data-ttu-id="035da-106">Zadanie jest przeznaczone dla roli Kierownik ds. operacyjnych sieci sprzedaży lub Menedżer ds. obsługi klienta.</span><span class="sxs-lookup"><span data-stu-id="035da-106">This task is intended for the Retail operations manager role or a Customer service manager role.</span></span>

1. <span data-ttu-id="035da-107">Przejdź do okna Karty lojalnościowe.</span><span class="sxs-lookup"><span data-stu-id="035da-107">Go to Loyalty cards.</span></span>
2. <span data-ttu-id="035da-108">Na liście znajdź i zaznacz odpowiedni rekord.</span><span class="sxs-lookup"><span data-stu-id="035da-108">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="035da-109">Na liście kliknij łącze w wybranym wierszu.</span><span class="sxs-lookup"><span data-stu-id="035da-109">In the list, click the link in the selected row.</span></span>
4. <span data-ttu-id="035da-110">Kliknij opcję Transakcje kartą.</span><span class="sxs-lookup"><span data-stu-id="035da-110">Click Card transactions.</span></span>
    * <span data-ttu-id="035da-111">Na tej stronie można wyświetlić wszystkie transakcje lojalnościowe dla wybranej karty lojalnościowej.</span><span class="sxs-lookup"><span data-stu-id="035da-111">On this page you can view all loyalty transactions for the selected loyalty card.</span></span>  
5. <span data-ttu-id="035da-112">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="035da-112">Close the page.</span></span>
6. <span data-ttu-id="035da-113">Kliknij opcję Korekty karty.</span><span class="sxs-lookup"><span data-stu-id="035da-113">Click Card adjustments.</span></span>
7. <span data-ttu-id="035da-114">Kliknij przycisk Nowy.</span><span class="sxs-lookup"><span data-stu-id="035da-114">Click New.</span></span>
8. <span data-ttu-id="035da-115">W polu Punkty lojalnościowe wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="035da-115">In the Reward point field, enter or select a value.</span></span>
9. <span data-ttu-id="035da-116">W polu Kwota lub ilość wprowadź liczbę.</span><span class="sxs-lookup"><span data-stu-id="035da-116">In the Amount or quantity field, enter a number.</span></span>

10. <span data-ttu-id="035da-118">W polu Program lojalnościowy wprowadź lub wybierz wartość.</span><span class="sxs-lookup"><span data-stu-id="035da-118">In the Loyalty program field, enter or select a value.</span></span>
11. <span data-ttu-id="035da-119">W polu Komentarz wpisz wartość.</span><span class="sxs-lookup"><span data-stu-id="035da-119">In the Comment field, type a value.</span></span>
12. <span data-ttu-id="035da-120">Kliknij przycisk Księguj korektę.</span><span class="sxs-lookup"><span data-stu-id="035da-120">Click Post adjustment.</span></span>
13. <span data-ttu-id="035da-121">Kliknij przycisk Tak.</span><span class="sxs-lookup"><span data-stu-id="035da-121">Click Yes.</span></span>
14. <span data-ttu-id="035da-122">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="035da-122">Close the page.</span></span>
    * <span data-ttu-id="035da-123">Normalnie na tym etapie należałoby odświeżyć stronę, aby wyświetlić wynik korekty punktów lojalnościowych na karcie Podsumowanie punktów lojalnościowych. Jednak jeśli wykonujesz procedurę w ramach przewodnika po zadaniach, nie odświeżaj teraz, ponieważ jeśli to zrobisz, przewodnik się zatrzyma.</span><span class="sxs-lookup"><span data-stu-id="035da-123">Normally at this point you'd refresh the page to see the result of the reward points adjustment in the Reward point summary tab. 15. <span data-ttu-id="035da-124">Kliknij opcję Transakcje kartą.</span><span class="sxs-lookup"><span data-stu-id="035da-124">Click Card transactions.</span></span>
16. <span data-ttu-id="035da-125">Zamknij stronę.</span><span class="sxs-lookup"><span data-stu-id="035da-125">Close the page.</span></span>
