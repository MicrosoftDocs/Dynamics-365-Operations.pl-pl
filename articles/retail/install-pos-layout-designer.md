5.  <span data-ttu-id="6d2ea-116">Po zakończeniu instalacji przejdź do strony **Zaloguj**, wprowadź nazwę i hasło użytkownika programu Microsoft Dynamics 365 for Retail, a następnie kliknij przycisk **Zaloguj**, aby uruchomić projektanta.</span><span class="sxs-lookup"><span data-stu-id="6d2ea-116">After the installation is completed, on the **Sign in** page, enter your Microsoft Dynamics 365 for Retail user name and password, and then click **Sign in** to start the designer.</span></span>
