<span data-ttu-id="dc185-105">Na podstawie ustawień w module Centrala sieci sprzedaży produkty można sklasyfikować jako seryjne lub nieseryjne.</span><span class="sxs-lookup"><span data-stu-id="dc185-105">Based on the settings in Retail headquarters, products can be classified as either serialized or non-serialized.</span></span> <span data-ttu-id="dc185-106">Jeżeli produkty są seryjne, do każdej pozycji można przypisać unikatowy numer ułatwiający śledzenie gwarancji, śledzenie pozycji i potwierdzenie własności.</span><span class="sxs-lookup"><span data-stu-id="dc185-106">When products are serialized, each item can be assigned a unique number that helps track warranties, trace items, and confirm ownership.</span></span> <span data-ttu-id="dc185-107">Mimo że możliwość podania numerów seryjnych dla produktów seryjnych była dostępna w aplikacji Modern/Cloud Point of Sale (POS), dodano ulepszenia zapewniające kasjerom oszczędność czasu i zwiększenie wydajności pracy.</span><span class="sxs-lookup"><span data-stu-id="dc185-107">Although, the ability to provide serial numbers for serialized products existed in our Modern/Cloud Point of Sale (POS), some improvements have been added to help cashiers save time and be more productive.</span></span>

- <span data-ttu-id="dc185-109">**Numery seryjne nie są wymagane aż do realizacji zamówienia** — wcześniej kasjer, który dodał do transakcji produkt seryjny musiał podać numer seryjny.</span><span class="sxs-lookup"><span data-stu-id="dc185-109">**Serial numbers aren't required until checkout** – Previously, a cashier who added a serialized product to the transaction was required to provide a serial number.</span></span> <span data-ttu-id="dc185-110">To wymaganie było problemem w scenariuszach obsługi relacji z klientami, jeżeli kasjerzy i sprzedawcy mieli możliwość sprzedaży dodatkowych products.</span></span> <span data-ttu-id="dc185-111">Do momentu płatności produkty były często aktualizowane w koszyku.</span><span class="sxs-lookup"><span data-stu-id="dc185-111">Until the payment step, the products were often updated in the cart.</span></span> <span data-ttu-id="dc185-112">Dlatego za każdym razem, gdy kasjer dodał nowy produkt, system wyświetlał monit o podanie numeru <span data-ttu-id="dc185-113">Okno dialogowe numeru seryjnego zawiera teraz przycisk **Dodaj później**.</span><span class="sxs-lookup"><span data-stu-id="dc185-113">The serial number dialog box now     <span data-ttu-id="dc185-118">Dla każdej pozycji seryjnej, dla której numer seryjny został pominięty pod wierszem transakcji wyświetlany jest komentarz.</span><span class="sxs-lookup"><span data-stu-id="dc185-118">For each serialized item where the serial number was skipped, a comment appears under the transaction line.</span></span> <span data-ttu-id="dc185-119">Ten komentarz informuje, że nie podano numeru seryjnego dla pozycji.</span><span class="sxs-lookup"><span data-stu-id="dc185-119">This comment states that the serial number hasn't been provided for the item.</span></span> <span data-ttu-id="dc185-120">Dlatego kasjer może szybko znaleźć pozycje, w których brakuje numeru seryjnego.</span><span class="sxs-lookup"><span data-stu-id="dc185-120">Therefore, the cashier can quickly find items that are missing a serial number.</span></span>

    <span data-ttu-id="dc185-121">Nowe operacja **Dodaj numer seryjny** także umożliwia podanie numeru seryjnego dla pozycji, dla których brakuje numeru seryjnego.</span><span class="sxs-lookup"><span data-stu-id="dc185-121">A new **Add serial number** operation also provides the serial number for items that are missing a serial number.</span></span> <span data-ttu-id="dc185-122">Po podaniu numeru seryjnego nie można go edytować.</span><span class="sxs-lookup"><span data-stu-id="dc185-122">After the serial number is provided, it can't be edited.</span></span> <span data-ttu-id="dc185-123">Kasjer musi unieważnić wiersz i dodać produkt ponownie.</span><span class="sxs-lookup"><span data-stu-id="dc185-123">The cashier must void the line and add the product again.</span></span> 
    
- <span data-ttu-id="dc185-124">**Numery seryjne nie są wymagane w celu złożenia zamówień odbiorcy** — zamówienia odbiorcy można złożyć w jednym sklepie, a zrealizować w innym.</span><span class="sxs-lookup"><span data-stu-id="dc185-124">**Serial numbers aren't required in order to place customer orders** – Customer orders can be placed in one store and fulfilled from another.</span></span> <span data-ttu-id="dc185-125">Kasjer, który składa zamówienie odbiorcy nie musi podawać numeru seryjnego.</span><span class="sxs-lookup"><span data-stu-id="dc185-125">A cashier who places a customer order doesn't have to provide the serial number.</span></span> <span data-ttu-id="dc185-126">Numer seryjny zostanie podany na etapie pobierania lub odbioru.</span><span class="sxs-lookup"><span data-stu-id="dc185-126">The serial number will be provided during the picking or pickup step.</span></span> <span data-ttu-id="dc185-127">Numer seryjny należy jednak podać dla wszystkich pozycji wierszy, dla których wybrano typ dostawy **Wynieś**.</span><span class="sxs-lookup"><span data-stu-id="dc185-127">However, a serial number must be provided for all line items that the **Carry out** delivery type is selected for.</span></span> <span data-ttu-id="dc185-128">W przeciwnym razie nie można dokończyć transakcji.</span><span class="sxs-lookup"><span data-stu-id="dc185-128">Otherwise, the transaction can't be completed.</span></span>
